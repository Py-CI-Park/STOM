# 조건식 - 장 마감 전 급등 전략 (14:00~14:30)

- STOM 주식 자동거래에 사용하기 위한 틱 데이터 조건식 문서
- Back_Testing_Guideline_Tick.md 에 있는 조건식을 사용하여 만든 조건식

## 소개

이 문서는 14:00~14:30 장 마감 전 급등 종목을 포착하는 틱 기반 전략입니다.
장 마감을 앞두고 단기 급등하는 종목을 선별하여 빠른 진입과 청산을 목표로 합니다.

## 전략 개요

**타겟 시간대**: 14:00 ~ 14:30

**핵심 아이디어**:
1. 마감 전 급격한 등락율 상승
2. 초당거래대금 폭발적 증가
3. 체결강도 급상승
4. 빠른 청산 (15분 이내 목표)

## 조건식

### 매수 조건식

```python
초당순매수금액 = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000

if not (140000 <= 시분초 < 143000):
    매수 = False
elif not (2000 < 현재가 <= 30000):
    매수 = False
elif not (7.0 < 등락율 <= 25.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (등락율각도(30) > 15):
    매수 = False
elif not (초당거래대금 > 초당거래대금평균(30) * 3.0):
    매수 = False
elif not (체결강도 > 180):
    매수 = False
elif not (초당매수수량 > 최고초당매수수량(60) * 0.8):
    매수 = False
elif not (초당순매수금액 > 50):
    매수 = False
elif 시가총액 < 2000:
    if not (전일비 > 100):
        매수 = False
    elif not (당일거래대금 > 400):
        매수 = False
else:
    if not (전일비 > 70):
        매수 = False
    elif not (당일거래대금 > 700):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 2.0:
    매도 = True
elif 수익률 <= -1.5:
    매도 = True
elif 최고수익률 - 수익률 >= 1.0:
    매도 = True
elif 보유시간 > 900:
    매도 = True
elif 시분초 >= 145500:
    매도 = True
elif 체결강도 < 100 and 수익률 > 0.5:
    매도 = True
elif (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.8 and 수익률 > 0:
    매도 = True
```
