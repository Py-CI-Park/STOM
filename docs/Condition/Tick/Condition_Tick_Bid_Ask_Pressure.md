## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **Bid-Ask Pressure Strategy**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:00:00 (장중 전체)
- **대상 종목**: 관심종목, 시가총액 차등 기준
- **전략 타입**: Tick 기반 단타 전략
- **핵심 변수**: 체결강도, 초당거래대금, 등락율각도, 회전율
- **업데이트 이력**:
  - 초기 문서 작성: Bid-Ask Pressure Strategy

## 소개

이 문서는 **호가창 압력 분석 전략(Bid-Ask Pressure Strategy)**을 설명합니다. 장 초반(09:00:00 ~ 09:20:00) 매수/매도 호가창의 불균형을 분석하여 강한 매수세가 감지되는 종목을 포착하는 전략입니다.

### 전략 개요
- **거래 시간**: 90000 ~ 92000 (9시 ~ 9시 20분)
- **목표**: 호가창 매수 압력 감지
- **핵심 지표**: 매수총잔량, 매도총잔량, 호가 불균형
- **리스크 관리**: 호가창 역전 시 빠른 청산

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 호가창 데이터를 실시간으로 분석합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `BAP`: Bid-Ask Pressure를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_BAP_B`: 호가창 압력 전략의 매수 조건식
   - `C_BAP_S`: 호가창 압력 전략의 매도 조건식

# Condition - Bid-Ask Pressure

## 조건식

### 매수 조건식

```python
# Bid-Ask Pressure Strategy - 호가창 매수 압력 분석 전략
# 거래 시간: 90000 ~ 92000 (9시 ~ 9시 20분)

매수 = False

# 시간 제한: 장 초반 20분만 거래
if not (90000 <= 시분초 < 92000):
    매수 = False

# 기본 필터링
elif not (1500 < 현재가 <= 40000):
    매수 = False
elif not (1.5 < 등락율 <= 28.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 시가총액별 조건
elif 시가총액 < 2500:
    # 소형주 (2500억 미만)
    # 매수 호가 압력 > 매도 호가 압력
    if not (매수총잔량 > 매도총잔량 * 1.5):
        매수 = False
    elif not (매수총잔량 > 매수총잔량N(1)):
        매수 = False
    elif not (매도총잔량 < 매도총잔량N(1)):
        매수 = False
    # 매수 1~3호가 집중도
    elif not (매수잔량1 > 매수잔량2 and 매수잔량2 > 매수잔량3):
        매수 = False
    elif not (매수잔량1 > 매수총잔량 * 25 / 100):
        매수 = False
    # 매도 호가 분산
    elif not (매도잔량1 < 매도총잔량 * 30 / 100):
        매수 = False
    # 초당 매수 우위
    elif not (초당매수수량 > 초당매도수량 * 1.8):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 40 / 100):
        매수 = False
    # 체결강도
    elif not (체결강도 > 140):
        매수 = False
    elif not (체결강도평균(30) > 120):
        매수 = False
    # 거래 활성도
    elif not (당일거래대금 > 8 * 100):
        매수 = False
    elif not (전일비 > 120):
        매수 = False
    elif not (전일동시간비 > 180):
        매수 = False
    # 가격 위치
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False
    elif not (등락율각도(30) > 6):
        매수 = False

else:
    # 중대형주 (2500억 이상)
    # 매수 호가 압력 > 매도 호가 압력
    if not (매수총잔량 > 매도총잔량 * 1.3):
        매수 = False
    elif not (매수총잔량 > 매수총잔량N(1)):
        매수 = False
    elif not (매도총잔량 < 매도총잔량N(1)):
        매수 = False
    # 매수 1~3호가 집중도
    elif not (매수잔량1 > 매수잔량2):
        매수 = False
    elif not (매수잔량1 > 매수총잔량 * 20 / 100):
        매수 = False
    # 매도 호가 분산
    elif not (매도잔량1 < 매도총잔량 * 35 / 100):
        매수 = False
    # 초당 매수 우위
    elif not (초당매수수량 > 초당매도수량 * 1.5):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 30 / 100):
        매수 = False
    # 체결강도
    elif not (체결강도 > 130):
        매수 = False
    elif not (체결강도평균(30) > 115):
        매수 = False
    # 거래 활성도
    elif not (당일거래대금 > 30 * 100):
        매수 = False
    elif not (전일비 > 90):
        매수 = False
    elif not (전일동시간비 > 120):
        매수 = False
    # 가격 위치
    elif not (현재가 > 고가 - (고가 - 저가) * 30 / 100):
        매수 = False
    elif not (등락율각도(30) > 8):
        매수 = False
```

### 매도 조건식

```python
# Bid-Ask Pressure Strategy - 호가창 역전 감지 매도 전략

매도 = False

# 상한가 근접 시 매도
if 등락율 > 29.5:
    매도 = True

# 수익률 기준 매도
elif 수익률 >= 3.5:
    매도 = True
elif 수익률 <= -2.5:
    매도 = True

# 최고수익률 대비 하락 시 매도
elif 최고수익률 >= 4.0 and 수익률 < 최고수익률 - 1.5:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    # 소형주 매도 조건 - 호가창 역전 감지
    if 매도총잔량 > 매수총잔량 * 1.3 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 30 / 100:
        매도 = True
    elif 매도총잔량 > 매도총잔량N(1) * 1.2 and 매수총잔량 < 매수총잔량N(1) * 0.9:
        매도 = True
    # 매도 1호가 집중
    elif 매도잔량1 > 매도총잔량 * 40 / 100 and 체결강도 < 100:
        매도 = True
    # 초당 매도 우위
    elif (초당매도수량 - 초당매수수량) >= 매수총잔량 * 50 / 100:
        매도 = True
    # 체결강도 급락
    elif 체결강도 < 90 and 체결강도N(1) > 130:
        매도 = True
    elif 체결강도평균(30) < 100 and 체결강도평균(30, 1) > 120:
        매도 = True
    # 시간 기준
    elif 보유시간 > 480:  # 8분 이상 보유 시 매도
        매도 = True
    # 가격 하락
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 20 / 100:
            매도 = True

else:
    # 중대형주 매도 조건 - 호가창 역전 감지
    if 매도총잔량 > 매수총잔량 * 1.2 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 40 / 100:
        매도 = True
    elif 매도총잔량 > 매도총잔량N(1) * 1.15 and 매수총잔량 < 매수총잔량N(1) * 0.92:
        매도 = True
    # 매도 1호가 집중
    elif 매도잔량1 > 매도총잔량 * 35 / 100 and 체결강도 < 110:
        매도 = True
    # 초당 매도 우위
    elif (초당매도수량 - 초당매수수량) >= 매수총잔량 * 45 / 100:
        매도 = True
    # 체결강도 급락
    elif 체결강도 < 100 and 체결강도N(1) > 125:
        매도 = True
    elif 체결강도평균(30) < 105 and 체결강도평균(30, 1) > 118:
        매도 = True
    # 시간 기준
    elif 보유시간 > 720:  # 12분 이상 보유 시 매도
        매도 = True
    # 가격 하락
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 25 / 100:
            매도 = True
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
  - 14:30-15:00: 단기 스캘핑 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **동적 손절/익절 시스템**
  - 변동성 기반 손절폭 조정 (ATR 활용)
  - 트레일링 스톱 최적화
  - 구현 난이도: 중

- **부분 매도 전략**
  - 목표가 1차 도달 시 50% 매도
  - 나머지 50%는 트레일링 스톱 적용
  - 구현 난이도: 낮음
