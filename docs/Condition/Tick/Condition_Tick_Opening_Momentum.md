## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반(09:00~09:20) 개장 모멘텀 전략(Opening Momentum)**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 시가총액 차등 (3,500억 기준), 가격대 2,000원~45,000원
- **전략 타입**: 갭 상승 + 강한 모멘텀 포착 (Opening Momentum)
- **핵심 변수**: 시가등락율, 시가대비등락율, 고가대비하락율, 등락율각도, 초당거래대금
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 **장 초반 모멘텀 전략(Opening Momentum Strategy)**을 설명합니다. 시가 대비 강한 상승 모멘텀을 보이며 갭 상승한 종목을 포착하여 추가 상승을 노리는 전략입니다.

### 전략 개요
- **거래 시간**: 90000 ~ 92000 (9시 ~ 9시 20분)
- **목표**: 갭 상승 + 강한 모멘텀 종목 포착
- **핵심 지표**: 시가대비등락율, 등락율각도, 고가대비위치
- **리스크 관리**: 모멘텀 약화 시 빠른 청산

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 시가와 현재가의 관계를 중점적으로 분석합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `OM`: Opening Momentum을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_OM_B`: 장 초반 모멘텀 전략의 매수 조건식
   - `C_OM_S`: 장 초반 모멘텀 전략의 매도 조건식

# Condition - Opening Momentum

## 조건식

### 매수 조건식

```python
# Opening Momentum Strategy - 갭 상승 모멘텀 포착 전략
# 거래 시간: 90000 ~ 92000 (9시 ~ 9시 20분)

매수 = False

# 계산 변수
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
고가대비하락율 = ((고가 - 현재가) / 고가) * 100

# 시간 제한: 장 초반 20분만 거래
if not (90000 <= 시분초 < 92000):
    매수 = False

# 기본 필터링
elif not (2000 < 현재가 <= 45000):
    매수 = False
elif not (2.5 < 등락율 <= 29.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 시가총액별 조건
elif 시가총액 < 3500:
    # 소형주 (3500억 미만)
    # 갭 상승 조건
    if not (2.0 <= 시가등락율 < 12.0):
        매수 = False
    # 시가 대비 추가 상승
    elif not (1.5 <= 시가대비등락율 < 10.0):
        매수 = False
    # 고가 근처 유지
    elif not (-3 < 고가대비하락율 < 10):
        매수 = False
    # 현재가 > 시가 확인
    elif not (현재가 > 시가):
        매수 = False
    elif not (현재가 > 현재가N(1)):
        매수 = False
    # 강한 상승 각도
    elif not (등락율각도(30) > 9):
        매수 = False
    # 거래량 급증
    elif not (초당거래대금 > 초당거래대금평균(30) * 2.5):
        매수 = False
    elif not (전일비 > 180):
        매수 = False
    elif not (전일동시간비 > 220):
        매수 = False
    # 체결강도
    elif not (체결강도 > 145):
        매수 = False
    elif not (체결강도평균(30) > 125):
        매수 = False
    # 매수 우위
    elif not (누적초당매수수량(30) > 누적초당매도수량(30) * 1.6):
        매수 = False
    # 거래대금
    elif not (당일거래대금 > 12 * 100):
        매수 = False
    # 회전율
    elif not (회전율 > 2.5):
        매수 = False
    # 고가 돌파 움직임
    elif not (현재가 > 고가 - (고가 - 저가) * 12 / 100):
        매수 = False

else:
    # 중대형주 (3500억 이상)
    # 갭 상승 조건
    if not (1.5 <= 시가등락율 < 10.0):
        매수 = False
    # 시가 대비 추가 상승
    elif not (1.0 <= 시가대비등락율 < 8.0):
        매수 = False
    # 고가 근처 유지
    elif not (-5 < 고가대비하락율 < 12):
        매수 = False
    # 현재가 > 시가 확인
    elif not (현재가 > 시가):
        매수 = False
    elif not (현재가 > 현재가N(1)):
        매수 = False
    # 강한 상승 각도
    elif not (등락율각도(30) > 11):
        매수 = False
    # 거래량 급증
    elif not (초당거래대금 > 초당거래대금평균(30) * 2.0):
        매수 = False
    elif not (전일비 > 130):
        매수 = False
    elif not (전일동시간비 > 160):
        매수 = False
    # 체결강도
    elif not (체결강도 > 135):
        매수 = False
    elif not (체결강도평균(30) > 118):
        매수 = False
    # 매수 우위
    elif not (누적초당매수수량(30) > 누적초당매도수량(30) * 1.4):
        매수 = False
    # 거래대금
    elif not (당일거래대금 > 60 * 100):
        매수 = False
    # 회전율
    elif not (회전율 > 1.8):
        매수 = False
    # 고가 돌파 움직임
    elif not (현재가 > 고가 - (고가 - 저가) * 18 / 100):
        매수 = False
```

### 매도 조건식

```python
# Opening Momentum Strategy - 모멘텀 약화 감지 매도 전략

매도 = False

# 상한가 근접 시 매도
if 등락율 > 29.5:
    매도 = True

# 수익률 기준 매도
elif 수익률 >= 5.0:
    매도 = True
elif 수익률 <= -2.0:
    매도 = True

# 최고수익률 대비 하락 시 매도
elif 최고수익률 >= 6.0 and 수익률 < 최고수익률 - 2.5:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 3500:
    # 소형주 매도 조건 - 모멘텀 약화 감지
    # 등락율 각도 기반 매도
    if 20 <= 등락율각도(30) and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 35 / 100:
        매도 = True
    elif 10 <= 등락율각도(30) < 20 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 55 / 100:
        매도 = True
    elif 0 <= 등락율각도(30) < 10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 35 / 100:
        매도 = True
    elif -10 <= 등락율각도(30) < 0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 25 / 100:
        매도 = True
    elif 등락율각도(30) < -10:
        매도 = True
    # 시가 아래로 하락
    elif 현재가 < 시가 and 수익률 < 2:
        매도 = True
    # 체결강도 약화
    elif 체결강도 < 95 and 체결강도평균(30) < 105:
        매도 = True
    # 연속 하락
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 25 / 100:
            매도 = True
    # 고가 대비 큰 하락
    elif (고가 - 현재가) / 고가 * 100 > 5 and 등락율각도(30) < 5:
        매도 = True
    # 시간 기준
    elif 보유시간 > 540:  # 9분 이상 보유 시 매도
        매도 = True

else:
    # 중대형주 매도 조건 - 모멘텀 약화 감지
    # 등락율 각도 기반 매도
    if 20 <= 등락율각도(30) and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 50 / 100:
        매도 = True
    elif 10 <= 등락율각도(30) < 20 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 45 / 100:
        매도 = True
    elif 0 <= 등락율각도(30) < 10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 45 / 100:
        매도 = True
    elif -10 <= 등락율각도(30) < 0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 35 / 100:
        매도 = True
    elif 등락율각도(30) < -10:
        매도 = True
    # 시가 아래로 하락
    elif 현재가 < 시가 and 수익률 < 2.5:
        매도 = True
    # 체결강도 약화
    elif 체결강도 < 105 and 체결강도평균(30) < 110:
        매도 = True
    # 연속 하락
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 30 / 100:
            매도 = True
    # 고가 대비 큰 하락
    elif (고가 - 현재가) / 고가 * 100 > 6 and 등락율각도(30) < 7:
        매도 = True
    # 시간 기준
    elif 보유시간 > 780:  # 13분 이상 보유 시 매도
        매도 = True
```

self.vars[58] = [1.4, 2.2]                             # 대형주 회전율 하한 (%)
self.vars[59] = [15, 21]                               # 대형주 고가위치 비율 (%)

# 세대 수 × 개체 수: 예) 20세대 × 50개체 = 1,000회

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
  - 14:30-15:00: 단기 스캘핑 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **동적 손절/익절 시스템**
  - 변동성 기반 손절폭 조정 (ATR 활용)
  - 트레일링 스톱 최적화
  - 구현 난이도: 중

- **부분 매도 전략**
  - 목표가 1차 도달 시 50% 매도
  - 나머지 50%는 트레일링 스톱 적용
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**

  - 일시적 스파이크 제거
  - 구현 난이도: 낮음

- **라운드 피겨 회피 강화**
  - 심리적 저항선 근처 진입 제한
  - 돌파 확인 후 진입
  - 구현 난이도: 낮음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **시가총액별 파라미터 세분화**

  - 구현 난이도: 중

- **백테스팅 기간 확대**

  - 구현 난이도: 낮음
