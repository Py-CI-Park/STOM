## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반(09:00~09:20) 거래량 폭발 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 시가총액 차등 (1,500억/3,000억 기준), 가격대 1,000원~40,000원
- **전략 타입**: 거래대금 급증 및 매수 폭발 모멘텀 포착 (Volume Explosion)
- **핵심 변수**: 초당거래대금, 초당매수수량, 거래대금배율, 매수폭발도, 체결강도, 등락율각도
- **업데이트 이력**:
  - 초기 문서 작성

---

## 전략 개요

### 핵심 아이디어
장 초반 초당거래대금이 평균 대비 급증하고, 초당매수수량이 폭발적으로 증가하는 순간을 포착합니다. 이는 큰 거래 주체의 진입을 의미하며 단기 급등 가능성이 높습니다.

### 주요 특징
1. **거래대금 비교**: 초당거래대금 vs 평균 배율 분석
2. **매수 폭발**: 초당매수수량의 급증 감지
3. **연속성 확인**: 여러 틱에 걸친 지속적 증가
4. **가격 모멘텀**: 등락율각도와 결합

## 조건식

### 매수 조건식

```python
# 파생 변수
거래대금배율 = 초당거래대금 / (초당거래대금평균(30) or 1)
매수폭발도 = 초당매수수량 / (누적초당매수수량(10) / 10 or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (1000 < 현재가 <= 40000):
    매수 = False
elif not (1.0 < 등락율 <= 28.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

elif 시가총액 < 1500:
    if not (거래대금배율 > 4.0):
        매수 = False
    elif not (매수폭발도 > 2.5):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) > 초당거래대금N(2)):
        매수 = False
    elif not (초당매수수량 > 최고초당매수수량(30, 1)):
        매수 = False
    elif not (체결강도 > 160):
        매수 = False
    elif not (등락율각도(20) > 8):
        매수 = False
    elif not (전일비 > 50):
        매수 = False
    elif not (당일거래대금 > 15 * 100):
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False
    elif not (최고초당매수수량(20) * 3.0 > 최고초당매도수량(20)):
        매수 = False

elif 시가총액 < 3000:
    if not (거래대금배율 > 3.5):
        매수 = False
    elif not (매수폭발도 > 2.2):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) > 초당거래대금N(2)):
        매수 = False
    elif not (초당매수수량 > 최고초당매수수량(30, 1)):
        매수 = False
    elif not (체결강도 > 145):
        매수 = False
    elif not (등락율각도(20) > 10):
        매수 = False
    elif not (전일비 > 60):
        매수 = False
    elif not (당일거래대금 > 40 * 100):
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False
    elif not (최고초당매수수량(20) * 2.5 > 최고초당매도수량(20)):
        매수 = False

else:
    if not (거래대금배율 > 3.0):
        매수 = False
    elif not (매수폭발도 > 2.0):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) > 초당거래대금N(2)):
        매수 = False
    elif not (초당매수수량 > 최고초당매수수량(30, 1)):
        매수 = False
    elif not (체결강도 > 135):
        매수 = False
    elif not (등락율각도(20) > 12):
        매수 = False
    elif not (전일비 > 70):
        매수 = False
    elif not (당일거래대금 > 70 * 100):
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False
    elif not (최고초당매수수량(20) * 2.0 > 최고초당매도수량(20)):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.0:
    매도 = True
elif 수익률 >= 5.0:
    매도 = True
elif 수익률 <= -4.0:
    매도 = True
elif 최고수익률 - 수익률 >= 3.0:
    매도 = True
elif 보유시간 > 240:
    매도 = True
elif 시가총액 < 1500:
    if 초당거래대금 < 초당거래대금평균(30) * 0.5:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(20) * 0.8:
        매도 = True
    elif 체결강도 < 70:
        매도 = True
    elif 현재가 < 이동평균(30) * 0.98:
        매도 = True
elif 시가총액 < 3000:
    if 초당거래대금 < 초당거래대금평균(30) * 0.6:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(20) * 0.7:
        매도 = True
    elif 체결강도 < 80:
        매도 = True
    elif 현재가 < 이동평균(30) * 0.98:
        매도 = True
else:
    if 초당거래대금 < 초당거래대금평균(30) * 0.7:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(20) * 0.6:
        매도 = True
    elif 체결강도 < 90:
        매도 = True
    elif 현재가 < 이동평균(30) * 0.98:
        매도 = True
```

## 최적화 조건식

### 매수 최적화 조건식

```python
거래대금배율 = 초당거래대금 / (초당거래대금평균(30) or 1)
매수폭발도 = 초당매수수량 / (누적초당매수수량(10) / 10 or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (self.vars[0] < 현재가 <= self.vars[1]):
    매수 = False
elif not (self.vars[2] < 등락율 <= self.vars[3]):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif 시가총액 < self.vars[4]:
    if not (거래대금배율 > self.vars[5]):
        매수 = False
    elif not (매수폭발도 > self.vars[6]):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) > 초당거래대금N(2)):
        매수 = False
    elif not (초당매수수량 > 최고초당매수수량(30, 1)):
        매수 = False
    elif not (체결강도 > self.vars[7]):
        매수 = False
    elif not (등락율각도(20) > self.vars[8]):
        매수 = False
    elif not (전일비 > self.vars[9]):
        매수 = False
    elif not (당일거래대금 > self.vars[10] * 100):
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False
    elif not (최고초당매수수량(20) * self.vars[11] > 최고초당매도수량(20)):
        매수 = False
```
