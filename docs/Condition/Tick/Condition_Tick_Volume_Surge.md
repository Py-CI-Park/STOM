## 소개

이 문서는 **거래량 급증 전략(Volume Surge Strategy)**을 설명합니다. 장 초반(09:00:00 ~ 09:20:00) 거래량이 급증하는 종목을 포착하여 매수하고, 빠르게 수익을 실현하는 전략입니다.

### 전략 개요
- **거래 시간**: 90000 ~ 92000 (9시 ~ 9시 20분)
- **목표**: 장 초반 거래량 급증 종목 포착
- **핵심 지표**: 초당거래대금, 체결강도, 누적초당매수수량
- **리스크 관리**: 빠른 손절/익절

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 틱 데이터 기반으로 1초 단위 거래 데이터를 분석합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `VS`: Volume Surge를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_VS_B`: 거래량 급증 전략의 매수 조건식
   - `C_VS_S`: 거래량 급증 전략의 매도 조건식

# Condition - Volume Surge

## 조건식

### 매수 조건식

```python
# Volume Surge Strategy - 거래량 급증 포착 매수 전략
# 거래 시간: 90000 ~ 92000 (9시 ~ 9시 20분)

매수 = False

# 시간 제한: 장 초반 20분만 거래
if not (90000 <= 시분초 < 92000):
    매수 = False

# 기본 필터링
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (2.0 < 등락율 <= 29.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 시가총액별 조건
elif 시가총액 < 3000:
    # 소형주 (3000억 미만)
    if not (초당거래대금 > 초당거래대금평균(30) * 3):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1)):
        매수 = False
    elif not (누적초당매수수량(30) > 누적초당매도수량(30) * 1.5):
        매수 = False
    elif not (체결강도 > 150):
        매수 = False
    elif not (체결강도평균(30) > 130):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2):
        매수 = False
    elif not (등락율각도(30) > 8):
        매수 = False
    elif not (전일비 > 150):
        매수 = False
    elif not (전일동시간비 > 200):
        매수 = False
    elif not (회전율 > 2):
        매수 = False
    elif not (당일거래대금 > 10 * 100):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 15 / 100):
        매수 = False
    elif not ((초당매수수량 - 초당매도수량) * 현재가 / 1_000_000 > 80):
        매수 = False

else:
    # 중대형주 (3000억 이상)
    if not (초당거래대금 > 초당거래대금평균(30) * 2.5):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1)):
        매수 = False
    elif not (누적초당매수수량(30) > 누적초당매도수량(30) * 1.3):
        매수 = False
    elif not (체결강도 > 140):
        매수 = False
    elif not (체결강도평균(30) > 120):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 1.5):
        매수 = False
    elif not (등락율각도(30) > 10):
        매수 = False
    elif not (전일비 > 100):
        매수 = False
    elif not (전일동시간비 > 150):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False
    elif not ((초당매수수량 - 초당매도수량) * 현재가 / 1_000_000 > 150):
        매수 = False
```

### 매도 조건식

```python
# Volume Surge Strategy - 빠른 익절/손절 매도 전략

매도 = False

# 상한가 근접 시 매도
if 등락율 > 29.5:
    매도 = True

# 수익률 기준 매도
elif 수익률 >= 4.0:
    매도 = True
elif 수익률 <= -3.0:
    매도 = True

# 최고수익률 대비 하락 시 매도
elif 최고수익률 >= 5.0 and 수익률 < 최고수익률 - 2.0:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 3000:
    # 소형주 매도 조건
    if 20 <= 등락율각도(30) and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 40 / 100:
        매도 = True
    elif 10 <= 등락율각도(30) < 20 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 60 / 100:
        매도 = True
    elif 0 <= 등락율각도(30) < 10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 40 / 100:
        매도 = True
    elif -10 <= 등락율각도(30) < 0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 20 / 100:
        매도 = True
    elif 등락율각도(30) < -10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 80 / 100:
        매도 = True
    elif 체결강도 < 90 and 체결강도N(1) > 120:
        매도 = True
    elif 보유시간 > 600:  # 10분 이상 보유 시 매도
        매도 = True

else:
    # 중대형주 매도 조건
    if 20 <= 등락율각도(30) and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 60 / 100:
        매도 = True
    elif 10 <= 등락율각도(30) < 20 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 50 / 100:
        매도 = True
    elif 0 <= 등락율각도(30) < 10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 50 / 100:
        매도 = True
    elif -10 <= 등락율각도(30) < 0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 30 / 100:
        매도 = True
    elif 등락율각도(30) < -10 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 70 / 100:
        매도 = True
    elif 체결강도 < 100 and 체결강도N(1) > 130:
        매도 = True
    elif 보유시간 > 900:  # 15분 이상 보유 시 매도
        매도 = True
```
