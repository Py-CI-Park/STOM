## 소개

이 문서는 RSI(Relative Strength Index) 지표를 활용한 과매도 구간 반등 전략입니다.
RSI가 과매도 구간(30 이하)에 진입한 후 반등하는 시점을 포착하여 매수하고,
과매수 구간(70 이상) 도달 또는 수익률 목표 달성 시 매도하는 역추세 전략입니다.

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 최적화 조건식은 변수의 범위를 설정하여 최적의 매수/매도 시점을 찾습니다.
- 최적화 범위는 self.vars[정수] = [[시작값, 종료값, 간격], 최적값] 형식으로 설정합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `T`: Tick을 의미합니다.
   - `RSI_REV`: RSI Reversal(반전) 전략을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.
   - `BO`: 매수 최적화 조건식을 의미합니다.
   - `SO`: 매도 최적화 조건식을 의미합니다.
   - `OR`: 매수 매도 통합 최적화 범위를 의미합니다.
   - `GA_R`: GA 최적화 범위를 의미합니다.

2. **예시**:
   - `C_T_RSI_REV_B`: RSI 반전 전략의 매수 조건식
   - `C_T_RSI_REV_S`: RSI 반전 전략의 매도 조건식
   - `C_T_RSI_REV_BO`: RSI 반전 전략의 매수 최적화 조건식
   - `C_T_RSI_REV_SO`: RSI 반전 전략의 매도 최적화 조건식

## 전략 개요

### 핵심 아이디어

1. **RSI 과매도 포착**: RSI가 30 이하로 내려간 후 상승 전환하는 시점 포착
2. **체결강도 확인**: 매수세가 매도세보다 강한 시점에만 진입
3. **거래대금 필터링**: 충분한 유동성이 있는 종목만 선택
4. **빠른 익절/손절**: 과매수 구간 도달 또는 목표 수익률 달성 시 신속 청산

### 매수 전략

- **시간대**: 09:00 ~ 15:00 (장중 거래)
- **가격대**: 1,000원 ~ 50,000원
- **등락율**: 0% ~ 29.5% (상한가 제외)
- **RSI 조건**: RSI_N(1) <= 30 and RSI > 32 (과매도 구간 반등)
- **체결강도**: 체결강도 >= 110 (매수세 우세)
- **거래대금**: 당일거래대금 >= 500억 (유동성 확보)
- **초당거래대금**: 초당거래대금 > 초당거래대금N(1) (거래 증가 확인)

### 매도 전략

- **목표 수익률**: 2.5% 익절
- **손절 수익률**: -1.5% 손절
- **RSI 과매수**: RSI >= 70 (과매수 구간 매도)
- **고점 대비 하락**: 최고수익률 - 수익률 >= 1.5% (고점 대비 하락)
- **VI 상한가**: 등락율 > 29.5% (상한가 근접 시 매도)
- **보유시간**: 최대 180초 (3분) 경과 시 강제 청산

## 조건식

### 매수 조건식 - C_T_RSI_REV_B

```python
# RSI 과매도 반등 전략 - 매수 조건식
# 작성일: 2025-01-17
# 전략: RSI 과매도 구간에서 반등 시 매수

# 공통 선결 조건
if not (시분초 >= 90000 and 시분초 < 150000):                               # 09:00~15:00 장중만 매수
    매수 = False
elif not (1000 < 현재가 <= 50000):                                           # 현재가 1,000원~50,000원 범위
    매수 = False
elif not (0 <= 등락율 <= 29.5):                                              # 등락율 0%~29.5% 범위 (상한가 제외)
    매수 = False
elif not (현재가 < VI아래5호가):                                              # VI 발동 방지
    매수 = False
elif 라운드피겨위5호가이내:                                                    # 라운드피겨 근처 회피
    매수 = False

# RSI 과매도 반등 조건
elif not (RSI_N(1) <= 30):                                                   # 이전 틱 RSI가 과매도 구간
    매수 = False
elif not (RSI > 32):                                                         # 현재 RSI가 반등 시작 (32 이상)
    매수 = False
elif not (RSI < 50):                                                         # RSI가 중립 구간 이하
    매수 = False

# 체결강도 및 거래량 조건
elif not (체결강도 >= 110):                                                   # 체결강도 110 이상 (매수세 우세)
    매수 = False
elif not (체결강도평균(30) > 100):                                            # 30틱 평균 체결강도 100 이상
    매수 = False
elif not (초당매수수량 > 초당매도수량 * 1.2):                                  # 매수 수량이 매도 수량의 120% 이상
    매수 = False

# 거래대금 조건
elif not (당일거래대금 >= 500 * 100):                                         # 당일거래대금 500억 이상
    매수 = False
elif not (초당거래대금 > 초당거래대금N(1)):                                    # 초당거래대금 증가 추세
    매수 = False
elif not (초당거래대금평균(30) > 50_000_000):                                 # 30틱 평균 초당거래대금 5천만 이상
    매수 = False

# 가격 위치 조건
elif not (현재가 > 저가):                                                     # 당일 저가보다 높은 가격
    매수 = False
elif not (현재가 < 고가 * 0.95):                                             # 당일 고가의 95% 이하
    매수 = False

# 등락율 각도 조건
elif not (등락율각도(30) > -10):                                             # 등락율 각도가 급락 상태 아님
    매수 = False
elif not (등락율각도(30) < 30):                                              # 등락율 각도가 급등 상태 아님
    매수 = False

# 시가총액 조건
elif not (시가총액 >= 1000):                                                  # 시가총액 1,000억 이상
    매수 = False
elif not (시가총액 <= 50000):                                                # 시가총액 5조 이하
    매수 = False

# 호가 잔량 조건
elif not (매수총잔량 > 매도총잔량 * 0.8):                                     # 매수 잔량이 매도 잔량의 80% 이상
    매수 = False
elif not (매도총잔량 * 현재가 / 100_000_000 < 20):                           # 매도 잔량 금액이 20억 미만
    매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_T_RSI_REV_S

```python
# RSI 과매도 반등 전략 - 매도 조건식
# 작성일: 2025-01-17
# 전략: RSI 과매수 도달, 목표 수익률 달성, 손절 조건 시 매도

# VI 상한가 근접 시 즉시 매도
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성 시 매도
elif 수익률 >= 2.5:
    매도 = True

# 손절 조건
elif 수익률 <= -1.5:
    매도 = True

# RSI 과매수 구간 진입 시 매도
elif RSI >= 70:
    매도 = True

# 고점 대비 하락 시 매도
elif 최고수익률 >= 1.5 and 최고수익률 - 수익률 >= 1.5:
    매도 = True

# 보유시간 초과 시 매도
elif 보유시간 > 180:                                                         # 3분(180초) 초과 보유 시 매도
    매도 = True

# 체결강도 급락 시 매도 (시가총액별 차별화)
elif 시가총액 < 2500:
    # 시가총액 2,500억 미만 종목
    if 수익률 > 0 and 체결강도 < 90 and 체결강도평균(30) < 95:
        매도 = True
    elif 초당매도수량 > 초당매수수량 * 2.0:                                   # 매도 수량이 매수 수량의 2배 초과
        매도 = True
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):                     # 2틱 연속 하락
        if 체결강도 < 95:
            매도 = True

else:
    # 시가총액 2,500억 이상 종목
    if 수익률 > 0 and 체결강도 < 95 and 체결강도평균(30) < 100:
        매도 = True
    elif 초당매도수량 > 초당매수수량 * 1.5:                                   # 매도 수량이 매수 수량의 1.5배 초과
        매도 = True
    elif 현재가 < 현재가N(1) and 현재가N(1) < 현재가N(2):                     # 2틱 연속 하락
        if 체결강도 < 100:
            매도 = True

# 최종 매도 실행
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

### 매수 최적화 조건식 - C_T_RSI_REV_BO

```python
# RSI 과매도 반등 전략 - 매수 최적화 조건식

# 공통 선결 조건
if not (시분초 >= 90000 and 시분초 < 150000):
    매수 = False
elif not (1000 < 현재가 <= self.vars[0]):                                    # 현재가 상한 최적화
    매수 = False
elif not (0 <= 등락율 <= 29.5):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# RSI 과매도 반등 조건
elif not (RSI_N(1) <= self.vars[1]):                                         # RSI 과매도 기준 최적화
    매수 = False
elif not (RSI > self.vars[2]):                                               # RSI 반등 기준 최적화
    매수 = False
elif not (RSI < 50):
    매수 = False

# 체결강도 및 거래량 조건
elif not (체결강도 >= self.vars[3]):                                         # 체결강도 기준 최적화
    매수 = False
elif not (체결강도평균(30) > self.vars[4]):                                  # 체결강도 평균 기준 최적화
    매수 = False
elif not (초당매수수량 > 초당매도수량 * self.vars[5]):                        # 매수/매도 비율 최적화
    매수 = False

# 거래대금 조건
elif not (당일거래대금 >= self.vars[6] * 100):                                # 당일거래대금 기준 최적화
    매수 = False
elif not (초당거래대금 > 초당거래대금N(1)):
    매수 = False
elif not (초당거래대금평균(30) > self.vars[7]):                               # 초당거래대금 평균 기준 최적화
    매수 = False

# 가격 위치 조건
elif not (현재가 > 저가):
    매수 = False
elif not (현재가 < 고가 * self.vars[8]):                                     # 고가 대비 비율 최적화
    매수 = False

# 등락율 각도 조건
elif not (등락율각도(30) > self.vars[9]):                                    # 등락율 각도 하한 최적화
    매수 = False
elif not (등락율각도(30) < self.vars[10]):                                   # 등락율 각도 상한 최적화
    매수 = False

# 시가총액 조건
elif not (시가총액 >= self.vars[11]):                                        # 시가총액 하한 최적화
    매수 = False
elif not (시가총액 <= self.vars[12]):                                        # 시가총액 상한 최적화
    매수 = False

# 호가 잔량 조건
elif not (매수총잔량 > 매도총잔량 * self.vars[13]):                          # 매수/매도 잔량 비율 최적화
    매수 = False
elif not (매도총잔량 * 현재가 / 100_000_000 < self.vars[14]):                # 매도 잔량 금액 상한 최적화
    매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매수 최적화 범위 - C_T_RSI_REV_BO_R
