# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 갭상승 지속 구간(09:00~09:20) 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 시가총액 차등 (2,500억 기준), 시가등락율 2% 이상 갭상승
- **전략 타입**: 갭상승 지속 (Gap Up Continuation)
- **핵심 변수**: 시가등락율, 시가대비등락율, 저가, 체결강도, 등락율각도, 매수총잔량, 매도총잔량
- **업데이트 이력**:
  - 초기 문서 작성: 갭상승 지속 전략

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
  - 14:30-15:00: 단기 스캘핑 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **동적 손절/익절 시스템**
  - 변동성 기반 손절폭 조정 (ATR 활용)
  - 트레일링 스톱 최적화
  - 구현 난이도: 중

- **부분 매도 전략**
  - 목표가 1차 도달 시 50% 매도
  - 나머지 50%는 트레일링 스톱 적용
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **허위 급등 필터링**
  - 급증 후 N초 지속 여부 확인
  - 일시적 스파이크 제거
  - 구현 난이도: 낮음

- **라운드 피겨 회피 강화**
  - 심리적 저항선 근처 진입 제한
  - 돌파 확인 후 진입
  - 구현 난이도: 낮음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **시가총액별 파라미터 세분화**
  - 초소형주 (1000억 미만) 별도 조건
  - 대형주 (1조 이상) 별도 조건
  - 구현 난이도: 중

- **백테스팅 기간 확대**
  - 다양한 시장 국면 반영
  - 과적합 방지
  - 구현 난이도: 낮음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 허위 급등 필터링
   - 호가창 두께 분석
   - 부분 매도 전략

2. **2단계 (1개월 내)**:
   - 거래대금 가속도 측정
   - 동적 손절/익절 시스템
   - 시간대별 전략 세분화

3. **3단계 (3개월 내)**:
   - 시가총액별 파라미터 세분화
   - 백테스팅 기간 확대
   - 복합 지표 개발

  - 시가등락율이 2% 이상 (갭상승)

   - `C_T_900_920_GU_B`: 900-920 갭상승 지속 패턴의 매수 조건식

# Condition - Tick Gap Up Continuation (갭상승 지속)

## 전략 개요

### 패턴 인식 로직

**갭상승 지속 (Gap Up Continuation) 정의**:
- 시가가 전일종가 대비 2% 이상 갭상승하여 시작
- 갭을 메우지 않고 추가 상승하는 패턴
- 저가가 시가 대비 -1% 이내 유지 (갭 유지)
- 현재가가 시가 대비 0.5% 이상 추가 상승

**갭 유지 확인**:
- 저가가 시가 대비 -1% 이내 (갭 메우지 않음)
- 시가등락율이 2% 이상 (명확한 갭상승)
- 현재가가 시가보다 높게 유지

**상승 지속 확인**:
- 시가대비등락율이 0.5% 이상
- 등락율각도가 5도 이상 상승
- 체결강도가 100 이상 유지
- 매수총잔량이 매도총잔량보다 많음

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 시가등락율 최소 3% 이상
- 시가대비등락율 최소 1% 이상
- 회전율 1.0% 이상

**대형주 (시가총액 >= 2500억)**:
- 시가등락율 최소 2% 이상
- 시가대비등락율 최소 0.5% 이상
- 회전율 0.6% 이상

## 조건식

### 매수 조건식 - C_T_900_920_GU_B

# ================================
#  공통 계산 지표
# ================================
전일종가          = 현재가 / (1 + (등락율 / 100))
시가등락율        = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
저가대비시가율    = ((저가 - 시가) / 시가) * 100
초당순매수금액    = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
갭유지여부        = (저가대비시가율 > -1.0)
상승지속여부      = (시가대비등락율 > 0.5 and 현재가 > 시가)


# ================================
#  매수 조건 (갭상승 지속)
# ================================
매수 = True


if not (관심종목 == 1):
    매수 = False


# ---------- 09:00:00 ~ 09:20:00 (갭상승 지속 패턴) ----------
elif 90000 <= 시분초 < 92000:
    # 기본 가격 및 시간 필터
    if not (1000 < 현재가 <= 50000):
        매수 = False
    elif not (2.5 < 등락율 <= 18.0):
        매수 = False
    elif not (고저평균대비등락율 > 0.0):
        매수 = False
    elif not (현재가 < VI아래5호가):
        매수 = False
    elif 라운드피겨위5호가이내:
        매수 = False
    elif not (데이터길이 >= 60):
        매수 = False

    # 갭상승 및 갭 유지 확인
    elif not (시가등락율 >= 2.0):
        매수 = False
    elif not 갭유지여부:
        매수 = False
    elif not 상승지속여부:
        매수 = False

    # 상승 추세 확인
    elif not (등락율각도(30) > 5):
        매수 = False
    elif not (당일거래대금각도(30) > 6):
        매수 = False
    elif not (현재가 > 현재가N(1)):
        매수 = False

    # 체결강도 및 호가 확인
    elif not (체결강도 >= 100 and 체결강도 <= 300):
        매수 = False
    elif not (매수총잔량 > 매도총잔량):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(60)):
        매수 = False

    # 시가총액별 세부 조건
    else:
        if 시가총액 < 2500:
            if not (시가등락율 >= 3.0):
                매수 = False
            elif not (시가대비등락율 >= 1.0 and 시가대비등락율 < 10.0):
                매수 = False
            elif not (15 < 초당순매수금액 < 2000):
                매수 = False
            elif not (전일비 > 80 and 전일동시간비 > 80):
                매수 = False
            elif not (회전율 > 1.0):
                매수 = False
            elif not (당일거래대금 > 200 * 100):
                매수 = False
            elif not (최고초당매수수량(60) > 최고초당매도수량(60)):
                매수 = False
            elif not (체결강도 > 체결강도평균(60)):
                매수 = False

        else:
            if not (시가등락율 >= 2.0):
                매수 = False
            elif not (시가대비등락율 >= 0.5 and 시가대비등락율 < 12.0):
                매수 = False
            elif not (30 < 초당순매수금액 < 3000):
                매수 = False
            elif not (전일비 > 60 and 전일동시간비 > 60):
                매수 = False
            elif not (회전율 > 0.6):
                매수 = False
            elif not (당일거래대금 > 350 * 100):
                매수 = False
            elif not (최고초당매수수량(60) > 최고초당매도수량(60) * 0.8):
                매수 = False
            elif not (체결강도 > 체결강도평균(60) * 0.95):
                매수 = False


# ---------- 09:20:00 이후 ----------
else:
    매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)

### 매도 조건식 - C_T_900_920_GU_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
갭메움여부        = (저가 < 시가 * 0.99)
상승동력소진      = (등락율각도(30) < 3 and 등락율각도(60) > 10)


# ================================
#  매도 조건 (갭상승 지속)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.0:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 5.0:
    매도 = True

# 손절 조건
elif 수익률 <= -3.5:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 7.0 and 최고수익률 * 0.70 >= 수익률:
    매도 = True

# 갭 메움 발생
elif 갭메움여부 and 수익률 > 0.5:
    매도 = True

# 상승 동력 소진
elif 상승동력소진 and 최고수익률 > 3.0:
    매도 = True

# 일반 매도 조건 (09:30:00 이전)
elif 시분초 < 93000:
    # 단기 이익 실현
    if 수익률 >= 3.5 and 보유시간 < 100:
        매도 = True

    # 체결강도 급락
    elif 체결강도 < 90 and 최고수익률 > 2.0:
        매도 = True

    # 시가총액별 매도 조건
    elif 시가총액 < 2500:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.45 and (현재가 / 현재가N(1) - 1) * 100 < -0.35:
            매도 = True
        elif 시가대비등락율 < 0.0 and 최고수익률 > 2.5:
            매도 = True
        elif 현재가 < 고가 * 0.97 and 최고수익률 > 4.5:
            매도 = True
    else:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.55 and (현재가 / 현재가N(1) - 1) * 100 < -0.4:
            매도 = True
        elif 시가대비등락율 < -0.3 and 최고수익률 > 2.0:
            매도 = True
        elif 현재가 < 고가 * 0.96 and 최고수익률 > 4.0:
            매도 = True

# 보유시간 초과
elif 보유시간 > 220:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
