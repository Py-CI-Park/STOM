# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 거래량 급증 돌파 구간(09:00~09:20) 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 시가총액 차등 (2,500억 기준), 등락율 2~15%
- **전략 타입**: 거래량 급증 돌파 (Volume Burst Breakout)
- **핵심 변수**: 초당거래대금평균(60), 체결강도, 초당매수수량, 매도총잔량, 전일비, 전일동시간비
- **업데이트 이력**:
  - 초기 문서 작성: 거래량 급증 돌파 전략

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **거래량 급증 돌파 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 09:20:00 (장 시작 즉시 ~ 20분)
- **패턴**: 거래량이 급증하면서 가격이 상승 돌파하는 형태
- **원리**: 장 시작 직후 대량 매수세가 유입되며 가격 급등
- **특징**:
  - 초당거래대금이 평균 대비 3배 이상 증가
  - 체결강도가 150 이상 강세
  - 매수총잔량이 매도총잔량을 압도
  - 전일비 및 전일동시간비가 100 이상

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `T`: Tick을 의미합니다.
   - `VB`: Volume Burst (거래량 급증)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_T_900_920_VB_B`: 900-920 거래량 급증 패턴의 매수 조건식
   - `C_T_900_920_VB_S`: 900-920 거래량 급증 패턴의 매도 조건식

# Condition - Tick Volume Burst (거래량 급증 돌파)

## 전략 개요

### 패턴 인식 로직

**거래량 급증 돌파 (Volume Burst Breakout) 정의**:
- 장 시작 직후 거래량이 급증하며 가격이 상승하는 패턴
- 초당거래대금이 평균 대비 3배 이상 증가
- 체결강도가 150 이상으로 강한 매수세 확인
- 매수총잔량이 매도총잔량보다 많아 추가 상승 여력 확보

**거래량 급증 확인**:
- 초당거래대금이 평균(60틱) 대비 3배 이상
- 초당매수수량이 매도총잔량의 60% 이상
- 전일비 및 전일동시간비가 100 이상 (전일 대비 거래량 증가)

**가격 상승 확인**:
- 등락율 2% 이상 15% 이하 (과도한 급등은 제외)
- 현재가가 시가보다 1% 이상 상승
- 현재가가 고가 대비 5% 이내 (고점 근처)

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 변동성이 크므로 더 강한 거래량 조건 요구
- 체결강도 최소 180 이상
- 초당순매수금액 최소 30백만원 이상
- 회전율 1.5% 이상

**대형주 (시가총액 >= 2500억)**:
- 더 완화된 거래량 조건
- 체결강도 최소 150 이상
- 초당순매수금액 최소 50백만원 이상
- 회전율 0.8% 이상

## 조건식

### 매수 조건식 - C_T_900_920_VB_B

```python
# ================================
#  공통 계산 지표
# ================================
전일종가          = 현재가 / (1 + (등락율 / 100))
시가등락율        = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
초당순매수금액    = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
거래대금폭증여부  = (초당거래대금 > 초당거래대금평균(60) * 3.0) if 데이터길이 > 60 else False
강력매수세여부    = (체결강도 >= 150 and 초당매수수량 > 매도총잔량 * 0.6)
고가근접여부      = ((고가 - 현재가) / 고가 * 100 < 5.0)


# ================================
#  매수 조건 (거래량 급증 돌파)
# ================================
매수 = True


if not (관심종목 == 1):
    매수 = False


# ---------- 09:00:00 ~ 09:20:00 (거래량 급증 패턴) ----------
elif 90000 <= 시분초 < 92000:
    # 기본 가격 및 시간 필터
    if not (1000 < 현재가 <= 50000):
        매수 = False
    elif not (2.0 < 등락율 <= 15.0):
        매수 = False
    elif not (시가대비등락율 > 1.0):
        매수 = False
    elif not (현재가 < VI아래5호가):
        매수 = False
    elif 라운드피겨위5호가이내:
        매수 = False
    elif not (데이터길이 >= 60):
        매수 = False

    # 거래량 급증 확인
    elif not 거래대금폭증여부:
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) * 1.2):
        매수 = False
    elif not (당일거래대금각도(30) > 8):
        매수 = False

    # 강력한 매수세 확인
    elif not 강력매수세여부:
        매수 = False
    elif not (매수총잔량 > 매도총잔량 * 1.3):
        매수 = False
    elif not (누적초당매수수량(30) > 누적초당매도수량(30) * 1.5):
        매수 = False

    # 가격 위치 확인
    elif not 고가근접여부:
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False

    # 시가총액별 세부 조건
    else:
        if 시가총액 < 2500:
            if not (-2.0 <= 시가등락율 < 8.0):
                매수 = False
            elif not (30 < 초당순매수금액 < 2000):
                매수 = False
            elif not (전일비 > 100 and 전일동시간비 > 100):
                매수 = False
            elif not (회전율 > 1.5):
                매수 = False
            elif not (당일거래대금 > 300 * 100):
                매수 = False
            elif not (체결강도 >= 180 and 체결강도 <= 400):
                매수 = False
            elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2.0):
                매수 = False

        else:
            if not (-1.0 <= 시가등락율 < 10.0):
                매수 = False
            elif not (50 < 초당순매수금액 < 3000):
                매수 = False
            elif not (전일비 > 80 and 전일동시간비 > 80):
                매수 = False
            elif not (회전율 > 0.8):
                매수 = False
            elif not (당일거래대금 > 500 * 100):
                매수 = False
            elif not (체결강도 >= 150 and 체결강도 <= 400):
                매수 = False
            elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 1.5):
                매수 = False


# ---------- 09:20:00 이후 ----------
else:
    매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_T_900_920_VB_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
거래량감소여부    = (초당거래대금 < 초당거래대금평균(30) * 0.5)
체결강도급락여부  = (체결강도 < 100 and 체결강도N(1) >= 100)


# ================================
#  매도 조건 (거래량 급증 돌파)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.0:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 6.0:
    매도 = True

# 손절 조건
elif 수익률 <= -3.0:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 8.0 and 최고수익률 * 0.65 >= 수익률:
    매도 = True

# 거래량 급감 시 매도
elif 거래량감소여부 and 최고수익률 > 3.0:
    매도 = True

# 체결강도 급락 시 매도
elif 체결강도급락여부 and 최고수익률 > 2.0:
    매도 = True

# 일반 매도 조건 (09:30:00 이전)
elif 시분초 < 93000:
    # 단기 이익 실현
    if 수익률 >= 4.0 and 보유시간 < 120:
        매도 = True

    # 시가총액별 매도 조건
    elif 시가총액 < 2500:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.5 and (현재가 / 현재가N(1) - 1) * 100 < -0.4:
            매도 = True
        elif 현재가 < 고가 * 0.96 and 최고수익률 > 5.0:
            매도 = True
    else:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.6 and (현재가 / 현재가N(1) - 1) * 100 < -0.5:
            매도 = True
        elif 현재가 < 고가 * 0.95 and 최고수익률 > 5.0:
            매도 = True

# 보유시간 초과
elif 보유시간 > 240:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
