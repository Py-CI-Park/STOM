## 소개

이 문서는 STOM 주식 자동거래 시스템에서 프라이스 액션 전략(Price Action Strategy)을 위한 조건식을 제공합니다. 가격 움직임 패턴과 고저가 관계를 분석하여 매매 신호를 포착하는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.

# Condition - Tick PriceAction

## 전략 개요

프라이스 액션 전략은 가격 움직임 패턴을 분석하여 매매 신호를 포착합니다:

- Higher Low 패턴 확인 (상승 추세)
- 고가 근접 매수 (강세 지속)
- 저가 반등 패턴 포착
- 가격 압축 후 확장 패턴

## 조건식

### 매수 조건식

```python
# ================================
#  공통 계산 지표
# ================================
최근고가              = 최고현재가(300) if 데이터길이 > 300 else 현재가
최근저가              = 최저현재가(300) if 데이터길이 > 300 else 현재가
이전고가              = 최고현재가(300, 300) if 데이터길이 > 600 else 현재가
이전저가              = 최저현재가(300, 300) if 데이터길이 > 600 else 현재가

# Higher Low 패턴 (최근 저가가 이전 저가보다 높음)
HigherLow여부         = (최근저가 > 이전저가) if 데이터길이 > 600 else False
# Higher High 패턴 (최근 고가가 이전 고가보다 높음)
HigherHigh여부        = (최근고가 > 이전고가) if 데이터길이 > 600 else False
상승추세여부          = (HigherLow여부 and HigherHigh여부)

고가근접도            = ((현재가 - 저가) / (고가 - 저가)) * 100 if (고가 - 저가) > 0 else 50
고가근접매수여부      = (고가근접도 >= 70)  # 고가 대비 70% 이상 위치

저가반등률            = ((현재가 - 최저현재가(60)) / 최저현재가(60)) * 100 if 데이터길이 > 60 else 0
저가반등여부          = (0.5 <= 저가반등률 <= 2.5)

거래량증가여부        = (초당거래대금 > 초당거래대금평균(120) * 1.5) if 데이터길이 > 120 else False
매수우위여부          = (초당매수수량 > 초당매도수량 * 1.4)

매수 = True

# 공통 필터
if not (관심종목 == 1):
    매수 = False
elif not (90500 <= 시분초 <= 143000):
    매수 = False
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 상승 추세 확인
elif not 상승추세여부:
    매수 = False

# 고가 근접 또는 저가 반등 확인 (둘 중 하나)
elif not (고가근접매수여부 or 저가반등여부):
    매수 = False

# 거래량 증가 확인
elif not 거래량증가여부:
    매수 = False

# 매수 우위 확인
elif not 매수우위여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (체결강도 >= 120):
        매수 = False
    elif not (회전율 > 1.8):
        매수 = False
    elif not (등락율각도(30) > 5):
        매수 = False
    elif not (현재가 > 이동평균(60)):
        매수 = False
    elif not (이동평균(60) > 이동평균(120)):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 2.0):
        매수 = False
elif 시가총액 < 5000:
    if not (체결강도 >= 110):
        매수 = False
    elif not (회전율 > 1.4):
        매수 = False
    elif not (등락율각도(30) > 4):
        매수 = False
    elif not (현재가 > 이동평균(60)):
        매수 = False
    elif not (이동평균(60) > 이동평균(120)):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 1.7):
        매수 = False
else:
    if not (체결강도 >= 105):
        매수 = False
    elif not (회전율 > 1.2):
        매수 = False
    elif not (등락율각도(30) > 3):
        매수 = False
    elif not (현재가 > 이동평균(60)):
        매수 = False
    elif not (이동평균(60) > 이동평균(120)):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 1.5):
        매수 = False

if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# ================================
#  공통 계산 지표
# ================================
추세전환신호          = (현재가 < 최저현재가(120, 60) and 현재가N(1) >= 최저현재가(120, 60)) if 데이터길이 > 180 else False
고가이탈신호          = (현재가 < 고가 * 0.98)  # 고가 대비 2% 하락
급락신호              = (등락율각도(30) < -10)

매도 = False

if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.0 or 수익률 <= -1.8:
    매도 = True
elif 최고수익률 > 3.5 and 최고수익률 * 0.70 >= 수익률:
    매도 = True
elif 보유시간 > 1200:  # 20분
    매도 = True
elif 추세전환신호:
    매도 = True
elif 고가이탈신호 and 보유시간 > 180:  # 3분 이후 고가 이탈
    매도 = True
elif 급락신호:
    매도 = True
elif 체결강도 < 95:
    매도 = True
elif 초당매도수량 > 초당매수수량 * 1.4:
    매도 = True
elif 시분초 >= 145500:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 조건식 개선 방향 연구

1) **패턴 인식 정교화**
   - Higher High, Higher Low 패턴 강화
   - Lower High, Lower Low 하락 패턴 회피
   - Swing High/Low 포인트 정의

2) **가격 위치 분석**
   - 고저가 범위 내 상대적 위치
   - Support/Resistance 레벨 활용
   - Fibonacci 되돌림 레벨 적용

3) **추세 강도 측정**
   - 추세선 각도 측정
   - 연속된 Higher Low 개수
   - 추세 지속 시간 평가

4) **거래량 프로파일**
   - 고가 근처 거래량 분석
   - 저가 근처 거래량 분석
   - Volume by Price 분석

5) **리스크 관리**
   - 추세 전환 신호 즉시 청산
   - 고가 이탈 시 청산 (3분 이후)
   - 트레일링 스톱 (최고수익의 70%)
