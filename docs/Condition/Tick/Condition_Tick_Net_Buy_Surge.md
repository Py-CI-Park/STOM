# 조건식 - 순매수 급증 전략

## 소개
**조건식 이름**: 순매수 급증 전략
**거래 시간**: 90000 ~ 92000
**목표**: 초당 순매수금액 급증 포착

## 조건식

### 매수 조건식
```python
순매수금액 = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
순매수비율 = 순매수금액 / (초당거래대금 / 1_000_000 or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (950 < 현재가 <= 39000):
    매수 = False
elif not (1.6 < 등락율 <= 27.8):
    매수 = False
elif not (순매수금액 > 80):
    매수 = False
elif not (순매수비율 > 0.6):
    매수 = False
elif 시가총액 < 1500:
    if not (순매수금액 > 순매수금액N(1) * 1.5):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.5):
        매수 = False
    elif not (체결강도 > 156):
        매수 = False
    elif not (등락율각도(19) > 8):
        매수 = False
    elif not (전일비 > 46):
        매수 = False
    elif not (당일거래대금 > 16 * 100):
        매수 = False
elif 시가총액 < 3000:
    if not (순매수금액 > 순매수금액N(1) * 1.3):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.2):
        매수 = False
    elif not (체결강도 > 146):
        매수 = False
    elif not (등락율각도(19) > 10):
        매수 = False
    elif not (전일비 > 56):
        매수 = False
    elif not (당일거래대금 > 42 * 100):
        매수 = False
else:
    if not (순매수금액 > 순매수금액N(1) * 1.2):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.0):
        매수 = False
    elif not (체결강도 > 136):
        매수 = False
    elif not (등락율각도(19) > 12):
        매수 = False
    elif not (전일비 > 66):
        매수 = False
    elif not (당일거래대금 > 68 * 100):
        매수 = False
```

### 매도 조건식
```python
순매도금액 = (초당매도수량 - 초당매수수량) * 현재가 / 1_000_000

if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 5.8:
    매도 = True
elif 수익률 <= -3.3:
    매도 = True
elif 최고수익률 - 수익률 >= 3.6:
    매도 = True
elif 보유시간 > 235:
    매도 = True
elif 순매도금액 > 60:
    if 체결강도 < 85:
        매도 = True
```
