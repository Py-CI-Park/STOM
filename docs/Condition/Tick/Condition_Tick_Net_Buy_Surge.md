# 조건식 - 순매수 급증 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 순매수 급증 구간(09:00~09:20) 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반)
- **대상 종목**: 가격대 950원~39,000원, 시가총액 차등 (1,500억/3,000억 기준)
- **전략 타입**: 순매수 급증 포착 (Net Buy Surge)
- **핵심 변수**: 순매수금액, 순매수비율, 초당매수수량, 누적초당매수수량(20), 체결강도, 등락율각도(19)
- **업데이트 이력**:
  - 초기 문서 작성: 순매수 급증 전략

## 소개
**조건식 이름**: 순매수 급증 전략
**거래 시간**: 90000 ~ 92000
**목표**: 초당 순매수금액 급증 포착

## 조건식

### 매수 조건식
```python
순매수금액 = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
순매수비율 = 순매수금액 / (초당거래대금 / 1_000_000 or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (950 < 현재가 <= 39000):
    매수 = False
elif not (1.6 < 등락율 <= 27.8):
    매수 = False
elif not (순매수금액 > 80):
    매수 = False
elif not (순매수비율 > 0.6):
    매수 = False
elif 시가총액 < 1500:
    if not (순매수금액 > 순매수금액N(1) * 1.5):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.5):
        매수 = False
    elif not (체결강도 > 156):
        매수 = False
    elif not (등락율각도(19) > 8):
        매수 = False
    elif not (전일비 > 46):
        매수 = False
    elif not (당일거래대금 > 16 * 100):
        매수 = False
elif 시가총액 < 3000:
    if not (순매수금액 > 순매수금액N(1) * 1.3):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.2):
        매수 = False
    elif not (체결강도 > 146):
        매수 = False
    elif not (등락율각도(19) > 10):
        매수 = False
    elif not (전일비 > 56):
        매수 = False
    elif not (당일거래대금 > 42 * 100):
        매수 = False
else:
    if not (순매수금액 > 순매수금액N(1) * 1.2):
        매수 = False
    elif not (초당매수수량 > 누적초당매수수량(20) / 20 * 2.0):
        매수 = False
    elif not (체결강도 > 136):
        매수 = False
    elif not (등락율각도(19) > 12):
        매수 = False
    elif not (전일비 > 66):
        매수 = False
    elif not (당일거래대금 > 68 * 100):
        매수 = False
```

### 매도 조건식
```python
순매도금액 = (초당매도수량 - 초당매수수량) * 현재가 / 1_000_000

if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 5.8:
    매도 = True
elif 수익률 <= -3.3:
    매도 = True
elif 최고수익률 - 수익률 >= 3.6:
    매도 = True
elif 보유시간 > 235:
    매도 = True
elif 순매도금액 > 60:
    if 체결강도 < 85:
        매도 = True
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
