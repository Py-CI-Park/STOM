# 조건식 - 등락율 모멘텀 가속 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 모멘텀 가속 구간(09:00~09:20) 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반)
- **대상 종목**: 가격대 1,100원~41,000원, 시가총액 차등 (1,500억/3,000억 기준)
- **전략 타입**: 등락율 모멘텀 가속 포착 (Momentum Acceleration)
- **핵심 변수**: 등락율가속도, 등락율변화, 등락율각도(17), 현재가(연속), 체결강도, 초당거래대금평균(23)
- **업데이트 이력**:
  - 초기 문서 작성: 모멘텀 가속 전략

## 소개
**조건식 이름**: 등락율 모멘텀 가속 전략
**거래 시간**: 90000 ~ 92000
**목표**: 등락율 가속도 증가로 상승 가속 포착

## 조건식

### 매수 조건식
```python
등락율변화 = 등락율 - 등락율N(1)
등락율가속도 = (등락율 - 등락율N(1)) - (등락율N(1) - 등락율N(2))

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (1100 < 현재가 <= 41000):
    매수 = False
elif not (2.0 < 등락율 <= 28.2):
    매수 = False
elif not (등락율가속도 > 0.15):
    매수 = False
elif not (등락율변화 > 0.08):
    매수 = False
elif 시가총액 < 1500:
    if not (등락율각도(17) > 등락율각도(17, 1) > 등락율각도(17, 2)):
        매수 = False
    elif not (현재가 > 현재가N(1) > 현재가N(2) > 현재가N(3)):
        매수 = False
    elif not (체결강도 > 164):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(23) * 3.0):
        매수 = False
    elif not (전일비 > 48):
        매수 = False
    elif not (당일거래대금 > 18 * 100):
        매수 = False
elif 시가총액 < 3000:
    if not (등락율각도(17) > 등락율각도(17, 1) > 등락율각도(17, 2)):
        매수 = False
    elif not (현재가 > 현재가N(1) > 현재가N(2) > 현재가N(3)):
        매수 = False
    elif not (체결강도 > 154):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(23) * 2.6):
        매수 = False
    elif not (전일비 > 58):
        매수 = False
    elif not (당일거래대금 > 44 * 100):
        매수 = False
else:
    if not (등락율각도(17) > 등락율각도(17, 1) > 등락율각도(17, 2)):
        매수 = False
    elif not (현재가 > 현재가N(1) > 현재가N(2) > 현재가N(3)):
        매수 = False
    elif not (체결강도 > 144):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(23) * 2.3):
        매수 = False
    elif not (전일비 > 68):
        매수 = False
    elif not (당일거래대금 > 70 * 100):
        매수 = False
```

### 매도 조건식
```python
등락율감속 = (등락율 - 등락율N(1)) - (등락율N(1) - 등락율N(2))

if 등락율 > 29.6:
    매도 = True
elif 수익률 >= 6.5:
    매도 = True
elif 수익률 <= -2.5:
    매도 = True
elif 최고수익률 - 수익률 >= 4.5:
    매도 = True
elif 보유시간 > 260:
    매도 = True
elif 등락율감속 < -0.12:
    if 체결강도 < 90:
        매도 = True
elif 등락율각도(17) < 등락율각도(17, 1) < 등락율각도(17, 2):
    매도 = True
```
