## 소개

이 문서는 STOM 주식 자동거래 시스템에서 모멘텀 반전 전략(Momentum Reversal Strategy)을 위한 조건식을 제공합니다. 급격한 하락 후 반등 신호를 포착하여 반전 매매를 수행하는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.

# Condition - Tick MomentumReversal

## 전략 개요

모멘텀 반전 전략은 급격한 가격 하락 후 반등 신호를 포착합니다:

- 단기 급락 후 반등 시작 포착
- 등락율 각도 급변 확인
- 저점 대비 반등률 측정
- 매수세 전환 확인

## 조건식

### 매수 조건식

```python
# ================================
#  공통 계산 지표
# ================================
급락여부              = (등락율각도(30) < -15 and 등락율각도(30, 1) < -10)
반등시작여부          = (등락율각도(30) > 5 and 등락율각도(30, 1) <= 0)
저점대비반등률        = ((현재가 - 최저현재가(60)) / 최저현재가(60)) * 100 if 데이터길이 > 60 else 0
매수세전환여부        = (초당매수수량 > 초당매도수량 * 1.5)
거래량증가여부        = (초당거래대금 > 초당거래대금평균(60) * 1.5) if 데이터길이 > 60 else False
하락폭충분여부        = (최고현재가(120) - 최저현재가(60)) / 최고현재가(120) * 100 > 2.0 if 데이터길이 > 120 else False

매수 = True

# 공통 필터
if not (관심종목 == 1):
    매수 = False
elif not (90500 <= 시분초 <= 145500):
    매수 = False
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 반등 시작 확인
elif not 반등시작여부:
    매수 = False

# 하락폭 충분 확인
elif not 하락폭충분여부:
    매수 = False

# 저점 대비 반등률 확인
elif not (0.5 <= 저점대비반등률 <= 3.0):
    매수 = False

# 매수세 전환 확인
elif not 매수세전환여부:
    매수 = False

# 거래량 증가 확인
elif not 거래량증가여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (현재가 > 최저현재가(60) * 1.005):
        매수 = False
    elif not (체결강도 >= 110):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (등락율각도(30) > 8):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 1.8):
        매수 = False
elif 시가총액 < 5000:
    if not (현재가 > 최저현재가(60) * 1.003):
        매수 = False
    elif not (체결강도 >= 105):
        매수 = False
    elif not (회전율 > 1.2):
        매수 = False
    elif not (등락율각도(30) > 6):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 1.5):
        매수 = False
else:
    if not (현재가 > 최저현재가(60) * 1.002):
        매수 = False
    elif not (체결강도 >= 100):
        매수 = False
    elif not (회전율 > 1.0):
        매수 = False
    elif not (등락율각도(30) > 5):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(120) * 1.3):
        매수 = False

if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# ================================
#  공통 계산 지표
# ================================
재하락신호            = (등락율각도(30) < -8)

if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 2.5 or 수익률 <= -1.5:
    매도 = True
elif 최고수익률 > 3.0 and 최고수익률 * 0.70 >= 수익률:
    매도 = True
elif 보유시간 > 600:  # 10분
    매도 = True
elif 재하락신호:
    매도 = True
elif 반등실패신호:
    매도 = True
elif 체결강도 < 95:
    매도 = True
elif 초당매도수량 > 초당매수수량 * 1.3:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 조건식 개선 방향 연구

1) **반등 강도 측정**
   - 반등 속도와 강도 정량화
   - 빠른 반등 vs 느린 반등 구분

2) **하락 패턴 분석**
   - 급락 vs 완만한 하락 구분
   - 하락 지속 시간 고려

3) **저점 신뢰도 평가**
   - 저점 테스트 횟수 확인
   - Double bottom 패턴 활용

4) **거래량 프로파일**
   - 하락 시 거래량 vs 반등 시 거래량
   - 거래량 증가율 추세 분석

5) **리스크 관리**
   - 짧은 보유시간 (10분 이내)
   - 재하락 신호 즉시 청산
   - 트레일링 스톱 적용
