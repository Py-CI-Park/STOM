## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **초반 급등 돌파 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 09:20:00 (장 시작 즉시 ~ 20분)
- **패턴**: 장 시작 직후 급등하여 전고점을 돌파하는 패턴
- **원리**: 장 시작과 동시에 강력한 매수세가 유입되어 가격 급등
- **특징**:
  - 데이터길이가 60틱 이내 (장 시작 1분 이내)
  - 등락율이 3% 이상 급등
  - 전일비 및 전일동시간비가 150 이상
  - 초당거래대금이 급격히 증가

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `T`: Tick을 의미합니다.
   - `EB`: Early Breakout (초반 급등 돌파)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_T_900_920_EB_B`: 900-920 초반 급등 돌파 패턴의 매수 조건식
   - `C_T_900_920_EB_S`: 900-920 초반 급등 돌파 패턴의 매도 조건식

# Condition - Tick Early Breakout (초반 급등 돌파)

## 전략 개요

### 패턴 인식 로직

**초반 급등 돌파 (Early Breakout) 정의**:
- 장 시작 직후 (60틱 이내) 급격한 가격 상승
- 등락율이 3% 이상 급등
- 전일 대비 거래량이 150% 이상 증가
- 초당거래대금이 급증하며 강한 매수세 확인

**초반 급등 확인**:
- 데이터길이 60틱 이내 (장 시작 1분 이내)
- 등락율 3% 이상
- 현재가가 시가 대비 2% 이상 상승

**거래량 폭증 확인**:
- 전일비 150% 이상
- 전일동시간비 150% 이상
- 초당거래대금이 급증 (이전 틱 대비 1.5배 이상)
- 당일거래대금이 일정 수준 이상

**강한 매수세 확인**:
- 체결강도 140 이상
- 초당매수수량 > 초당매도수량 * 2.0
- 매수총잔량 > 매도총잔량

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 등락율 최소 4% 이상
- 체결강도 최소 160 이상
- 당일거래대금 최소 400억 이상

**대형주 (시가총액 >= 2500억)**:
- 등락율 최소 3% 이상
- 체결강도 최소 140 이상
- 당일거래대금 최소 600억 이상

## 조건식

### 매수 조건식 - C_T_900_920_EB_B

```python
# ================================
#  공통 계산 지표
# ================================
전일종가          = 현재가 / (1 + (등락율 / 100))
시가등락율        = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
초당순매수금액    = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
초반급등여부      = (데이터길이 <= 60 and 등락율 >= 3.0 and 시가대비등락율 >= 2.0)
거래량폭증여부    = (전일비 >= 150 and 전일동시간비 >= 150)
강한매수세여부    = (체결강도 >= 140 and 초당매수수량 > 초당매도수량 * 2.0)


# ================================
#  매수 조건 (초반 급등 돌파)
# ================================
매수 = True


if not (관심종목 == 1):
    매수 = False


# ---------- 09:00:00 ~ 09:20:00 (초반 급등 돌파 패턴) ----------
elif 90000 <= 시분초 < 92000:
    # 기본 가격 및 시간 필터
    if not (1000 < 현재가 <= 50000):
        매수 = False
    elif not (3.0 < 등락율 <= 22.0):
        매수 = False
    elif not (현재가 < VI아래5호가):
        매수 = False
    elif 라운드피겨위5호가이내:
        매수 = False

    # 초반 급등 확인
    elif not 초반급등여부:
        매수 = False
    elif not (데이터길이 <= 60):
        매수 = False

    # 거래량 폭증 확인
    elif not 거래량폭증여부:
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1) * 1.5):
        매수 = False

    # 강한 매수세 확인
    elif not 강한매수세여부:
        매수 = False
    elif not (매수총잔량 > 매도총잔량):
        매수 = False

    # 가격 위치 확인
    elif not (현재가 > 고가 * 0.97):
        매수 = False

    # 시가총액별 세부 조건
    else:
        if 시가총액 < 2500:
            if not (등락율 >= 4.0):
                매수 = False
            elif not (시가대비등락율 >= 2.5):
                매수 = False
            elif not (40 < 초당순매수금액 < 3000):
                매수 = False
            elif not (회전율 > 1.5):
                매수 = False
            elif not (당일거래대금 > 400 * 100):
                매수 = False
            elif not (체결강도 >= 160 and 체결강도 <= 450):
                매수 = False
            elif not (초당매수수량 > 매도총잔량 * 0.7):
                매수 = False

        else:
            if not (등락율 >= 3.0):
                매수 = False
            elif not (시가대비등락율 >= 2.0):
                매수 = False
            elif not (60 < 초당순매수금액 < 4000):
                매수 = False
            elif not (회전율 > 1.0):
                매수 = False
            elif not (당일거래대금 > 600 * 100):
                매수 = False
            elif not (체결강도 >= 140 and 체결강도 <= 450):
                매수 = False
            elif not (초당매수수량 > 매도총잔량 * 0.6):
                매수 = False


# ---------- 09:20:00 이후 ----------
else:
    매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_T_900_920_EB_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
급등후조정여부    = (등락율각도(30) < 0 and 등락율 > 5)
거래량급감여부    = (초당거래대금 < 초당거래대금평균(30) * 0.4)


# ================================
#  매도 조건 (초반 급등 돌파)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 28.8:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 8.0:
    매도 = True

# 손절 조건
elif 수익률 <= -2.5:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 12.0 and 최고수익률 * 0.60 >= 수익률:
    매도 = True

# 급등 후 조정 시작
elif 급등후조정여부 and 최고수익률 > 5.0:
    매도 = True

# 거래량 급감
elif 거래량급감여부 and 최고수익률 > 3.5:
    매도 = True

# 일반 매도 조건 (09:30:00 이전)
elif 시분초 < 93000:
    # 단기 이익 실현
    if 수익률 >= 6.0 and 보유시간 < 80:
        매도 = True

    # 체결강도 급락
    elif 체결강도 < 110 and 최고수익률 > 4.0:
        매도 = True

    # 시가총액별 매도 조건
    elif 시가총액 < 2500:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.5 and (현재가 / 현재가N(1) - 1) * 100 < -0.5:
            매도 = True
        elif 현재가 < 고가 * 0.95 and 최고수익률 > 7.0:
            매도 = True
    else:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.6 and (현재가 / 현재가N(1) - 1) * 100 < -0.6:
            매도 = True
        elif 현재가 < 고가 * 0.94 and 최고수익률 > 6.0:
            매도 = True

# 보유시간 초과
elif 보유시간 > 180:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
