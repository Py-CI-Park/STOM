# 조건식 - 연속 매수틱 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 연속 매수틱 구간(09:00~09:20) 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반)
- **대상 종목**: 가격대 800원~35,000원, 시가총액 차등 (1,500억/3,000억 기준)
- **전략 타입**: 연속 매수틱 포착 (Continuous Buy Tick)
- **핵심 변수**: 연속상승틱, 연속매수우위, 체결강도(연속), 등락율각도(15), 이동평균(20), 최고초당매수수량(25)
- **업데이트 이력**:
  - 초기 문서 작성: 연속 매수틱 전략

## 소개
**조건식 이름**: 연속 매수틱 전략 (Continuous Buy Tick Strategy)
**거래 시간**: 90000 ~ 92000
**목표**: 연속적인 매수 체결 패턴 포착

## 전략 개요
장 초반 여러 틱에 걸쳐 지속적으로 현재가가 상승하고 초당매수수량이 매도수량을 압도하는 패턴을 감지합니다.

## 조건식

```python
    elif not (당일거래대금 > 8 * 100):
        매수 = False
    elif not (현재가 > 이동평균(20)):
        매수 = False
    elif not (최고초당매수수량(25) > 최고초당매도수량(25) * 2.5):

elif 시가총액 < 3000:
    if not (체결강도 > 145 and 체결강도N(1) > 135 and 체결강도N(2) > 125):

else:
    if not (체결강도 > 135 and 체결강도N(1) > 125 and 체결강도N(2) > 115):
        매수 = False
    elif not (등락율 > 등락율N(1) > 등락율N(2)):
```

### 매도 조건식
```python
연속하락틱 = (현재가 < 현재가N(1) < 현재가N(2))

if 등락율 > 29.2:
    매도 = True
elif 수익률 >= 5.5:
    매도 = True
elif 수익률 <= -3.2:
    매도 = True
elif 최고수익률 - 수익률 >= 3.5:
    매도 = True
elif 보유시간 > 220:
    매도 = True
elif 연속하락틱:
    if 초당매도수량 > 초당매수수량 * 1.5:
        매도 = True
elif 시가총액 < 1500:
    if 체결강도 < 체결강도N(1) < 체결강도N(2):
        매도 = True
    elif 현재가 < 이동평균(20) * 0.97:
        매도 = True
```
