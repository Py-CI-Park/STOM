## 소개

이 문서는 장초 09:10:00~09:15:00 구간에서 거래대금의 폭발적 증가를 감지하여 매수하는 전략입니다.
평균 대비 3배 이상의 거래대금이 발생할 때 강한 모멘텀으로 판단하고 진입합니다.

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `VE`: Volume Explosion을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_VE_B`: Volume Explosion 매수 조건식
   - `C_VE_S`: Volume Explosion 매도 조건식

### 매수 조건식

```python
# 거래대금 폭발 전략 (09:10:00 ~ 09:15:00)

# 시간 조건
if not (91000 <= 시분초 <= 91500):
    매수 = False

# 가격 및 등락율 조건
elif not (2000 < 현재가 <= 45000):
    매수 = False
elif not (1.0 < 등락율 <= 20.0):
    매수 = False

# VI 및 라운드피겨 제외
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 거래대금 폭발 조건 (핵심)
elif not (초당거래대금 > 초당거래대금평균(60) * 3.0):
    매수 = False
elif not (초당거래대금 > 초당거래대금N(1) * 1.2):
    매수 = False
elif not (초당거래대금 > 초당거래대금N(2) * 1.2):
    매수 = False

# 거래대금 각도 조건
elif not (당일거래대금각도(30) > 20):
    매수 = False

# 체결강도 조건
elif not (체결강도 > 120):
    매수 = False
elif not (체결강도평균(20) > 110):
    매수 = False

# 전일비 및 전일동시간비 조건
elif not (전일비 > 100):
    매수 = False
elif not (전일동시간비 > 150):
    매수 = False

# 시가총액별 세분화 조건
elif 시가총액 < 1500:
    # 초소형주: 매우 강한 폭발력 요구
    if not ((초당매수수량 - 초당매도수량) * 현재가 / 1_000_000 > 100):
        매수 = False
    elif not (등락율각도(30) > 18):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 4):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (회전율 > 3):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 10 / 100):
        매수 = False

elif 시가총액 < 3000:
    # 소형주
    if not ((초당매수수량 - 초당매도수량) * 현재가 / 1_000_000 > 120):
        매수 = False
    elif not (등락율각도(30) > 15):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 3):
        매수 = False
    elif not (당일거래대금 > 80 * 100):
        매수 = False
    elif not (회전율 > 2.5):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 15 / 100):
        매수 = False

else:
    # 중대형주
    if not ((초당매수수량 - 초당매도수량) * 현재가 / 1_000_000 > 150):
        매수 = False
    elif not (등락율각도(30) > 12):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2):
        매수 = False
    elif not (당일거래대금 > 150 * 100):
        매수 = False
    elif not (회전율 > 2):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# 폭발 후 급락 방어 전략
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.5:
    매도 = True
elif 수익률 <= -2.0:
    매도 = True
elif 최고수익률 - 수익률 >= 2.5:
    매도 = True
elif 보유시간 > 150:  # 2분 30초 초과 보유 시 청산
    매도 = True

# 거래대금 급감 조건
elif 초당거래대금 < 초당거래대금평균(30) * 0.5:
    if 수익률 > 0:
        매도 = True

# 체결강도 급락 조건
elif 체결강도 < 90 and 체결강도평균(20) > 110:
    매도 = True

# 시가총액별 세분화 매도 조건
elif 시가총액 < 1500:
    # 초소형주: 매우 빠른 청산
    if 1.0 <= 수익률 < 3.5 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.9:
        매도 = True
    elif 당일거래대금각도(20) < 5:
        매도 = True
    elif 등락율각도(20) < 8:
        매도 = True

elif 시가총액 < 3000:
    # 소형주
    if 1.0 <= 수익률 < 3.5 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.7:
        매도 = True
    elif 당일거래대금각도(20) < 8:
        매도 = True
    elif 등락율각도(20) < 10:
        매도 = True

else:
    # 중대형주
    if 1.0 <= 수익률 < 3.5 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.6:
        매도 = True
    elif 당일거래대금각도(20) < 10:
        매도 = True
    elif 등락율각도(20) < 11:
        매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
  - 14:30-15:00: 단기 스캘핑 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **동적 손절/익절 시스템**
  - 변동성 기반 손절폭 조정 (ATR 활용)
  - 트레일링 스톱 최적화
  - 구현 난이도: 중

- **부분 매도 전략**
  - 목표가 1차 도달 시 50% 매도
  - 나머지 50%는 트레일링 스톱 적용
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **허위 급등 필터링**
  - 급증 후 N초 지속 여부 확인
  - 일시적 스파이크 제거
  - 구현 난이도: 낮음

- **라운드 피겨 회피 강화**
  - 심리적 저항선 근처 진입 제한
  - 돌파 확인 후 진입
  - 구현 난이도: 낮음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **시가총액별 파라미터 세분화**
  - 초소형주 (1000억 미만) 별도 조건
  - 대형주 (1조 이상) 별도 조건
  - 구현 난이도: 중

- **백테스팅 기간 확대**
  - 다양한 시장 국면 반영
  - 과적합 방지
  - 구현 난이도: 낮음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 허위 급등 필터링
   - 호가창 두께 분석
   - 부분 매도 전략

2. **2단계 (1개월 내)**:
   - 거래대금 가속도 측정
   - 동적 손절/익절 시스템
   - 시간대별 전략 세분화

3. **3단계 (3개월 내)**:
   - 시가총액별 파라미터 세분화
   - 백테스팅 기간 확대
   - 복합 지표 개발
