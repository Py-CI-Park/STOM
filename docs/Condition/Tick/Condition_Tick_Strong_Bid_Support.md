## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반(09:00~09:20) 강력 매수호가 지지 패턴 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 시가총액 차등 (3,000억/10,000억 기준), 가격대 필터링 적용
- **전략 타입**: 매수호가 집중 + 호가잔량 우위 포착 (Strong Bid Support)
- **핵심 변수**: 매수총잔량, 매도총잔량, 매수잔량1~3, 초당매수수량, 체결강도
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **강력 매수호가 지지 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 09:20:00 (장 시작 즉시 ~ 20분)
- **패턴**: 매수호가에 강력한 지지가 형성되는 패턴
- **원리**: 대량 매수 주문이 호가창에 쌓여 가격 지지
- **특징**:
  - 매수총잔량이 매도총잔량의 2배 이상
  - 매수5호가잔량합이 일정 규모 이상
  - 매수잔량1~3의 합이 전체 매수잔량의 60% 이상 (집중도)
  - 초당매수수량이 지속적으로 매도수량 초과

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `T`: Tick을 의미합니다.
   - `SBS`: Strong Bid Support (강력 매수호가 지지)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_T_900_920_SBS_B`: 900-920 강력 매수호가 지지 패턴의 매수 조건식
   - `C_T_900_920_SBS_S`: 900-920 강력 매수호가 지지 패턴의 매도 조건식

# Condition - Tick Strong Bid Support (강력 매수호가 지지)

## 전략 개요

### 패턴 인식 로직

**강력 매수호가 지지 (Strong Bid Support) 정의**:
- 매수호가에 대량 주문이 집중되어 가격 하락 방어
- 매수총잔량이 매도총잔량의 2배 이상
- 매수잔량이 하위 1~3호가에 60% 이상 집중
- 초당매수수량이 초당매도수량보다 지속적으로 많음

**호가 집중도 확인**:
- 매수잔량1~3의 합이 매수5호가잔량합의 60% 이상
- 매수호가1의 잔량이 일정 규모 이상
- 매수총잔량 * 현재가가 일정 금액 이상

**매수세 우위 확인**:
- 체결강도가 110 이상
- 누적초당매수수량(60) > 누적초당매도수량(60)
- 초당매수수량 > 매도총잔량 * 0.4

### 시장 상황별 전략

**소형주 (시가총액 < 3000억)**:
- 매수총잔량 * 현재가 > 3억원
- 매수총잔량 > 매도총잔량 * 2.5배
- 체결강도 최소 130 이상

**중형주 (3000억 <= 시가총액 < 10000억)**:
- 매수총잔량 * 현재가 > 5억원
- 매수총잔량 > 매도총잔량 * 2.0배
- 체결강도 최소 110 이상

**대형주 (시가총액 >= 10000억)**:
- 매수총잔량 * 현재가 > 10억원
- 매수총잔량 > 매도총잔량 * 1.8배
- 체결강도 최소 100 이상

## 조건식

### 매수 조건식 - C_T_900_920_SBS_B

```python
# ================================
#  공통 계산 지표
# ================================
전일종가          = 현재가 / (1 + (등락율 / 100))
시가등락율        = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
초당순매수금액    = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
매수호가집중도    = ((매수잔량1 + 매수잔량2 + 매수잔량3) / 매수5호가잔량합 * 100) if 매수5호가잔량합 > 0 else 0
매수총잔량금액    = 매수총잔량 * 현재가 / 100_000_000
강력매수호가여부  = (매수총잔량 > 매도총잔량 * 2.0 and 매수호가집중도 > 60)


# ================================
#  매수 조건 (강력 매수호가 지지)
# ================================
매수 = True


if not (관심종목 == 1):
    매수 = False


# ---------- 09:00:00 ~ 09:20:00 (강력 매수호가 지지 패턴) ----------
elif 90000 <= 시분초 < 92000:
    # 기본 가격 및 시간 필터
    if not (1000 < 현재가 <= 50000):
        매수 = False
    elif not (1.0 < 등락율 <= 18.0):
        매수 = False
    elif not (고저평균대비등락율 > -2.0):
        매수 = False
    elif not (현재가 < VI아래5호가):
        매수 = False
    elif 라운드피겨위5호가이내:
        매수 = False
    elif not (데이터길이 >= 60):
        매수 = False

    # 강력한 매수호가 확인
    elif not 강력매수호가여부:
        매수 = False
    elif not (매수호가집중도 > 60):
        매수 = False
    elif not (매수잔량1 > 0):
        매수 = False

    # 매수세 우위 확인
    elif not (체결강도 >= 110):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 0.4):
        매수 = False
    elif not (누적초당매수수량(60) > 누적초당매도수량(60)):
        매수 = False

    # 가격 상승 확인
    elif not (등락율각도(30) > 3):
        매수 = False
    elif not (현재가 > 현재가N(1)):
        매수 = False

    # 시가총액별 세부 조건
    else:
        if 시가총액 < 3000:
            if not (-1.0 <= 시가등락율 < 10.0):
                매수 = False
            elif not (0.5 <= 시가대비등락율 < 12.0):
                매수 = False
            elif not (15 < 초당순매수금액 < 2000):
                매수 = False
            elif not (전일비 > 60 and 전일동시간비 > 60):
                매수 = False
            elif not (회전율 > 1.0):
                매수 = False
            elif not (당일거래대금 > 200 * 100):
                매수 = False
            elif not (매수총잔량금액 > 3):
                매수 = False
            elif not (매수총잔량 > 매도총잔량 * 2.5):
                매수 = False
            elif not (체결강도 >= 130):
                매수 = False

        elif 시가총액 < 10000:
            if not (-0.5 <= 시가등락율 < 12.0):
                매수 = False
            elif not (1.0 <= 시가대비등락율 < 14.0):
                매수 = False
            elif not (25 < 초당순매수금액 < 3000):
                매수 = False
            elif not (전일비 > 50 and 전일동시간비 > 50):
                매수 = False
            elif not (회전율 > 0.7):
                매수 = False
            elif not (당일거래대금 > 350 * 100):
                매수 = False
            elif not (매수총잔량금액 > 5):
                매수 = False
            elif not (매수총잔량 > 매도총잔량 * 2.0):
                매수 = False
            elif not (체결강도 >= 110):
                매수 = False

        else:
            if not (0.0 <= 시가등락율 < 15.0):
                매수 = False
            elif not (1.5 <= 시가대비등락율 < 16.0):
                매수 = False
            elif not (40 < 초당순매수금액 < 4000):
                매수 = False
            elif not (전일비 > 40 and 전일동시간비 > 40):
                매수 = False
            elif not (회전율 > 0.5):
                매수 = False
            elif not (당일거래대금 > 500 * 100):
                매수 = False
            elif not (매수총잔량금액 > 10):
                매수 = False
            elif not (매수총잔량 > 매도총잔량 * 1.8):
                매수 = False
            elif not (체결강도 >= 100):
                매수 = False


# ---------- 09:20:00 이후 ----------
else:
    매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_T_900_920_SBS_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
매수호가약화여부  = (매도총잔량 > 매수총잔량)
호가지지붕괴여부  = ((매수잔량1 + 매수잔량2 + 매수잔량3) / 매수5호가잔량합 * 100 < 50) if 매수5호가잔량합 > 0 else True


# ================================
#  매도 조건 (강력 매수호가 지지)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.0:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 5.5:
    매도 = True

# 손절 조건
elif 수익률 <= -2.8:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 8.0 and 최고수익률 * 0.68 >= 수익률:
    매도 = True

# 매수호가 지지 약화
elif 매수호가약화여부 and 최고수익률 > 2.5:
    매도 = True

# 호가 집중도 붕괴
elif 호가지지붕괴여부 and 최고수익률 > 3.0:
    매도 = True

# 일반 매도 조건 (09:30:00 이전)
elif 시분초 < 93000:
    # 단기 이익 실현
    if 수익률 >= 4.0 and 보유시간 < 110:
        매도 = True

    # 체결강도 급락
    elif 체결강도 < 95 and 최고수익률 > 2.5:
        매도 = True

    # 시가총액별 매도 조건
    elif 시가총액 < 10000:
        if (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.4 and (현재가 / 현재가N(1) - 1) * 100 < -0.35:
            매도 = True
        elif 매도총잔량 > 매수총잔량 * 1.2 and 최고수익률 > 3.0:
            매도 = True
        elif 현재가 < 고가 * 0.97 and 최고수익률 > 5.0:
            매도 = True

# 보유시간 초과
elif 보유시간 > 210:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
