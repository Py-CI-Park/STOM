# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **이동평균 돌파 스캘핑 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 시작 20분)
- **대상 종목**: 전 종목 대상
- **전략 타입**: 이동평균선 돌파 스캘핑 전략
- **핵심 변수**: 이동평균(60틱), 초당거래대금, 체결강도
- **업데이트 이력**:
  - 초기 문서 작성: 이동평균 돌파 스캘핑 전략

## 소개

이 문서는 장초 09:00:00~09:20:00 전체 구간에서 이동평균선 돌파를 활용한 스캘핑 전략입니다.
단기 이동평균(60틱)을 상향 돌파하면서 거래량이 동반될 때 진입하여 빠른 수익을 실현합니다.

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `MABS`: MA Breakout Scalping을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_MABS_B`: MA Breakout Scalping 매수 조건식
   - `C_MABS_S`: MA Breakout Scalping 매도 조건식

## 조건식

### 매수 조건식

```python
# ================================
#  공통 계산 지표
# ================================
초당순매수금액 = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000   # 단위: 백만원

# ================================
#  매수 조건
# ================================
매수 = True

# 시간 조건
if not (90000 <= 시분초 <= 92000):
    매수 = False

# 데이터 충분성 확인 (이동평균 계산을 위해)
elif not (데이터길이 > 90):
    매수 = False

# 가격 및 등락율 조건
elif not (2000 < 현재가 <= 50000):
    매수 = False
elif not (1.0 < 등락율 <= 18.0):
    매수 = False

# VI 및 라운드피겨 제외
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 이동평균 돌파 조건 (핵심)
# 현재가가 60틱 이동평균을 상향 돌파
elif not (현재가 > 이동평균(60)):
    매수 = False
elif not (현재가N(1) <= 이동평균(60, 1) or 현재가N(2) <= 이동평균(60, 2)):
    매수 = False

# 이동평균 상승 추세 확인
elif not (이동평균(60) > 이동평균(60, 1)):
    매수 = False
elif not (이동평균(60, 1) > 이동평균(60, 2)):
    매수 = False

# 단기 이평 > 장기 이평 (정배열)
elif not (이동평균(30) > 이동평균(90)):
    매수 = False

# 거래량 동반 조건
elif not (초당거래대금 > 초당거래대금평균(60) * 1.5):
    매수 = False
elif not (초당거래대금 > 초당거래대금N(1)):
    매수 = False

# 체결강도 조건
elif not (체결강도 > 130):
    매수 = False
elif not (체결강도평균(30) > 115):
    매수 = False

# 상승 각도 조건
elif not (등락율각도(30) > 8):
    매수 = False

# 시가총액별 세분화 조건
elif 시가총액 < 2500:
    # 소형주
    if not (초당순매수금액 > 60):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2.5):
        매수 = False
    elif not (당일거래대금 > 30 * 100):
        매수 = False
    elif not (전일비 > 70):
        매수 = False
    elif not (전일동시간비 > 100):
        매수 = False
    elif not (회전율 > 2):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False
    elif not (매수총잔량 > 매도총잔량):
        매수 = False

else:
    # 중대형주
    if not (초당순매수금액 > 100):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2.0):
        매수 = False
    elif not (당일거래대금 > 80 * 100):
        매수 = False
    elif not (전일비 > 50):
        매수 = False
    elif not (전일동시간비 > 80):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 30 / 100):
        매수 = False
    elif not (매수총잔량 > 매도총잔량 * 0.8):
        매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# ================================
#  매도 조건
# ================================
매도 = False

# 이동평균 이탈 및 익절 전략
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.0:
    매도 = True
elif 수익률 <= -2.5:
    매도 = True
elif 최고수익률 - 수익률 >= 2.0:
    매도 = True
elif 보유시간 > 200:  # 3분 20초 초과 보유 시 청산
    매도 = True

# 이동평균 하향 이탈 (핵심 매도 조건)
elif 현재가 < 이동평균(60) * 0.995:
    if 수익률 > 0:
        매도 = True
elif 현재가N(1) >= 이동평균(60, 1) and 현재가 < 이동평균(60):
    if 수익률 > 0.3:
        매도 = True

# 이동평균 하락 전환
elif 이동평균(60) < 이동평균(60, 1) and 이동평균(60, 1) < 이동평균(60, 2):
    if 수익률 > 0.5:
        매도 = True

# 체결강도 약화
elif 체결강도 < 100:
    매도 = True
elif 체결강도평균(20) < 105:
    if 수익률 > 0.5:
        매도 = True

# 시가총액별 세분화 매도 조건
elif 시가총액 < 2500:
    # 소형주: 빠른 청산
    if 1.0 <= 수익률 < 3.0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.7:
        매도 = True
    elif 초당거래대금 < 초당거래대금평균(30) * 0.6:
        매도 = True
    elif 등락율각도(20) < 3:
        매도 = True

else:
    # 중대형주
    if 1.0 <= 수익률 < 3.0 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.6:
        매도 = True
    elif 초당거래대금 < 초당거래대금평균(30) * 0.7:
        매도 = True
    elif 등락율각도(20) < 5:
        매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **호가창 두께 분석 추가**
  - 매수/매도 잔량 비율을 통한 지지/저항 강도 측정
  - 얇은 호가창 종목 필터링으로 급등락 리스크 감소
  - 구현 난이도: 낮음

- **거래대금 가속도 측정**
  - 1차 미분, 2차 미분 계산으로 거래량 가속 감지
  - 가속도가 양수일 때 매수 신호 강화
  - 구현 난이도: 중

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 초반/중반/후반 조건 차등 적용**
  - 09:00-09:30: 갭 상승 전략 강화
  - 10:00-14:00: 모멘텀 추종 전략
  - 14:30-15:00: 단기 스캘핑 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **동적 손절/익절 시스템**
  - 변동성 기반 손절폭 조정 (ATR 활용)
  - 트레일링 스톱 최적화
  - 구현 난이도: 중

- **부분 매도 전략**
  - 목표가 1차 도달 시 50% 매도
  - 나머지 50%는 트레일링 스톱 적용
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **허위 급등 필터링**
  - 급증 후 N초 지속 여부 확인
  - 일시적 스파이크 제거
  - 구현 난이도: 낮음

- **라운드 피겨 회피 강화**
  - 심리적 저항선 근처 진입 제한
  - 돌파 확인 후 진입
  - 구현 난이도: 낮음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **시가총액별 파라미터 세분화**
  - 초소형주 (1000억 미만) 별도 조건
  - 대형주 (1조 이상) 별도 조건
  - 구현 난이도: 중

- **백테스팅 기간 확대**
