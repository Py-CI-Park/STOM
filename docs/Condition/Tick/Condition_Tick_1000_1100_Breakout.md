## 개요

본 문서는 STOM 주식 자동거래 시스템에서 ****을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:00:00 (장중 전체)
- **대상 종목**: 관심종목, 시가총액 차등 기준
- **전략 타입**: Tick 기반 단타 전략
- **핵심 변수**: 체결강도, 초당거래대금, 등락율각도, 회전율
- **업데이트 이력**:
  - 초기 문서 작성: 

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 10:00:00 ~ 11:00:00 구간의 돌파 전략(Breakout)을 위한 조건식과 최적화 범위를 제공합니다. 장 시작 1시간 후 안정화된 구간에서 나타나는 상승 돌파 패턴을 포착하여 진입하는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.
- 최적화 조건식은 `self.vars[i]`로 치환하여 탐색 범위를 별도로 정의합니다.
- 최적화 범위 포맷: `self.vars[정수] = [[시작값, 종료값, 간격], 최적값]`

### 저장 이름 규칙

1. 기본 규칙:
   - `C`: Condition
   - `T`: Tick
   - `1000_1100`: 시간대
   - `BO`: Breakout
   - `B`/`S`: 매수/매도 조건식
   - `BO`/`SO`: 매수/매도 최적화 조건식
   - `BOR`/`SOR`: 매수/매도 최적화 범위
   - `OR`: 매수/매도 통합 최적화 범위
   - `GAR`: GA 최적화 범위

# Condition - Tick 1000_1100 Breakout

## 전략 개요

장 시작 1시간 후(10:00~11:00)는 초반 변동성이 안정화되고, 진정한 방향성이 나타나는 구간입니다. 이 시간대에 다음과 같은 패턴을 포착합니다:

- 최근 고가 돌파를 통한 상승 모멘텀 확인
- 안정적인 거래량 증가 패턴
- 체결강도 지속적 상승
- 이동평균선 정배열 확인

## 조건식

### 매수 조건식 - C_T_1000_1100_BO_B

```python
이평정배열여부        = (이동평균(60) > 이동평균(120) > 이동평균(180))
거래대금증가율        = (초당거래대금 / 초당거래대금평균(180)) if 초당거래대금평균(180) != 0 else 0
체결강도상승여부      = (체결강도 > 체결강도평균(30) and 체결강도평균(30) > 체결강도평균(60))

# ================================
#  매수 조건
# ================================

    매수 = False

# 고가 돌파 확인
elif not 고가돌파여부:
    매수 = False

# 이동평균선 정배열 확인
elif not 이평정배열여부:
    매수 = False

# 거래 활성화 확인
elif not (거래대금증가율 >= 1.5):
    매수 = False

# 체결강도 상승 추세 확인
elif not 체결강도상승여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (0.5 < 등락율 <= 15.0):
        매수 = False

        매수 = False

        매수 = False
    elif not (전일비 > 50):
        매수 = False
    elif not (체결강도 >= 100 and 체결강도 <= 400):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 0.30):
        매수 = False
    elif not (등락율각도(60) > 0):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
elif 시가총액 < 5000:
    if not (0.3 < 등락율 <= 12.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.40)):
        매수 = False
    elif not (회전율 > 1.0):
        매수 = False
    elif not (전일비 > 30):
        매수 = False
    elif not (체결강도 >= 90 and 체결강도 <= 400):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 0.25):
        매수 = False
    elif not (등락율각도(60) > 0):
        매수 = False
    elif not (당일거래대금 > 100 * 100):
        매수 = False
else:
    if not (0.2 < 등락율 <= 10.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.50)):
        매수 = False
    elif not (회전율 > 0.5):
        매수 = False
    elif not (전일비 > 20):
        매수 = False
    elif not (체결강도 >= 80 and 체결강도 <= 400):
        매수 = False
    elif not (초당매수수량 > 매도총잔량 * 0.20):
        매수 = False
    elif not (등락율각도(60) > 0):
        매수 = False

```

### 매도 조건식 - C_T_1000_1100_BO_S

```python
# ================================
#  공통 계산 지표
# ================================
단기하락률            = (현재가 / max(현재가N(1), 1) - 1) * 100

급락신호              = (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.8 and 단기하락률 < -1.0
체결강도약화여부      = (체결강도 < 체결강도평균(30) * 0.7)

매도 = False

# 상한가 부근 안전장치
if 등락율 > 29.5:
    매도 = True

# 손절 및 목표가 도달
elif 수익률 >= 3.0 or 수익률 <= -3.0:
    매도 = True

# 트레일링 스톱
elif 최고수익률 > 4.0 and 최고수익률 * 0.65 >= 수익률:
    매도 = True

# 보유시간 기반 청산
elif 보유시간 > 1800:  # 30분
    매도 = True

# 급락 신호
elif 급락신호:
    매도 = True

# 이동평균선 이탈
elif 이평이탈여부:
    매도 = True

# 체결강도 약화
elif 체결강도약화여부:
    매도 = True

# 등락율 각도 하락 전환
elif 등락율각도(30) < -5:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

### 매도 최적화 조건식 - C_T_1000_1100_BO_SO

```python
단기하락률            = (현재가 / max(현재가N(1), 1) - 1) * 100
이평이탈여부          = (현재가 < 이동평균(60)) or (이동평균(60) < 이동평균(60, 1))
급락신호              = (초당매도수량 - 초당매수수량) >= 매수총잔량 * self.vars[32] and 단기하락률 < -self.vars[33]
체결강도약화여부      = (체결강도 < 체결강도평균(30) * self.vars[34])

매도 = False
```

   - 가속도가 양수일 때 매수 신호 강화

   - 과열 구간 진입 방지

   - RSI 50~70 구간에서만 진입
   - 과매수 구간 진입 방지
