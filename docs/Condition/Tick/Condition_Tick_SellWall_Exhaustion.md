# 조건식 - 매도벽 소진 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 소개

**조건식 이름**: 매도벽 소진 전략 (Sell Wall Exhaustion Strategy)

이 문서는 매도잔량의 급격한 감소를 통해 상승 압력을 포착하는 전략을 설명합니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **매도벽 소진 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반 20분)
- **대상 종목**: 시가총액 차등 기준 (1,500억, 3,000억)
- **전략 타입**: 매도벽 소진 포착 전략
- **핵심 변수**: 매도잔량감소율, 매도벽비율, 초당매수수량
- **업데이트 이력**:
  - 초기 문서 작성: 매도벽 소진 전략

## 조건식

### 매수 조건식
```python
매도잔량감소율 = (매도총잔량N(1) - 매도총잔량) / (매도총잔량N(1) or 1)
매도벽비율 = 매도총잔량 / 매수총잔량

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (900 < 현재가 <= 42000):
    매수 = False
elif not (1.3 < 등락율 <= 27.5):
    매수 = False
elif not (매도잔량감소율 > 0.3):
    매수 = False
elif not (매도벽비율 < 0.6):
    매수 = False
elif 시가총액 < 1500:
    if not (초당매수수량 > 매도총잔량 * 1.2):
        매수 = False
    elif not (체결강도 > 162):
        매수 = False
    elif not (등락율각도(22) > 7):
        매수 = False
    elif not (전일비 > 42):
        매수 = False
    elif not (당일거래대금 > 14 * 100):
        매수 = False
elif 시가총액 < 3000:
    if not (초당매수수량 > 매도총잔량 * 1.0):
        매수 = False
    elif not (체결강도 > 152):
        매수 = False
    elif not (등락율각도(22) > 9):
        매수 = False
    elif not (전일비 > 52):
        매수 = False
    elif not (당일거래대금 > 38 * 100):
        매수 = False
else:
    if not (초당매수수량 > 매도총잔량 * 0.8):
        매수 = False
    elif not (체결강도 > 142):
        매수 = False
    elif not (등락율각도(22) > 11):
        매수 = False
    elif not (전일비 > 62):
        매수 = False
    elif not (당일거래대금 > 65 * 100):
        매수 = False
```

### 매도 조건식
```python
매도잔량증가율 = (매도총잔량 - 매도총잔량N(1)) / (매도총잔량N(1) or 1)

if 등락율 > 29.3:
    매도 = True
elif 수익률 >= 5.2:
    매도 = True
elif 수익률 <= -3.0:
    매도 = True
elif 최고수익률 - 수익률 >= 3.8:
    매도 = True
elif 보유시간 > 230:
    매도 = True
elif 매도잔량증가율 > 0.5:
    if 초당매도수량 > 초당매수수량 * 1.4:
        매도 = True
```
