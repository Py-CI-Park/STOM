## 개요

본 문서는 STOM 주식 자동거래 시스템에서 ****을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:00:00 (장중 전체)
- **대상 종목**: 관심종목, 시가총액 차등 기준
- **전략 타입**: Tick 기반 단타 전략
- **핵심 변수**: 체결강도, 초당거래대금, 등락율각도, 회전율
- **업데이트 이력**:
  - 초기 문서 작성: 

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 09:30:00 ~ 10:00:00 구간의 고가 돌파 후 재진입 전략(PostBreakout)을 위한 조건식과 최적화 범위를 제공합니다. 장 시작 후 고가 돌파 이후 나타나는 재조정 구간에서 재진입하는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.
- 최적화 조건식은 `self.vars[i]`로 치환하여 탐색 범위를 별도로 정의합니다.
- 최적화 범위 포맷: `self.vars[정수] = [[시작값, 종료값, 간격], 최적값]`

### 저장 이름 규칙

1. 기본 규칙:
   - `C`: Condition
   - `T`: Tick
   - `0930_1000`: 시간대
   - `PB`: PostBreakout
   - `B`/`S`: 매수/매도 조건식
   - `BO`/`SO`: 매수/매도 최적화 조건식
   - `BOR`/`SOR`: 매수/매도 최적화 범위
   - `GAR`: GA 최적화 범위

## 전략 개요

장 시작 30분 후(09:30~10:00)는 초반 변동성이 진정되고 재조정이 나타나는 구간입니다. 이 시간대의 특징:

- 고가 돌파 후 일시적 조정
- 조정 완료 후 재상승 포착
- 안정적인 이동평균선 지지
- 거래량 안정화 확인

    매도 = True

    매도 = True
elif 조정심화여부:
    매도 = True
elif 등락율각도(30) < -self.vars[44]:
    매도 = True
elif 체결강도 < 체결강도평균(60) * self.vars[45]:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)

## 조건식 개선 방향 연구

1) **돌파 패턴 분류**
   - 진짜 돌파 vs 가짜 돌파 구분
   - 거래량 동반 여부 확인

2) **조정 깊이 측정**
   - 피보나치 되돌림 수준 활용
   - 38.2%, 50%, 61.8% 되돌림 확인

3) **지지선 확인**
   - 이동평균선 지지 여부
   - 전일 고가/저가 지지 확인

4) **재상승 신호 강화**
   - 거래량 재증가 확인
   - 체결강도 회복 확인

5) **리스크 관리**
   - 조정이 5% 이상 심화되면 진입 중단
   - 이동평균선 이탈 시 즉시 청산
