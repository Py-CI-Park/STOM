# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성한 Tick 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **거래량 급증 포착 전략**을 정의한다.

- **대상 시간 구간**: 09:02:00 ~ 15:00:00 (장중 전체)
- **대상 종목**: 관심종목, 가격대 1,000원~50,000원
- **전략 타입**: 거래량 급증 감지 (Volume Spike)
- **핵심 변수**: 초당거래대금평균(120), 초당매수수량, 초당매도수량, 등락율각도(30/60), 체결강도평균(60)
- **업데이트 이력**:
  - 초기 문서 작성: 거래량 급증 전략

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 거래량 급증 전략(Volume Spike Strategy)을 위한 조건식과 최적화 범위를 제공합니다. 평균 대비 급격한 거래량 증가를 포착하여 큰 움직임의 초기 단계를 잡아내는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.
- 최적화 조건식은 `self.vars[i]`로 치환하여 탐색 범위를 별도로 정의합니다.

### 저장 이름 규칙

- `C_T_VS`: Condition_Tick_VolumeSpike
- `B`/`S`: 매수/매도 조건식
- `BO`/`SO`: 매수/매도 최적화 조건식
- `BOR`/`SOR`: 매수/매도 최적화 범위

# Condition - Tick VolumeSpike

## 전략 개요

거래량 급증 전략은 평균 대비 급격한 거래량 증가를 감지하여 큰 가격 변동의 시작을 포착합니다:

- 초당거래대금이 평균 대비 3배 이상 급증
- 초당매수수량이 초당매도수량보다 현저히 많음
- 등락율 각도 급상승
- 체결강도 급격한 증가

## 조건식

### 매수 조건식

```python
# ================================
#  공통 계산 지표
# ================================
거래대금급증여부      = (초당거래대금 > 초당거래대금평균(120) * 3.0)
매수압력우위여부      = (초당매수수량 > 초당매도수량 * 2.0)
각도급상승여부        = (등락율각도(30) > 10 and 등락율각도(30) > 등락율각도(60))
체결강도급증여부      = (체결강도 > 체결강도평균(60) * 1.5 and 체결강도 > 150)
연속증가여부          = (초당거래대금 > 초당거래대금N(1) and 초당거래대금N(1) > 초당거래대금N(2))

매수 = True

# 공통 필터
if not (관심종목 == 1):
    매수 = False
elif not (90200 <= 시분초 <= 150000):
    매수 = False
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 거래량 급증 확인
elif not 거래대금급증여부:
    매수 = False

# 매수 압력 우위 확인
elif not 매수압력우위여부:
    매수 = False

# 각도 급상승 확인
elif not 각도급상승여부:
    매수 = False

# 체결강도 급증 확인
elif not 체결강도급증여부:
    매수 = False

# 연속 증가 확인
elif not 연속증가여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (1.0 < 등락율 <= 20.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.15)):
        매수 = False
    elif not (회전율 > 2.5):
        매수 = False
    elif not (전일비 > 80):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 2.0):
        매수 = False
    elif not (당일거래대금 > 150 * 100):
        매수 = False
elif 시가총액 < 5000:
    if not (0.8 < 등락율 <= 15.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.20)):
        매수 = False
    elif not (회전율 > 2.0):
        매수 = False
    elif not (전일비 > 60):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 1.8):
        매수 = False
    elif not (당일거래대금 > 250 * 100):
        매수 = False
else:
    if not (0.5 < 등락율 <= 12.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.25)):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (전일비 > 40):
        매수 = False
    elif not (최고초당매수수량(30) > 최고초당매도수량(30) * 1.5):
        매수 = False
    elif not (당일거래대금 > 400 * 100):
        매수 = False

if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# ================================
#  공통 계산 지표
# ================================
거래대금급감여부      = (초당거래대금 < 초당거래대금평균(30) * 0.5)
매도압력우위여부      = (초당매도수량 > 초당매수수량 * 1.8)
각도급하락여부        = (등락율각도(30) < -8)

매도 = False

# 상한가 부근 안전장치
if 등락율 > 29.5:
    매도 = True

# 손절 및 목표가 도달 (보수적)
elif 수익률 >= 2.5 or 수익률 <= -2.0:
    매도 = True

# 트레일링 스톱 (공격적)
elif 최고수익률 > 3.5 and 최고수익률 * 0.70 >= 수익률:
    매도 = True

# 보유시간 기반 청산
elif 보유시간 > 1200:  # 20분
    매도 = True

# 거래대금 급감
elif 거래대금급감여부:
    매도 = True

# 매도 압력 우위
elif 매도압력우위여부:
    매도 = True

# 각도 급하락
elif 각도급하락여부:
    매도 = True

# 체결강도 급락
elif 체결강도 < 체결강도평균(30) * 0.6:
    매도 = True

# 거래량 급증 후 급감
elif 초당거래대금 < 초당거래대금평균(120) * 0.8:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

```python
거래대금급감여부      = (초당거래대금 < 초당거래대금평균(30) * self.vars[27])
```

### 매수·매도 통합 최적화 범위 - C_T_VS_OR

핵심 변수만 선별하여 통합 최적화 수행

```python
# 매수 핵심 변수 (BOR에서 선별)
# 공통
self.vars[1]  = [[2.5, 3.5, 0.5], 3.0]               # 거래대금 급증 비율
```

## 조건식 개선 방향 연구

1) **거래량 가속도 측정**
   - 1차 미분, 2차 미분 계산
   - 가속도가 양수일 때 신호 강화

2) **거래대금 임계값 동적 조정**
   - 시간대별, 시가총액별 차등 적용
   - 장 초반/후반 다른 임계값

3) **허위 급등 필터링**
   - 급증 후 N초 지속 여부 확인
   - 일시적 스파이크 제거

4) **호가창 두께 분석**
   - 매수/매도 잔량 분석
   - 얇은 호가창 회피

5) **리스크 관리 강화**
   - 급등 후 급락 대비 손절 강화
   - 보유시간 짧게 설정
