## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **모멘텀 급등 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 09:20:00 (장 시작 즉시 ~ 20분)
- **패턴**: 가격 상승 모멘텀이 강하게 형성되는 패턴
- **원리**: 연속적인 상승으로 강한 상승 추세 확인
- **특징**:
  - 등락율각도가 급격히 상승 (15도 이상)
  - 현재가가 연속 3틱 이상 상승
  - 이동평균선들이 정배열
  - 체결강도가 지속적으로 평균 이상 유지

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `T`: Tick을 의미합니다.
   - `MS`: Momentum Surge (모멘텀 급등)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_T_900_920_MS_B`: 900-920 모멘텀 급등 패턴의 매수 조건식
   - `C_T_900_920_MS_S`: 900-920 모멘텀 급등 패턴의 매도 조건식

# Condition - Tick Momentum Surge (모멘텀 급등)

## 전략 개요

### 패턴 인식 로직

**모멘텀 급등 (Momentum Surge) 정의**:
- 가격이 연속적으로 상승하며 강한 모멘텀을 보이는 패턴
- 등락율각도가 15도 이상으로 급격히 상승
- 현재가가 연속 3틱 이상 상승 (현재가 > 현재가N(1) > 현재가N(2))
- 이동평균선이 상승 추세 (단기 > 중기 > 장기)

**모멘텀 지속 확인**:
- 체결강도가 평균 이상을 지속적으로 유지
- 전일비각도가 20~60도 범위 (과도하지 않은 안정적 상승)
- 당일거래대금각도가 10도 이상

**추세 강도 확인**:
- 이동평균(30) > 이동평균(60) > 이동평균(90) 정배열
- 현재가가 이동평균(30) 위에 위치
- 최고현재가(60) 갱신 중

### 시장 상황별 전략

**소형주 (시가총액 < 3000억)**:
- 등락율각도 최소 18도 이상
- 체결강도 최소 120 이상
- 회전율 1.2% 이상

**중형주 (3000억 <= 시가총액 < 10000억)**:
- 등락율각도 최소 15도 이상
- 체결강도 최소 100 이상
- 회전율 0.8% 이상

**대형주 (시가총액 >= 10000억)**:
- 등락율각도 최소 12도 이상
- 체결강도 최소 80 이상
- 회전율 0.5% 이상

## 조건식

### 매수 조건식 - C_T_900_920_MS_B

```python
# ================================
#  공통 계산 지표
# ================================
전일종가          = 현재가 / (1 + (등락율 / 100))
시가등락율        = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
초당순매수금액    = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000
연속상승여부      = (현재가 > 현재가N(1) and 현재가N(1) > 현재가N(2))
이평정배열여부    = (이동평균(30) > 이동평균(60) and 이동평균(60) > 이동평균(90)) if 데이터길이 > 90 else False
강한모멘텀여부    = (등락율각도(30) > 15 and 당일거래대금각도(30) > 10)


# ================================
#  매수 조건 (모멘텀 급등)
# ================================
매수 = True


if not (관심종목 == 1):
    매수 = False


# ---------- 09:00:00 ~ 09:20:00 (모멘텀 급등 패턴) ----------
elif 90000 <= 시분초 < 92000:
    # 기본 가격 및 시간 필터
    if not (1000 < 현재가 <= 50000):
        매수 = False
    elif not (1.5 < 등락율 <= 20.0):
        매수 = False
    elif not (고저평균대비등락율 > -3.0):
        매수 = False
    elif not (현재가 < VI아래5호가):
        매수 = False
    elif 라운드피겨위5호가이내:
        매수 = False
    elif not (데이터길이 >= 90):
        매수 = False

    # 모멘텀 확인
    elif not 강한모멘텀여부:
        매수 = False
    elif not (전일비각도(30) > 20 and 전일비각도(30) < 60):
        매수 = False
    elif not 연속상승여부:
        매수 = False

    # 추세 확인
    elif not 이평정배열여부:
        매수 = False
    elif not (현재가 > 이동평균(30)):
        매수 = False
    elif not (이동평균(30) > 이동평균(30, 1)):
        매수 = False

    # 체결강도 및 거래량
    elif not (체결강도 > 체결강도평균(60)):
        매수 = False
    elif not (초당거래대금 > 초당거래대금N(1)):
        매수 = False

    # 시가총액별 세부 조건
    else:
        if 시가총액 < 3000:
            if not (-1.0 <= 시가등락율 < 10.0):
                매수 = False
            elif not (0.5 <= 시가대비등락율 < 12.0):
                매수 = False
            elif not (20 < 초당순매수금액 < 2500):
                매수 = False
            elif not (전일비 > 50 and 전일동시간비 > 50):
                매수 = False
            elif not (회전율 > 1.2):
                매수 = False
            elif not (당일거래대금 > 250 * 100):
                매수 = False
            elif not (체결강도 >= 120 and 체결강도 <= 350):
                매수 = False
            elif not (등락율각도(30) > 18):
                매수 = False
            elif not (현재가 > 최고현재가(60, 1)):
                매수 = False

        elif 시가총액 < 10000:
            if not (-0.5 <= 시가등락율 < 12.0):
                매수 = False
            elif not (1.0 <= 시가대비등락율 < 15.0):
                매수 = False
            elif not (30 < 초당순매수금액 < 3500):
                매수 = False
            elif not (전일비 > 40 and 전일동시간비 > 40):
                매수 = False
            elif not (회전율 > 0.8):
                매수 = False
            elif not (당일거래대금 > 400 * 100):
                매수 = False
            elif not (체결강도 >= 100 and 체결강도 <= 350):
                매수 = False
            elif not (등락율각도(30) > 15):
                매수 = False

        else:
            if not (0.0 <= 시가등락율 < 15.0):
                매수 = False
            elif not (1.5 <= 시가대비등락율 < 18.0):
                매수 = False
            elif not (50 < 초당순매수금액 < 5000):
                매수 = False
            elif not (전일비 > 30 and 전일동시간비 > 30):
                매수 = False
            elif not (회전율 > 0.5):
                매수 = False
            elif not (당일거래대금 > 600 * 100):
                매수 = False
            elif not (체결강도 >= 80 and 체결강도 <= 350):
                매수 = False
            elif not (등락율각도(30) > 12):
                매수 = False


# ---------- 09:20:00 이후 ----------
else:
    매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_T_900_920_MS_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율    = ((현재가 - 시가) / 시가) * 100
모멘텀소진여부    = (등락율각도(30) < 5 and 등락율각도(60) > 15)
이평이탈여부      = (현재가 < 이동평균(30) and 현재가N(1) >= 이동평균(30, 1))


# ================================
#  매도 조건 (모멘텀 급등)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.0:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 7.0:
    매도 = True

# 손절 조건
elif 수익률 <= -2.5:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 10.0 and 최고수익률 * 0.70 >= 수익률:
    매도 = True

# 모멘텀 소진
elif 모멘텀소진여부 and 최고수익률 > 4.0:
    매도 = True

# 이동평균 이탈
elif 이평이탈여부 and 수익률 > 1.0:
    매도 = True

# 일반 매도 조건 (09:30:00 이전)
elif 시분초 < 93000:
    # 단기 이익 실현
    if 수익률 >= 5.0 and 보유시간 < 90:
        매도 = True

    # 체결강도 약화
    elif 체결강도 < 체결강도평균(60) * 0.75 and 최고수익률 > 3.0:
        매도 = True

    # 시가총액별 매도 조건
    elif 시가총액 < 10000:
        if 등락율각도(30) >= 15 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.4 and (현재가 / 현재가N(1) - 1) * 100 < -0.4:
            매도 = True
        elif 5 <= 등락율각도(30) < 15 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.6 and (현재가 / 현재가N(1) - 1) * 100 < -0.5:
            매도 = True
        elif 등락율각도(30) < 5 and (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.7 and (현재가 / 현재가N(1) - 1) * 100 < -0.3:
            매도 = True
        elif 현재가 < 고가 * 0.965 and 최고수익률 > 6.0:
            매도 = True

# 보유시간 초과
elif 보유시간 > 200:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
