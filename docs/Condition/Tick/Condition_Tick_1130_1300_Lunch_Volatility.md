# 조건식 - 점심시간 전후 변동성 전략 (11:30~13:00)

- STOM 주식 자동거래에 사용하기 위한 틱 데이터 조건식 문서
- Back_Testing_Guideline_Tick.md 에 있는 조건식을 사용하여 만든 조건식

## 소개

이 문서는 11:30~13:00 점심시간 전후 변동성이 증가하는 시점을 포착하는 틱 기반 전략입니다.
점심시간 직전과 직후에 발생하는 변동성 확대 구간에서 매매 기회를 찾습니다.

## 전략 개요

**타겟 시간대**: 11:30 ~ 13:00 (점심시간 포함)

**핵심 아이디어**:
1. 등락율각도 급격한 변화 감지
2. 저변동성에서 고변동성으로 전환
3. 전일비각도 상승 확인
4. 점심시간 전 빠른 청산

## 조건식

### 매수 조건식

```python
초당순매수금액 = (초당매수수량 - 초당매도수량) * 현재가 / 1_000_000

    매수 = False

elif not (최고체결강도(60) - 최저체결강도(60) > 40):
    매수 = False
elif not (등락율각도(60) > 등락율각도(60, 30)):
    매수 = False
elif not (등락율각도(30) > 7):
    매수 = False
elif not (전일비각도(60) > 5):
    매수 = False
elif not (체결강도 > 130):
    매수 = False
elif not (초당거래대금 > 초당거래대금평균(60) * 2.2):
    매수 = False
elif not (초당매수수량 > 매도총잔량 * 45 / 100):
    매수 = False

    매수 = False

    if not (전일비 > 70):
        매수 = False

    elif not (당일거래대금 > 350):

    if not (전일비 > 50):

        매수 = False
    elif not (당일거래대금 > 600):

    if not (전일비 > 35):
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 2.8:
    매도 = True
elif 수익률 <= -2.2:
    매도 = True
elif 최고수익률 - 수익률 >= 1.8:
    매도 = True
elif 시분초 >= 115500 and 시분초 < 120000 and 보유시간 > 120:
    매도 = True
elif 시분초 >= 130000 and 보유시간 > 600:
    매도 = True
elif 시가총액 < 4000:
    if 체결강도 < 체결강도평균(30) - 25 and 수익률 > 0.5:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.65 and 수익률 > 0.3:
        매도 = True
    elif 등락율각도(30) < 2 and 수익률 > 1.2:
        매도 = True
else:
    if 체결강도 < 체결강도평균(30) - 15 and 수익률 > 0.5:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 매수총잔량 * 0.55 and 수익률 > 0.3:
        매도 = True
    elif 등락율각도(30) < 4 and 수익률 > 1.2:
        매도 = True
```
