# 조건식 - 호가 스프레드 축소 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 소개

**조건식 이름**: 호가 스프레드 축소 전략 (Ask Spread Narrow Strategy)

이 문서는 매수/매도 호가 스프레드의 축소를 통해 체결 임박 신호를 감지하는 전략을 설명합니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **호가 스프레드 축소 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반 20분)
- **대상 종목**: 시가총액 차등 기준 (1,500억, 3,000억)
- **전략 타입**: 호가 스프레드 축소 감지 전략
- **핵심 변수**: 스프레드, 스프레드비율, 매수/매도총잔량
- **업데이트 이력**:
  - 초기 문서 작성: 호가 스프레드 축소 전략

## 조건식

### 매수 조건식
```python
스프레드 = 매도호가1 - 매수호가1
스프레드비율 = 스프레드 / 현재가

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (1200 < 현재가 <= 38000):
    매수 = False
elif not (1.5 < 등락율 <= 28.5):
    매수 = False
elif not (스프레드비율 < 0.003):  # 0.3% 이내
    매수 = False
elif not (스프레드 < 스프레드N(1) < 스프레드N(2)):
    매수 = False
elif 시가총액 < 1500:
    if not (매수총잔량 > 매도총잔량 * 2.5):
        매수 = False
    elif not (초당매수수량 > 매도잔량1 * 0.8):
        매수 = False
    elif not (체결강도 > 158):
        매수 = False
    elif not (등락율각도(18) > 6):
        매수 = False
    elif not (전일비 > 38):
        매수 = False
elif 시가총액 < 3000:
    if not (매수총잔량 > 매도총잔량 * 2.2):
        매수 = False
    elif not (초당매수수량 > 매도잔량1 * 0.7):
        매수 = False
    elif not (체결강도 > 148):
        매수 = False
    elif not (등락율각도(18) > 8):
        매수 = False
    elif not (전일비 > 48):
        매수 = False
else:
    if not (매수총잔량 > 매도총잔량 * 2.0):
        매수 = False
    elif not (초당매수수량 > 매도잔량1 * 0.6):
        매수 = False
    elif not (체결강도 > 138):
        매수 = False
    elif not (등락율각도(18) > 10):
        매수 = False
    elif not (전일비 > 58):
        매수 = False
```

### 매도 조건식
```python
if 등락율 > 29.1:
    매도 = True
elif 수익률 >= 4.8:
    매도 = True
elif 수익률 <= -3.5:
    매도 = True
elif 최고수익률 - 수익률 >= 3.2:
    매도 = True
elif 보유시간 > 210:
    매도 = True
elif 스프레드 > 스프레드N(1) > 스프레드N(2):
    if 초당매도수량 > 초당매수수량 * 1.3:
        매도 = True
```
