# 조건식 - 체결강도 급반전 전략

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Tick]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 소개

**조건식 이름**: 체결강도 급반전 전략 (Strength Reversal Strategy)

이 문서는 체결강도의 급격한 반전을 통해 매수세 전환을 포착하는 전략을 설명합니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **체결강도 급반전 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 09:20:00 (장 초반 20분)
- **대상 종목**: 시가총액 차등 기준 (1,500억, 3,000억)
- **전략 타입**: 체결강도 급반전 포착 전략
- **핵심 변수**: 체결강도, 체결강도차이, 체결강도평균대비
- **업데이트 이력**:
  - 초기 문서 작성: 체결강도 급반전 전략

## 전략 개요

장 초반 체결강도가 100 이하에서 150 이상으로 급반전하는 순간을 포착하여 매수세 전환을 선제적으로 진입합니다.

## 조건식

### 매수 조건식
```python
체결강도차이 = 체결강도 - 체결강도N(1)
체결강도평균대비 = 체결강도 / (체결강도평균(30) or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (1000 < 현재가 <= 45000):
    매수 = False
elif not (0.8 < 등락율 <= 27.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 시가총액 < 1500:
    if not (체결강도 > 160):
        매수 = False
    elif not (체결강도차이 > 50):
        매수 = False
    elif not (체결강도N(1) < 100):
        매수 = False
    elif not (체결강도평균대비 > 1.5):
        매수 = False
    elif not (초당매수수량 > 초당매도수량 * 2.0):
        매수 = False
    elif not (등락율각도(20) > 6):
        매수 = False
    elif not (전일비 > 40):
        매수 = False
    elif not (당일거래대금 > 12 * 100):
        매수 = False
elif 시가총액 < 3000:
    if not (체결강도 > 150):
        매수 = False
    elif not (체결강도차이 > 45):
        매수 = False
    elif not (체결강도N(1) < 105):
        매수 = False
    elif not (체결강도평균대비 > 1.4):
        매수 = False
    elif not (초당매수수량 > 초당매도수량 * 1.8):
        매수 = False
    elif not (등락율각도(20) > 8):
        매수 = False
    elif not (전일비 > 50):
        매수 = False
    elif not (당일거래대금 > 35 * 100):
        매수 = False
else:
    if not (체결강도 > 140):
        매수 = False
    elif not (체결강도차이 > 40):
        매수 = False
    elif not (체결강도N(1) < 110):
        매수 = False
    elif not (체결강도평균대비 > 1.3):
        매수 = False
    elif not (초당매수수량 > 초당매도수량 * 1.6):
        매수 = False
    elif not (등락율각도(20) > 10):
        매수 = False
    elif not (전일비 > 60):
        매수 = False
    elif not (당일거래대금 > 60 * 100):
        매수 = False
```

### 매도 조건식
```python
if 등락율 > 29.0:
    매도 = True
elif 수익률 >= 4.5:
    매도 = True
elif 수익률 <= -3.8:
    매도 = True
elif 최고수익률 - 수익률 >= 2.8:
    매도 = True
elif 보유시간 > 200:
    매도 = True
elif 시가총액 < 1500:
    if 체결강도 < 80 and 체결강도N(1) > 140:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(30) * 0.7:
        매도 = True
elif 시가총액 < 3000:
    if 체결강도 < 90 and 체결강도N(1) > 135:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(30) * 0.6:
        매도 = True
else:
    if 체결강도 < 100 and 체결강도N(1) > 130:
        매도 = True
    elif (초당매도수량 - 초당매수수량) >= 최고초당매수수량(30) * 0.5:
        매도 = True
```
