# 조건식 - 변동성 확대 전략

## 소개
**조건식 이름**: 변동성 확대 전략
**거래 시간**: 90000 ~ 92000
**목표**: 고가-저가 폭 확대로 변동성 증가 포착

## 조건식

### 매수 조건식
```python
변동폭 = 고가 - 저가
변동폭비율 = 변동폭 / 저가
변동폭증가율 = 변동폭 / (변동폭N(30) or 1)

if not (90000 <= 시분초 < 92000):
    매수 = False
elif not (850 < 현재가 <= 36000):
    매수 = False
elif not (1.8 < 등락율 <= 26.5):
    매수 = False
elif not (변동폭비율 > 0.08):
    매수 = False
elif not (변동폭증가율 > 2.0):
    매수 = False
elif 시가총액 < 1500:
    if not (현재가 > 저가 + 변동폭 * 0.65):
        매수 = False
    elif not (체결강도 > 154):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(25) * 2.8):
        매수 = False
    elif not (등락율각도(16) > 9):
        매수 = False
    elif not (전일비 > 44):
        매수 = False
elif 시가총액 < 3000:
    if not (현재가 > 저가 + 변동폭 * 0.70):
        매수 = False
    elif not (체결강도 > 144):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(25) * 2.5):
        매수 = False
    elif not (등락율각도(16) > 11):
        매수 = False
    elif not (전일비 > 54):
        매수 = False
else:
    if not (현재가 > 저가 + 변동폭 * 0.75):
        매수 = False
    elif not (체결강도 > 134):
        매수 = False
    elif not (초당거래대금 > 초당거래대금평균(25) * 2.2):
        매수 = False
    elif not (등락율각도(16) > 13):
        매수 = False
    elif not (전일비 > 64):
        매수 = False
```

### 매도 조건식
```python
if 등락율 > 29.4:
    매도 = True
elif 수익률 >= 6.0:
    매도 = True
elif 수익률 <= -2.8:
    매도 = True
elif 최고수익률 - 수익률 >= 4.2:
    매도 = True
elif 보유시간 > 250:
    매도 = True
elif 현재가 < 저가 + 변동폭 * 0.3:
    매도 = True
```
