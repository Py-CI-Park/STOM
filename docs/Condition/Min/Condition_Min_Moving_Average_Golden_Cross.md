- **대상 시간 구간**: 09:00 ~ 15:18 (장 전체)
- **대상 종목**: 전 종목 대상
- **전략 타입**: 이동평균선 골든크로스/데드크로스 기반 추세 전환 전략
- **핵심 변수**: 이동평균(단기/장기), 골든크로스 발생 시점
- **업데이트 이력**:
  - 초기 문서 작성: 이동평균선 골든크로스 전략

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **이동평균선 골든크로스 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 15:18:00 (장중 전체)
- **패턴**: 단기 이동평균선이 장기 이동평균선을 상향 돌파
- **원리**: 이동평균선 골든크로스는 강력한 상승 신호
- **특징**:
  - 20분 이동평균선이 60분 이동평균선을 상향 돌파
  - 두 이동평균선 모두 상승 추세
  - 현재가가 두 이동평균선보다 위에 위치
  - 거래량 증가 동반

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `M`: Min (분봉)을 의미합니다.
   - `MAG`: Moving Average Golden Cross (이동평균선 골든크로스)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_M_900_1518_MAG_B`: 900-1518 이동평균선 골든크로스 패턴의 매수 조건식
   - `C_M_900_1518_MAG_S`: 900-1518 이동평균선 골든크로스 패턴의 매도 조건식

# Condition - Min Moving Average Golden Cross (이동평균선 골든크로스)

## 전략 개요

### 패턴 인식 로직

**이동평균선 골든크로스 (MA Golden Cross) 정의**:
- 단기 이동평균(20분)이 장기 이동평균(60분)을 상향 돌파
- 이전봉: 이동평균(20) <= 이동평균(60)
- 현재봉: 이동평균(20) > 이동평균(60)
- 두 이동평균선 모두 상승 추세

**추세 확인**:
- 이동평균(20) > 이동평균(20, 1) (단기 이평 상승)
- 이동평균(60) > 이동평균(60, 1) (장기 이평 상승)
- 현재가 > 이동평균(20) > 이동평균(60)
- 등락율각도(30) > 6도

**거래량 확인**:
- 분당거래대금이 평균(60) 이상
- 분당거래대금이 전봉 대비 증가
- 체결강도가 105 이상
- 전일비 및 전일동시간비가 양수

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 등락율각도(30) 최소 8도 이상
- 체결강도 최소 115 이상
- 분당거래대금이 평균 대비 1.2배 이상

**대형주 (시가총액 >= 2500억)**:
- 등락율각도(30) 최소 6도 이상
- 체결강도 최소 105 이상
- 분당거래대금이 평균 대비 1.1배 이상

## 조건식

### 매수 조건식 - C_M_900_1518_MAG_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
골든크로스여부 = (이동평균(20) > 이동평균(60) and 이동평균(20, 1) <= 이동평균(60, 1)) if 데이터길이 > 60 else False
이평상승여부 = (이동평균(20) > 이동평균(20, 1) and 이동평균(60) > 이동평균(60, 1)) if 데이터길이 > 60 else False
가격위치정상여부 = (현재가 > 이동평균(20) and 이동평균(20) > 이동평균(60)) if 데이터길이 > 60 else False


# ================================
#  매수 조건 (이동평균선 골든크로스)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (0.8 < 등락율 <= 18.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# 골든크로스 확인
elif not 골든크로스여부:
    매수 = False
elif not 이평상승여부:
    매수 = False
elif not 가격위치정상여부:
    매수 = False

# 추세 확인
elif not (등락율각도(30) > 6):
    매수 = False
elif not (당일거래대금각도(30) > 5):
    매수 = False

# 거래량 확인
elif not (분당거래대금 > 분당거래대금평균(60)):
    매수 = False
elif not (분당거래대금 > 분당거래대금N(1)):
    매수 = False
elif not (체결강도 >= 105):
    매수 = False
elif not (전일비 > 0 and 전일동시간비 > 0):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (0.5 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (1.0 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (10 < 분당순매수금액 and 분당순매수금액 < 550):
            매수 = False
        elif not (전일비 > 28 and 전일동시간비 > 28):
            매수 = False
        elif not (회전율 > 0.65):
            매수 = False
        elif not (당일거래대금 > 95 * 100):
            매수 = False
        elif not (등락율각도(30) > 8):
            매수 = False
        elif not (체결강도 >= 115):
            매수 = False
        elif not (분당거래대금 > 분당거래대금평균(60) * 1.2):
            매수 = False
        elif not (현재가 > 최고현재가(30, 1)):
            매수 = False
    else:
        if not (0.7 <= 시가등락율 and 시가등락율 < 14.0):
            매수 = False
        elif not (1.2 <= 시가대비등락율 and 시가대비등락율 < 16.0):
            매수 = False
        elif not (18 < 분당순매수금액 and 분당순매수금액 < 750):
            매수 = False
        elif not (전일비 > 22 and 전일동시간비 > 22):
            매수 = False
        elif not (회전율 > 0.45):
            매수 = False
        elif not (당일거래대금 > 170 * 100):
            매수 = False
        elif not (등락율각도(30) > 6):
            매수 = False
        elif not (체결강도 >= 105):
            매수 = False
        elif not (분당거래대금 > 분당거래대금평균(60) * 1.1):
            매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_900_1518_MAG_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
데드크로스여부 = (이동평균(20) < 이동평균(60) and 이동평균(20, 1) >= 이동평균(60, 1)) if 데이터길이 > 60 else False
단기이평이탈여부 = (현재가 < 이동평균(20) and 현재가N(1) >= 이동평균(20, 1))


# ================================
#  매도 조건 (이동평균선 골든크로스)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 4.8:
    매도 = True

# 손절 조건
elif 수익률 <= -2.3:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 6.5 and 최고수익률 * 0.71 >= 수익률:
    매도 = True

# 데드크로스 발생
elif 데드크로스여부 and 수익률 > 0.3:
    매도 = True

# 단기 이평선 이탈
elif 단기이평이탈여부 and 수익률 > 0.8:
    매도 = True

# 이평선 하향 전환
elif 이동평균(20) < 이동평균(20, 1) and 최고수익률 > 2.5:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.57:
        매도 = True
    elif 수익률 < -1.4 and 체결강도 < 96:
        매도 = True
    elif 최고수익률 > 3.5 and 현재가 < 이동평균(60):
        매도 = True
else:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.51:
        매도 = True
    elif 수익률 < -1.2 and 체결강도 < 93:
        매도 = True
    elif 최고수익률 > 3.0 and 현재가 < 이동평균(60):
        매도 = True

# 보유시간 초과
if 보유시간 > 170:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
