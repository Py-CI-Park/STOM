- STOM 주식 자동거래에 사용하기 위한 조건식 문서

- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **볼린저밴드 반등 전략**을 정의한다.

- **대상 시간 구간**: 09:00 ~ 15:18 (장 전체)
- **대상 종목**: 전 종목 대상
- **전략 타입**: 볼린저밴드 하단 반등 포착 전략
- **핵심 변수**: BBL, BBU, BBM, 밴드 하단 접촉 여부
- **업데이트 이력**:
  - 초기 문서 작성: 볼린저밴드 반등 전략

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **볼린저밴드 반등 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 15:18:00 (장중 전체)
- **패턴**: 볼린저밴드 하단에서 반등하는 패턴
- **원리**: 과매도 구간에서의 반등 포착
- **특징**:
  - 이전봉에서 볼린저밴드 하단 터치 또는 이탈
  - 현재봉에서 볼린저밴드 하단 위로 복귀
  - RSI가 30 이하에서 35 이상으로 회복
  - 체결강도가 110 이상으로 반등

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `M`: Min (분봉)을 의미합니다.
   - `BB`: Bollinger Bounce (볼린저밴드 반등)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_M_900_1518_BB_B`: 900-1518 볼린저밴드 반등 패턴의 매수 조건식
   - `C_M_900_1518_BB_S`: 900-1518 볼린저밴드 반등 패턴의 매도 조건식

# Condition - Min Bollinger Bounce (볼린저밴드 반등)

## 전략 개요

### 패턴 인식 로직

**볼린저밴드 반등 (Bollinger Bounce) 정의**:
- 가격이 볼린저밴드 하단을 터치하거나 이탈했다가 반등
- 이전봉에서 현재가 <= BBL (볼린저밴드 하단)
- 현재봉에서 현재가 > BBL (볼린저밴드 하단 위로 복귀)
- RSI가 과매도 구간(30 이하)에서 회복

**반등 강도 확인**:
- RSI가 30 이하에서 35 이상으로 회복
- 체결강도가 110 이상
- 분당매수수량 > 분당매도수량 (매수세 우위)
- 현재가 < BBM (볼린저밴드 중심선 이하, 추가 상승 여력)

**거래량 확인**:
- 분당거래대금이 평균 이상
- 전일비 및 전일동시간비가 양수
- 체결강도평균(30)이 100 이상

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- RSI 회복폭 최소 8 이상 (30 이하 → 38 이상)
- 체결강도 최소 120 이상
- 회전율 0.6% 이상

**대형주 (시가총액 >= 2500억)**:
- RSI 회복폭 최소 5 이상 (30 이하 → 35 이상)
- 체결강도 최소 110 이상
- 회전율 0.4% 이상

## 조건식

### 매수 조건식 - C_M_900_1518_BB_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
BBL하단터치여부 = (현재가N(1) <= BBL_N(1))
BBL복귀여부 = (현재가 > BBL)
RSI반등여부 = (RSI_N(1) <= 30 and RSI > 35)
볼린저반등여부 = (BBL하단터치여부 and BBL복귀여부 and RSI반등여부)


# ================================
#  매수 조건 (볼린저밴드 반등)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (-2.0 < 등락율 <= 15.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# 볼린저밴드 반등 확인
elif not 볼린저반등여부:
    매수 = False
elif not (현재가 < BBM):
    매수 = False
elif not (현재가 > 현재가N(1)):
    매수 = False

# 체결강도 및 거래량
elif not (체결강도 >= 110):
    매수 = False
elif not (분당매수수량 > 분당매도수량):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False
elif not (체결강도평균(30) >= 100):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (-5.0 <= 시가등락율 and 시가등락율 < 8.0):
            매수 = False
        elif not (-3.0 <= 시가대비등락율 and 시가대비등락율 < 10.0):
            매수 = False
        elif not (5 < 분당순매수금액 and 분당순매수금액 < 400):
            매수 = False
        elif not (전일비 > 20 and 전일동시간비 > 20):
            매수 = False
        elif not (회전율 > 0.6):
            매수 = False
        elif not (당일거래대금 > 80 * 100):
            매수 = False
        elif not (RSI > 38):
            매수 = False
        elif not (체결강도 >= 120):
            매수 = False
        elif not (분당매수수량 > 분당매도수량 * 1.2):
            매수 = False
    else:
        if not (-3.0 <= 시가등락율 and 시가등락율 < 10.0):
            매수 = False
        elif not (-2.0 <= 시가대비등락율 and 시가대비등락율 < 12.0):
            매수 = False
        elif not (10 < 분당순매수금액 and 분당순매수금액 < 600):
            매수 = False
        elif not (전일비 > 15 and 전일동시간비 > 15):
            매수 = False
        elif not (회전율 > 0.4):
            매수 = False
        elif not (당일거래대금 > 150 * 100):
            매수 = False
        elif not (RSI > 35):
            매수 = False
        elif not (체결강도 >= 110):
            매수 = False
        elif not (분당매수수량 > 분당매도수량 * 1.1):
            매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_900_1518_BB_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
BBM도달여부 = (현재가 >= BBM and 현재가N(1) < BBM_N(1))
BBU근접여부 = (현재가 > BBU * 0.98)


# ================================
#  매도 조건 (볼린저밴드 반등)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 3.5:
    매도 = True

# 손절 조건
elif 수익률 <= -2.0:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 5.0 and 최고수익률 * 0.75 >= 수익률:
    매도 = True

# 볼린저밴드 중심선 도달
elif BBM도달여부 and 수익률 > 1.5:
    매도 = True

# 볼린저밴드 상단 근접
elif BBU근접여부 and 수익률 > 2.0:
    매도 = True

# RSI 과매수
elif RSI > 70 and 수익률 > 0.8:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.55:
        매도 = True
    elif 수익률 < -1.2 and 체결강도 < 95:
        매도 = True
    elif 최고수익률 > 3.0 and 현재가 < BBM and 현재가N(1) >= BBM_N(1):
        매도 = True
else:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.5:
        매도 = True
    elif 수익률 < -1.0 and 체결강도 < 95:
        매도 = True
    elif 최고수익률 > 2.5 and 현재가 < BBM and 현재가N(1) >= BBM_N(1):
        매도 = True

# 보유시간 초과
if 보유시간 > 150:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
