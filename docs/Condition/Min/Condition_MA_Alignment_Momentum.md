## 소개

이 문서는 전체 거래시간 09:00~15:18 구간에서 이동평균선 정배열을 활용한 추세 추종 분봉 전략입니다.
단기, 중기, 장기 이동평균선이 정배열되고 모두 상승 중일 때 강한 상승 추세로 판단하고 진입합니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **MA Alignment Momentum (이동평균 정배열 모멘텀 전략)**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:18:00 (장 전체)
- **대상 종목**: 시가총액 2,000억 차등 기준
- **전략 타입**: 분봉 기반 이동평균 정배열 추세 추종 전략
- **핵심 변수**: 이동평균선 정배열, 분당거래대금, 체결강도, RSI
- **업데이트 이력**:
  - 초기 문서 작성: 이동평균 정배열 모멘텀 전략

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 분봉 데이터 기반으로 900~1518 (09:00~15:18) 시간에만 거래합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `MAM`: MA Alignment Momentum을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.
   - `M`: Min (분봉) 조건식을 의미합니다.

2. **예시**:
   - `C_MAM_M_B`: MA Alignment Momentum 분봉 매수 조건식
   - `C_MAM_M_S`: MA Alignment Momentum 분봉 매도 조건식

## 조건식

### 매수 조건식

```python
# 이동평균 정배열 모멘텀 전략 (09:00 ~ 15:18)

# 시간 조건
if not (90000 <= 시분초 <= 151800):
    매수 = False

# 데이터 충분성 확인 (이동평균 계산을 위해)
elif not (데이터길이 > 80):
    매수 = False

# 가격 및 등락율 조건
elif not (2000 < 현재가 <= 50000):
    매수 = False
elif not (1.0 < 등락율 <= 20.0):
    매수 = False

# VI 및 라운드피겨 제외
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 이동평균 정배열 조건 (핵심)
# 1) 단기 > 중기 > 장기 (5분 > 20분 > 60분)
elif not (이동평균(5) > 이동평균(20) > 이동평균(60)):
    매수 = False

# 2) 모든 이동평균선 상승 중
elif not (이동평균(5) > 이동평균(5, 1)):
    매수 = False
elif not (이동평균(20) > 이동평균(20, 1)):
    매수 = False
elif not (이동평균(60) > 이동평균(60, 1)):
    매수 = False

# 3) 현재가가 단기 이평 위에 위치
elif not (현재가 > 이동평균(5)):
    매수 = False

# 4) 단기와 중기 이평 간 적정 간격 유지 (급등 후 과열 제외)
elif not (이동평균(5) < 이동평균(20) * 1.05):
    매수 = False

# 분봉 패턴 조건
elif not (현재가 > 분봉시가):
    매수 = False
elif not (분봉고가 > 분봉고가N(1)):
    매수 = False

# 거래량 조건
elif not (분당거래대금 > 분당거래대금평균(30) * 1.2):
    매수 = False

# 체결강도 조건
elif not (체결강도 > 115):
    매수 = False
elif not (체결강도평균(20) > 110):
    매수 = False

# 등락율 각도 조건 (상승 추세)
elif not (등락율각도(20) > 5):
    매수 = False

# 시가총액별 세분화 조건
elif 시가총액 < 2500:
    # 소형주
    if not (분당매수수량 > 분당매도수량 * 1.5):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (전일비 > 80):
        매수 = False
    elif not (전일동시간비 > 120):
        매수 = False
    elif not (회전율 > 2.5):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 10):
        매수 = False
    elif not (매수총잔량 > 매도총잔량 * 1.2):
        매수 = False

elif 시가총액 < 5000:
    # 중형주
    if not (분당매수수량 > 분당매도수량 * 1.3):
        매수 = False
    elif not (당일거래대금 > 100 * 100):
        매수 = False
    elif not (전일비 > 60):
        매수 = False
    elif not (전일동시간비 > 100):
        매수 = False
    elif not (회전율 > 2.0):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 25 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 8):
        매수 = False
    elif not (매수총잔량 > 매도총잔량):
        매수 = False

else:
    # 대형주
    if not (분당매수수량 > 분당매도수량 * 1.2):
        매수 = False
    elif not (당일거래대금 > 200 * 100):
        매수 = False
    elif not (전일비 > 50):
        매수 = False
    elif not (전일동시간비 > 80):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 30 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 7):
        매수 = False
    elif not (매수총잔량 > 매도총잔량 * 0.9):
        매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# 정배열 이탈 및 익절 전략
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 6.0:
    매도 = True
elif 수익률 <= -3.0:
    매도 = True
elif 최고수익률 - 수익률 >= 4.0:
    매도 = True
elif 보유시간 > 300:  # 300분 (5시간) 초과 보유 시 청산
    매도 = True

# 정배열 붕괴 조건 (핵심 매도 조건)
# 1) 단기 이평이 중기 이평 하회
elif 이동평균(5) < 이동평균(20):
    if 수익률 > 0:
        매도 = True

# 2) 단기 이평 하락 전환
elif 이동평균(5) < 이동평균(5, 1) and 이동평균(5, 1) < 이동평균(5, 2):
    if 수익률 > 1.0:
        매도 = True

# 3) 현재가가 단기 이평 하회
elif 현재가 < 이동평균(5):
    if 수익률 > 0.5:
        매도 = True

# 중기 이평 하락 전환 (추세 약화)
elif 이동평균(20) < 이동평균(20, 1):
    if 수익률 > 2.0:
        매도 = True

# 체결강도 약화
elif 체결강도 < 100:
    매도 = True
elif 체결강도평균(20) < 105:
    if 수익률 > 1.0:
        매도 = True

# 분봉 음봉 연속
elif 현재가 < 분봉시가 and 현재가N(1) < 분봉시가N(1):
    if 수익률 > 1.5:
        매도 = True

# 등락율 각도 하락
elif 등락율각도(20) < 0:
    if 수익률 > 0.5:
        매도 = True

# 시가총액별 세분화 매도 조건
elif 시가총액 < 2500:
    # 소형주: 적극적 익절
    if 3.0 <= 수익률 < 6.0 and 분당매도수량 > 분당매수수량 * 1.8:
        매도 = True
    elif 거래대금증감 < 0:
        매도 = True
    elif 분당거래대금 < 분당거래대금평균(30) * 0.7:
        매도 = True

elif 시가총액 < 5000:
    # 중형주
    if 3.0 <= 수익률 < 6.0 and 분당매도수량 > 분당매수수량 * 1.6:
        매도 = True
    elif 거래대금증감 < -50 * 100:
        매도 = True
    elif 분당거래대금 < 분당거래대금평균(30) * 0.75:
        매도 = True

else:
    # 대형주
    if 3.0 <= 수익률 < 6.0 and 분당매도수량 > 분당매수수량 * 1.5:
        매도 = True
    elif 거래대금증감 < -100 * 100:
        매도 = True
    elif 분당거래대금 < 분당거래대금평균(30) * 0.8:
        매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화
