# 조건식 - ATR 변동성 돌파 전략 (분봉)

- STOM 주식 자동거래에 사용하기 위한 분봉 데이터 조건식 문서
- [[Back_Testing_Guideline_Min]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 소개

이 문서는 ATR(Average True Range) 지표를 활용하여 변동성 확대 구간에서 돌파를 포착하는 분봉 기반 전략입니다.
저변동성에서 고변동성으로 전환되는 시점을 진입 신호로 활용합니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **ATR Breakout (ATR 변동성 돌파 전략)**을 정의한다.

- **대상 시간 구간**: 09:30:00 ~ 14:30:00
- **대상 종목**: 가격대 3,000원~25,000원, 등락율 3%~20%
- **전략 타입**: 분봉 기반 ATR 변동성 돌파 전략
- **핵심 변수**: ATR, 분봉고가 돌파, 볼린저밴드, 분당거래대금
- **업데이트 이력**:
  - 초기 문서 작성: ATR 변동성 돌파 전략

## 전략 개요

**핵심 아이디어**:
1. ATR이 최근 평균 대비 급증하는 시점 포착
2. 분봉고가 돌파와 동시에 발생하는 변동성 확대
3. 거래대금 증가 동반 확인
4. 볼린저밴드 상단 돌파 보조 확인

## 조건식

### 매수 조건식

```python
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000

if not (93000 <= 시분초 < 143000):
    매수 = False
elif not (3000 < 현재가 <= 25000):
    매수 = False
elif not (3.0 < 등락율 <= 20.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif not (현재가 > 최고분봉고가(20, 1)):
    매수 = False
elif not (ATR > ATR_N(1) * 1.3):
    매수 = False
elif not (ATR > ATR_N(5) * 1.5):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(60) * 2.0):
    매수 = False
elif not (체결강도 > 130):
    매수 = False
elif not (현재가 > BBM):
    매수 = False
elif not (BBU - BBL > (BBU_N(30) - BBL_N(30))):
    매수 = False
elif not (분당매수수량 > 분당매도수량 * 1.5):
    매수 = False
elif not (분당순매수금액 > 30):
    매수 = False
elif 시가총액 < 2000:
    if not (전일비 > 60):
        매수 = False
    elif not (회전율 > 2.5):
        매수 = False
    elif not (등락율각도(30) > 8):
        매수 = False
elif 시가총액 < 4000:
    if not (전일비 > 45):
        매수 = False
    elif not (회전율 > 2.0):
        매수 = False
    elif not (등락율각도(30) > 6):
        매수 = False
else:
    if not (전일비 > 30):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (등락율각도(30) > 5):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.0:
    매도 = True
elif 수익률 <= -2.5:
    매도 = True
elif 최고수익률 - 수익률 >= 2.0:
    매도 = True
elif ATR < ATR_N(1) * 0.7 and 수익률 > 0.5:
    매도 = True
elif 현재가 < BBM and 현재가_N(1) >= BBM_N(1) and 수익률 > 1.0:
    매도 = True
elif 보유시간 > 120:
    매도 = True
elif 시가총액 < 4000:
    if 체결강도 < 체결강도평균(30) - 25 and 수익률 > 0.5:
        매도 = True
    elif 분당매도수량 > 분당매수수량 * 1.8 and 수익률 > 0.3:
        매도 = True
else:
    if 체결강도 < 체결강도평균(30) - 18 and 수익률 > 0.5:
        매도 = True
    elif 분당매도수량 > 분당매수수량 * 1.5 and 수익률 > 0.3:
        매도 = True
```
