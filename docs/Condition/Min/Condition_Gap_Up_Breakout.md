## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반(09:00~10:00) 갭상승 돌파 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 10:00:00 (장 시작 1시간)
- **대상 종목**: 가격대 2,000원~50,000원, 등락율 2~20%
- **전략 타입**: 갭상승 + 고가 돌파 포착 (Gap Up Breakout)
- **핵심 변수**: 시가갭상승율, 최고현재가(5), MACD, RSI, 체결강도
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 장초 09:00~10:00 구간에서 갭상승 이후 돌파 패턴을 포착하는 분봉 전략입니다.
시가 대비 2% 이상 갭상승한 종목이 고가를 돌파할 때 추가 상승을 기대하고 진입합니다.

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.
- 분봉 데이터 기반으로 900~1518 (09:00~15:18) 시간에만 거래합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `GUB`: Gap Up Breakout을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.
   - `M`: Min (분봉) 조건식을 의미합니다.

2. **예시**:
   - `C_GUB_M_B`: Gap Up Breakout 분봉 매수 조건식
   - `C_GUB_M_S`: Gap Up Breakout 분봉 매도 조건식

## 조건식

### 매수 조건식

```python
# 장초 갭상승 돌파 전략 (09:00 ~ 10:00)
전일종가 = 현재가 / (1 + (등락율 / 100))
시가갭상승율 = ((시가 - 전일종가) / 전일종가) * 100

# 시간 조건 (09:00 ~ 10:00)
if not (90000 <= 시분초 <= 100000):
    매수 = False

# 데이터 충분성 확인
elif not (데이터길이 > 5):
    매수 = False

# 가격 및 등락율 조건
elif not (2000 < 현재가 <= 50000):
    매수 = False
elif not (2.0 < 등락율 <= 20.0):
    매수 = False

# VI 제외
elif not (현재가 < VI아래5호가):
    매수 = False

# 갭상승 조건 (핵심)
elif not (시가갭상승율 >= 2.0):
    매수 = False

# 고가 돌파 조건
elif not (현재가 > 고가N(1)):
    매수 = False
elif not (분봉고가 > 분봉고가N(1)):
    매수 = False

# 분봉 양봉 확인
elif not (현재가 > 분봉시가):
    매수 = False

# 거래량 조건
elif not (분당거래대금 > 분당거래대금평균(10) * 1.5):
    매수 = False
elif not (분당거래대금 > 분당거래대금N(1)):
    매수 = False

# 체결강도 조건
elif not (체결강도 > 120):
    매수 = False
elif not (체결강도 > 체결강도N(1)):
    매수 = False

# 전일비 및 전일동시간비 조건
elif not (전일비 > 100):
    매수 = False
elif not (전일동시간비 > 150):
    매수 = False

# 시가총액별 세분화 조건
elif 시가총액 < 2000:
    # 소형주: 강한 모멘텀 요구
    if not (등락율각도(10) > 12):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (회전율 > 3):
        매수 = False
    elif not (분당매수수량 > 분당매도수량 * 1.5):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 15 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 12):
        매수 = False

elif 시가총액 < 5000:
    # 중형주
    if not (등락율각도(10) > 10):
        매수 = False
    elif not (당일거래대금 > 100 * 100):
        매수 = False
    elif not (회전율 > 2):
        매수 = False
    elif not (분당매수수량 > 분당매도수량 * 1.3):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 20 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 10):
        매수 = False

else:
    # 대형주
    if not (등락율각도(10) > 8):
        매수 = False
    elif not (당일거래대금 > 200 * 100):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (분당매수수량 > 분당매도수량 * 1.2):
        매수 = False
    elif not (현재가 > 고가 - (고가 - 저가) * 25 / 100):
        매수 = False
    elif not ((현재가 / 저가 - 1) * 100 <= 8):
        매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식

```python
# 돌파 실패 및 익절 전략
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 4.0:
    매도 = True
elif 수익률 <= -3.0:
    매도 = True
elif 최고수익률 - 수익률 >= 3.0:
    매도 = True
elif 보유시간 > 120:  # 120분 (2시간) 초과 보유 시 청산
    매도 = True

# 고가 대비 하락 조건
elif 현재가 < 고가 * 0.97:
    if 수익률 > 0.5:
        매도 = True

# 분봉 음봉 전환
elif 현재가 < 분봉시가:
    if 수익률 > 1.0:
        매도 = True

# 체결강도 약화
elif 체결강도 < 100:
    매도 = True

# 거래량 급감
elif 분당거래대금 < 분당거래대금평균(10) * 0.6:
    if 수익률 > 0:
        매도 = True

# 시가총액별 세분화 매도 조건
elif 시가총액 < 2000:
    # 소형주: 빠른 청산
    if 1.5 <= 수익률 < 4.0 and 분당매도수량 > 분당매수수량 * 1.8:
        매도 = True
    elif 등락율각도(10) < 5:
        매도 = True

elif 시가총액 < 5000:
    # 중형주
    if 1.5 <= 수익률 < 4.0 and 분당매도수량 > 분당매수수량 * 1.6:
        매도 = True
    elif 등락율각도(10) < 6:
        매도 = True

else:
    # 대형주
    if 1.5 <= 수익률 < 4.0 and 분당매도수량 > 분당매수수량 * 1.5:
        매도 = True
    elif 등락율각도(10) < 7:
        매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
