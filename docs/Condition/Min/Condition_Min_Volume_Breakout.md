## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장중(09:05~14:30) 거래량 돌파 전략**을 정의한다.

- **대상 시간 구간**: 09:05:00 ~ 14:30:00 (장중)
- **대상 종목**: 관심종목, 가격대 1,000원~50,000원
- **전략 타입**: 거래량 급증 + 가격 돌파 포착 (Volume Breakout)
- **핵심 변수**: 분당거래대금, 최고현재가(20), 분당매수수량, 체결강도
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 거래량 돌파 전략(Volume Breakout Strategy)을 위한 조건식을 제공합니다. 비정상적인 거래량 급증과 가격 돌파를 함께 포착하여 강한 추세 전환을 활용하는 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.

# Condition - Min Volume Breakout

## 전략 개요

거래량 돌파 전략은 비정상적인 거래량 급증과 가격 돌파를 포착합니다:

- 거래량 평균 대비 3배 이상 급증
- 고가 돌파 동반 확인
- 매수세 우위 확인
- 연속 상승 확인

## 조건식

### 매수 조건식

```python
# ================================
#  공통 계산 지표
# ================================
거래량급증여부        = (분당거래대금 > 분당거래대금평균(30) * 3.0)
연속거래량증가여부    = (분당거래대금 > 분당거래대금N(1) and 분당거래대금N(1) > 분당거래대금N(2))
고가돌파여부          = (현재가 > 최고현재가(20)) if 데이터길이 > 20 else False
가격상승률            = ((현재가 - 현재가N(5)) / 현재가N(5)) * 100 if 데이터길이 > 5 else 0
매수세우위여부        = (분당매수수량 > 분당매도수량 * 2.0)
체결강도강함여부      = (체결강도 >= 130)

매수 = True

# 공통 필터
if not (관심종목 == 1):
    매수 = False
elif not (90500 <= 시분초 <= 143000):
    매수 = False
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False

# 거래량 급증 확인
elif not 거래량급증여부:
    매수 = False

# 연속 거래량 증가 확인
elif not 연속거래량증가여부:
    매수 = False

# 고가 돌파 확인
elif not 고가돌파여부:
    매수 = False

# 가격 상승률 확인
elif not (1.0 <= 가격상승률 <= 10.0):
    매수 = False

# 매수세 우위 확인
elif not 매수세우위여부:
    매수 = False

# 체결강도 확인
elif not 체결강도강함여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (회전율 > 2.5):
        매수 = False
    elif not (현재가 > 이동평균(5)):
        매수 = False
    elif not (이동평균(5) > 이동평균(20)):
        매수 = False
    elif not (RSI > 50 and RSI < 80):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(30) * 3.5):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.20)):  # 고가 대비 80% 이상
        매수 = False
elif 시가총액 < 5000:
    if not (회전율 > 2.0):
        매수 = False
    elif not (현재가 > 이동평균(5)):
        매수 = False
    elif not (이동평균(5) > 이동평균(20)):
        매수 = False
    elif not (RSI > 48 and RSI < 75):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(30) * 3.2):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.25)):
        매수 = False
else:
    if not (회전율 > 1.5):
        매수 = False
    elif not (현재가 > 이동평균(5)):
        매수 = False
    elif not (이동평균(5) > 이동평균(20)):
        매수 = False
    elif not (RSI > 45 and RSI < 70):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(30) * 3.0):
        매수 = False
    elif not (현재가 > (고가 - (고가 - 저가) * 0.30)):
        매수 = False

if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```
