## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장중 전체(09:00~15:18) RSI 반전 패턴 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:18:00 (장중 전체)
- **대상 종목**: 시가총액 차등 (2,500억 기준), 등락율 -5~15%
- **전략 타입**: RSI 과매도 반전 + 스토캐스틱 동반 확인 (RSI Reversal)
- **핵심 변수**: RSI, STOCHSK, MFI, 체결강도, 분당매수수량, 분봉고가
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **RSI 반전 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 15:18:00 (장중 전체)
- **패턴**: RSI가 과매도 구간에서 반등하는 패턴
- **원리**: RSI가 30 이하 과매도 구간에서 반등 시 매수
- **특징**:
  - RSI가 30 이하 → 40 이상으로 반등
  - 스토캐스틱(%K)이 20 이하 → 30 이상으로 반등
  - 체결강도가 110 이상으로 회복
  - 현재가가 이전봉 대비 상승

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `M`: Min (분봉)을 의미합니다.
   - `RR`: RSI Reversal (RSI 반전)을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_M_900_1518_RR_B`: 900-1518 RSI 반전 패턴의 매수 조건식
   - `C_M_900_1518_RR_S`: 900-1518 RSI 반전 패턴의 매도 조건식

# Condition - Min RSI Reversal (RSI 반전)

## 전략 개요

### 패턴 인식 로직

**RSI 반전 (RSI Reversal) 정의**:
- RSI가 과매도 구간(30 이하)에서 반등
- 이전봉: RSI <= 30
- 현재봉: RSI > 40
- 스토캐스틱 %K도 동반 반등 (20 이하 → 30 이상)

**반등 확인**:
- 현재가 > 현재가N(1) (가격 상승)
- 체결강도 >= 110 (매수세 우위)
- 분당매수수량 > 분당매도수량 * 1.2
- 분봉고가 > 분봉고가N(1) (고가 갱신)

**추가 필터**:
- 등락율 -5% ~ 15% 범위
- MFI (Money Flow Index) > 40 (자금 유입)
- 분당거래대금이 평균 이상

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- RSI 반등폭 최소 12 이상 (30 이하 → 42 이상)
- 체결강도 최소 120 이상
- 스토캐스틱 %K > 35

**대형주 (시가총액 >= 2500억)**:
- RSI 반등폭 최소 10 이상 (30 이하 → 40 이상)
- 체결강도 최소 110 이상
- 스토캐스틱 %K > 30

## 조건식

### 매수 조건식 - C_M_900_1518_RR_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
RSI반등여부 = (RSI_N(1) <= 30 and RSI > 40)
스토캐스틱반등여부 = (STOCHSK_N(1) <= 20 and STOCHSK > 30)
가격반등여부 = (현재가 > 현재가N(1) and 분봉고가 > 분봉고가N(1))


# ================================
#  매수 조건 (RSI 반전)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (-5.0 < 등락율 <= 15.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# RSI 및 스토캐스틱 반등 확인
elif not RSI반등여부:
    매수 = False
elif not 스토캐스틱반등여부:
    매수 = False

# 가격 반등 확인
elif not 가격반등여부:
    매수 = False
elif not (현재가 > 분봉시가):
    매수 = False

# 체결강도 및 거래량
elif not (체결강도 >= 110):
    매수 = False
elif not (분당매수수량 > 분당매도수량 * 1.2):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False
elif not (MFI > 40):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (-8.0 <= 시가등락율 and 시가등락율 < 10.0):
            매수 = False
        elif not (-4.0 <= 시가대비등락율 and 시가대비등락율 < 12.0):
            매수 = False
        elif not (5 < 분당순매수금액 and 분당순매수금액 < 450):
            매수 = False
        elif not (전일비 > 18 and 전일동시간비 > 18):
            매수 = False
        elif not (회전율 > 0.5):
            매수 = False
        elif not (당일거래대금 > 70 * 100):
            매수 = False
        elif not (RSI > 42):
            매수 = False
        elif not (STOCHSK > 35):
            매수 = False
        elif not (체결강도 >= 120):
            매수 = False
        elif not (분봉저가 > 분봉저가N(1)):
            매수 = False
    else:
        if not (-5.0 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (-3.0 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (12 < 분당순매수금액 and 분당순매수금액 < 650):
            매수 = False
        elif not (전일비 > 15 and 전일동시간비 > 15):
            매수 = False
        elif not (회전율 > 0.35):
            매수 = False
        elif not (당일거래대금 > 130 * 100):
            매수 = False
        elif not (RSI > 40):
            매수 = False
        elif not (STOCHSK > 30):
            매수 = False
        elif not (체결강도 >= 110):
            매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_900_1518_RR_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
RSI과매수여부 = (RSI > 70)
스토캐스틱과매수여부 = (STOCHSK > 80)


# ================================
#  매도 조건 (RSI 반전)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 4.0:
    매도 = True

# 손절 조건
elif 수익률 <= -2.0:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 5.5 and 최고수익률 * 0.73 >= 수익률:
    매도 = True

# RSI 과매수
elif RSI과매수여부 and 수익률 > 1.2:
    매도 = True

# 스토캐스틱 과매수
elif 스토캐스틱과매수여부 and 수익률 > 1.5:
    매도 = True

# RSI 재하락
elif RSI < 35 and RSI_N(1) >= 35 and 최고수익률 > 2.0:
    매도 = True

# MFI 하락
elif MFI < 40 and MFI_N(1) >= 40 and 수익률 > 0.5:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.6:
        매도 = True
    elif 수익률 < -1.2 and 체결강도 < 92:
        매도 = True
    elif 최고수익률 > 3.2 and RSI < 50 and RSI_N(1) >= 50:
        매도 = True
else:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.53:
        매도 = True
    elif 수익률 < -1.0 and 체결강도 < 92:
        매도 = True
    elif 최고수익률 > 2.8 and RSI < 50 and RSI_N(1) >= 50:
        매도 = True

# 보유시간 초과
if 보유시간 > 145:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
