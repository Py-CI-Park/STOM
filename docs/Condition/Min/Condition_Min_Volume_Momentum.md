# 조건식 - 볼륨 급증 모멘텀 전략 (분봉)

- STOM 주식 자동거래에 사용하기 위한 분봉 데이터 조건식 문서
- Back_Testing_Guideline_Min.md 에 있는 조건식을 사용하여 만든 조건식

## 소개

이 문서는 분당거래대금과 거래량의 급증을 감지하여 모멘텀을 포착하는 분봉 기반 전략입니다.
평균 대비 폭발적인 거래량 증가와 가격 상승을 동시에 확인합니다.

## 전략 개요

**핵심 아이디어**:
1. 분당거래대금이 평균 대비 3배 이상 급증
2. 누적 분당매수수량이 매도수량을 크게 상회
3. 최고 분당거래대금 갱신
4. 등락율각도 급상승 확인

## 조건식

### 매수 조건식

```python
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000

if not (93000 <= 시분초 < 142000):
    매수 = False
elif not (2800 < 현재가 <= 24000):
    매수 = False
elif not (4.0 < 등락율 <= 22.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(60) * 3.0):
    매수 = False
elif not (분당거래대금 > 분당거래대금N(1) * 1.5):
    매수 = False
elif not (분당거래대금 >= 최고분당매수수량(30) * 현재가 / 1_000_000):
    매수 = False
elif not (누적분당매수수량(20) > 누적분당매도수량(20) * 2.0):
    매수 = False
elif not (체결강도 > 140):
    매수 = False
elif not (체결강도 > 체결강도평균(30) * 1.4):
    매수 = False
elif not (등락율각도(30) > 10):
    매수 = False
elif not (전일비각도(30) > 6):
    매수 = False
elif not (분당매수수량 > 매도총잔량 * 50 / 100):
    매수 = False
elif not (분당순매수금액 > 35):
    매수 = False
elif 시가총액 < 1800:
    if not (전일비 > 70):
        매수 = False
    elif not (회전율 > 2.8):
        매수 = False
    elif not (당일거래대금 > 380):
        매수 = False
elif 시가총액 < 3600:
    if not (전일비 > 52):
        매수 = False
    elif not (회전율 > 2.2):
        매수 = False
    elif not (당일거래대금 > 620):
        매수 = False
else:
    if not (전일비 > 38):
        매수 = False
    elif not (회전율 > 1.6):
        매수 = False
    elif not (당일거래대금 > 850):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.5:
    매도 = True
elif 수익률 <= -2.3:
    매도 = True
elif 최고수익률 - 수익률 >= 2.2:
    매도 = True
elif 분당거래대금 < 분당거래대금평균(60) * 0.5 and 수익률 > 0.6:
    매도 = True
elif 체결강도 < 체결강도평균(30) * 0.7 and 수익률 > 0.8:
    매도 = True
elif 등락율각도(20) < 3 and 수익률 > 1.5:
    매도 = True
elif 보유시간 > 130:
    매도 = True
elif 시가총액 < 3600:
    if (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.7 and 수익률 > 0.3:
        매도 = True
    elif 분당매도수량 > 최고분당매도수량(30) * 0.9 and 수익률 > 0.5:
        매도 = True
    elif 누적분당매도수량(10) > 누적분당매수수량(10) * 1.5 and 수익률 > 0.4:
        매도 = True
else:
    if (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.58 and 수익률 > 0.3:
        매도 = True
    elif 분당매도수량 > 최고분당매도수량(30) * 0.85 and 수익률 > 0.5:
        매도 = True
    elif 누적분당매도수량(10) > 누적분당매수수량(10) * 1.3 and 수익률 > 0.4:
        매도 = True
```
