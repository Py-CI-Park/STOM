# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Min]] 을(를) 기반으로 작성한 Min 조건식
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장중(09:30~14:20) 볼륨 급증 모멘텀 전략**을 정의한다.

- **대상 시간 구간**: 09:30:00 ~ 14:20:00 (장중)
- **대상 종목**: 시가총액 차등 (1,800억/3,600억 기준), 가격대 2,800원~24,000원
- **전략 타입**: 거래량 폭발 + 모멘텀 포착 (Volume Momentum)
- **핵심 변수**: 분당거래대금, 분당매수수량, 체결강도, 등락율각도, 전일비각도
- **업데이트 이력**:
  - 초기 문서 작성

# 조건식 - 볼륨 급증 모멘텀 전략 (분봉)

## 소개

이 문서는 분당거래대금과 거래량의 급증을 감지하여 모멘텀을 포착하는 분봉 기반 전략입니다.
평균 대비 폭발적인 거래량 증가와 가격 상승을 동시에 확인합니다.

## 전략 개요

**핵심 아이디어**:
1. 분당거래대금이 평균 대비 3배 이상 급증
2. 누적 분당매수수량이 매도수량을 크게 상회
3. 최고 분당거래대금 갱신
4. 등락율각도 급상승 확인

## 조건식

### 매수 조건식

```python
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000

if not (93000 <= 시분초 < 142000):
    매수 = False
elif not (2800 < 현재가 <= 24000):
    매수 = False
elif not (4.0 < 등락율 <= 22.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(60) * 3.0):
    매수 = False
elif not (분당거래대금 > 분당거래대금N(1) * 1.5):
    매수 = False
elif not (분당거래대금 >= 최고분당매수수량(30) * 현재가 / 1_000_000):
    매수 = False
elif not (누적분당매수수량(20) > 누적분당매도수량(20) * 2.0):
    매수 = False
elif not (체결강도 > 140):
    매수 = False
elif not (체결강도 > 체결강도평균(30) * 1.4):
    매수 = False
elif not (등락율각도(30) > 10):
    매수 = False
elif not (전일비각도(30) > 6):
    매수 = False
elif not (분당매수수량 > 매도총잔량 * 50 / 100):
    매수 = False
elif not (분당순매수금액 > 35):
    매수 = False
elif 시가총액 < 1800:
    if not (전일비 > 70):
        매수 = False
    elif not (회전율 > 2.8):
        매수 = False
    elif not (당일거래대금 > 380):
        매수 = False
elif 시가총액 < 3600:
    if not (전일비 > 52):
        매수 = False
    elif not (회전율 > 2.2):
        매수 = False
    elif not (당일거래대금 > 620):
        매수 = False
else:
    if not (전일비 > 38):
        매수 = False
    elif not (회전율 > 1.6):
        매수 = False
    elif not (당일거래대금 > 850):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.5:
    매도 = True
elif 수익률 <= -2.3:
    매도 = True
elif 최고수익률 - 수익률 >= 2.2:
    매도 = True
elif 분당거래대금 < 분당거래대금평균(60) * 0.5 and 수익률 > 0.6:
    매도 = True
elif 체결강도 < 체결강도평균(30) * 0.7 and 수익률 > 0.8:
    매도 = True
elif 등락율각도(20) < 3 and 수익률 > 1.5:
    매도 = True
elif 보유시간 > 130:
    매도 = True
elif 시가총액 < 3600:
    if (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.7 and 수익률 > 0.3:
        매도 = True
    elif 분당매도수량 > 최고분당매도수량(30) * 0.9 and 수익률 > 0.5:
        매도 = True
    elif 누적분당매도수량(10) > 누적분당매수수량(10) * 1.5 and 수익률 > 0.4:
        매도 = True
else:
    if (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.58 and 수익률 > 0.3:
        매도 = True
    elif 분당매도수량 > 최고분당매도수량(30) * 0.85 and 수익률 > 0.5:
        매도 = True
    elif 누적분당매도수량(10) > 누적분당매수수량(10) * 1.3 and 수익률 > 0.4:
        매도 = True
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
