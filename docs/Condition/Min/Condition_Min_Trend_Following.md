## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **트렌드 추종 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 15:18:00 (장중 전체)
- **패턴**: 명확한 상승 추세를 확인하고 추세에 편승
- **원리**: 이동평균선 정배열과 각도를 이용한 추세 확인
- **특징**:
  - 단기/중기/장기 이동평균선 정배열
  - 등락율각도가 지속적으로 양수 유지
  - 분당거래대금이 평균 이상 유지
  - 현재가가 이동평균선 위에 위치

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `M`: Min (분봉)을 의미합니다.
   - `TF`: Trend Following (트렌드 추종)을 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_M_900_1518_TF_B`: 900-1518 트렌드 추종 패턴의 매수 조건식
   - `C_M_900_1518_TF_S`: 900-1518 트렌드 추종 패턴의 매도 조건식

# Condition - Min Trend Following (트렌드 추종)

## 전략 개요

### 패턴 인식 로직

**트렌드 추종 (Trend Following) 정의**:
- 이동평균선이 정배열을 이루며 상승 추세 확인
- 단기(20분) > 중기(60분) > 장기(120분) 이동평균선 정배열
- 각 이동평균선이 전봉 대비 상승 중
- 등락율각도가 양수를 유지하며 추세 지속

**추세 강도 확인**:
- 등락율각도(30) > 8도 이상
- 당일거래대금각도(30) > 7도 이상
- 현재가 > 이동평균(20) > 이동평균(60) > 이동평균(120)
- 이동평균(20)이 상승 중 (이동평균(20) > 이동평균(20, 1))

**거래량 확인**:
- 분당거래대금이 평균(30분) 이상
- 체결강도가 100 이상 유지
- 전일비 및 전일동시간비가 양수

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 등락율각도 최소 10도 이상
- 체결강도 최소 110 이상
- 회전율 0.8% 이상

**대형주 (시가총액 >= 2500억)**:
- 등락율각도 최소 8도 이상
- 체결강도 최소 100 이상
- 회전율 0.5% 이상

## 조건식

### 매수 조건식 - C_M_900_1518_TF_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
이평정배열여부 = (이동평균(20) > 이동평균(60) and 이동평균(60) > 이동평균(120)) if 데이터길이 > 120 else False
이평상승여부 = (이동평균(20) > 이동평균(20, 1) and 이동평균(60) > 이동평균(60, 1)) if 데이터길이 > 60 else False
강한추세여부 = (등락율각도(30) > 8 and 당일거래대금각도(30) > 7)


# ================================
#  매수 조건 (트렌드 추종)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (1.5 < 등락율 <= 20.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 120):
    매수 = False

# 추세 확인
elif not 이평정배열여부:
    매수 = False
elif not 이평상승여부:
    매수 = False
elif not (현재가 > 이동평균(20)):
    매수 = False
elif not 강한추세여부:
    매수 = False

# 거래량 및 체결강도
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False
elif not (체결강도 >= 100):
    매수 = False
elif not (전일비 > 0 and 전일동시간비 > 0):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (0.5 <= 시가등락율 and 시가등락율 < 10.0):
            매수 = False
        elif not (0.8 <= 시가대비등락율 and 시가대비등락율 < 12.0):
            매수 = False
        elif not (10 < 분당순매수금액 and 분당순매수금액 < 500):
            매수 = False
        elif not (전일비 > 30 and 전일동시간비 > 30):
            매수 = False
        elif not (회전율 > 0.8):
            매수 = False
        elif not (당일거래대금 > 100 * 100):
            매수 = False
        elif not (등락율각도(30) > 10):
            매수 = False
        elif not (체결강도 >= 110):
            매수 = False
        elif not (분당매수수량 > 매도총잔량 * 0.2):
            매수 = False
    else:
        if not (0.5 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (1.0 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (20 < 분당순매수금액 and 분당순매수금액 < 800):
            매수 = False
        elif not (전일비 > 20 and 전일동시간비 > 20):
            매수 = False
        elif not (회전율 > 0.5):
            매수 = False
        elif not (당일거래대금 > 200 * 100):
            매수 = False
        elif not (등락율각도(30) > 8):
            매수 = False
        elif not (체결강도 >= 100):
            매수 = False
        elif not (분당매수수량 > 매도총잔량 * 0.15):
            매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_900_1518_TF_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
추세약화여부 = (등락율각도(30) < 3 and 등락율각도(60) > 10)
이평이탈여부 = (현재가 < 이동평균(20) and 현재가N(1) >= 이동평균(20, 1))


# ================================
#  매도 조건 (트렌드 추종)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 4.0:
    매도 = True

# 손절 조건
elif 수익률 <= -2.5:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 6.0 and 최고수익률 * 0.70 >= 수익률:
    매도 = True

# 추세 약화
elif 추세약화여부 and 최고수익률 > 2.5:
    매도 = True

# 이동평균 이탈
elif 이평이탈여부 and 수익률 > 0.5:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.6:
        매도 = True
    elif 수익률 < -1.5 and 체결강도 < 체결강도평균(30):
        매도 = True
    elif 최고수익률 > 3.5 and 현재가 < 이동평균(60):
        매도 = True
else:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.5:
        매도 = True
    elif 수익률 < -1.5 and 체결강도 < 체결강도평균(30):
        매도 = True
    elif 최고수익률 > 3.0 and 현재가 < 이동평균(60):
        매도 = True

# 보유시간 초과
if 보유시간 > 180:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
