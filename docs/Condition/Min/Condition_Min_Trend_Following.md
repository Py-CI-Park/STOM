## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장중 전체 트렌드 추종 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 15:18:00 (장중 전체)
- **대상 종목**: 시가총액 2,500억 기준 대형/소형 분류, 가격대 1,000원~30,000원
- **전략 타입**: 이동평균선 정배열 기반 트렌드 추종 (Trend Following)
- **핵심 변수**: 이동평균(20/60/120), 등락율각도(30), 당일거래대금각도(30), 체결강도, 분당거래대금, 회전율
- **업데이트 이력**:
  - 초기 문서 작성

**전략의 핵심 개념**:
- **패턴**: 명확한 상승 추세를 확인하고 추세에 편승
- **원리**: 이동평균선 정배열과 각도를 이용한 추세 확인
- **특징**:
  - 단기/중기/장기 이동평균선 정배열
  - 등락율각도가 지속적으로 양수 유지
  - 분당거래대금이 평균 이상 유지
  - 현재가가 이동평균선 위에 위치

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

# Condition - Min Trend Following (트렌드 추종)

## 전략 개요

### 패턴 인식 로직

**트렌드 추종 (Trend Following) 정의**:
- 이동평균선이 정배열을 이루며 상승 추세 확인
- 단기(20분) > 중기(60분) > 장기(120분) 이동평균선 정배열
- 각 이동평균선이 전봉 대비 상승 중
- 등락율각도가 양수를 유지하며 추세 지속

**추세 강도 확인**:
- 등락율각도(30) > 8도 이상
- 당일거래대금각도(30) > 7도 이상
- 현재가 > 이동평균(20) > 이동평균(60) > 이동평균(120)
- 이동평균(20)이 상승 중 (이동평균(20) > 이동평균(20, 1))

**거래량 확인**:
- 분당거래대금이 평균(30분) 이상
- 체결강도가 100 이상 유지
- 전일비 및 전일동시간비가 양수

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 등락율각도 최소 10도 이상
- 체결강도 최소 110 이상
- 회전율 0.8% 이상

**대형주 (시가총액 >= 2500억)**:
- 등락율각도 최소 8도 이상
- 체결강도 최소 100 이상
- 회전율 0.5% 이상

## 조건식

### 매수 조건식 - C_M_900_1518_TF_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
이평정배열여부 = (이동평균(20) > 이동평균(60) and 이동평균(60) > 이동평균(120)) if 데이터길이 > 120 else False
이평상승여부 = (이동평균(20) > 이동평균(20, 1) and 이동평균(60) > 이동평균(60, 1)) if 데이터길이 > 60 else False
강한추세여부 = (등락율각도(30) > 8 and 당일거래대금각도(30) > 7)


# ================================
#  매수 조건 (트렌드 추종)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (1.5 < 등락율 <= 20.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 120):
    매수 = False

# 추세 확인
elif not 이평정배열여부:
    매수 = False
elif not 이평상승여부:
    매수 = False
elif not (현재가 > 이동평균(20)):
    매수 = False
elif not 강한추세여부:
    매수 = False

# 거래량 및 체결강도
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False
elif not (체결강도 >= 100):
    매수 = False
elif not (전일비 > 0 and 전일동시간비 > 0):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (0.5 <= 시가등락율 and 시가등락율 < 10.0):
            매수 = False
        elif not (0.8 <= 시가대비등락율 and 시가대비등락율 < 12.0):
            매수 = False
        elif not (10 < 분당순매수금액 and 분당순매수금액 < 500):
            매수 = False
        elif not (전일비 > 30 and 전일동시간비 > 30):
            매수 = False
        elif not (회전율 > 0.8):
            매수 = False
        elif not (당일거래대금 > 100 * 100):
            매수 = False
        elif not (등락율각도(30) > 10):
            매수 = False
        elif not (체결강도 >= 110):
            매수 = False
        elif not (분당매수수량 > 매도총잔량 * 0.2):
            매수 = False
    else:
        if not (0.5 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (1.0 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (20 < 분당순매수금액 and 분당순매수금액 < 800):
```
