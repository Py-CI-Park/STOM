## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **분봉 패턴 인식 전략**을 정의한다.

- **대상 시간 구간**: 09:00 ~ 15:18 (장 전체)
- **대상 종목**: 가격대 900~45,000원, 시가총액 차등 (1,500억/3,000억 기준)
- **전략 타입**: 분봉 캔들 패턴 인식 및 반전 포착 전략
- **핵심 변수**: 분봉실체, 분봉전체, 실체비율, 양봉/강한양봉 패턴
- **업데이트 이력**:
  - 초기 문서 작성: 분봉 패턴 인식 전략

## 조건식

### 매수 조건식
```python
# 패턴 정의
분봉실체 = abs(현재가 - 분봉시가)
분봉전체 = 분봉고가 - 분봉저가
실체비율 = 분봉실체 / (분봉전체 or 1)

# 양봉 확인
양봉 = (현재가 > 분봉시가)
강한양봉 = (양봉 and 실체비율 > 0.6)

# 해머형 (하락 후 반등)
하락추세 = (분봉고가N(1) > 분봉고가N(2) > 분봉고가N(3))
긴아래꼬리 = ((분봉저가 < 분봉시가) and ((분봉시가 - 분봉저가) > 분봉실체 * 2))
해머패턴 = (긴아래꼬리 and 양봉)

# 삼병형 (연속 상승)
삼병패턴 = (현재가 > 분봉시가 and 현재가N(1) > 분봉시가N(1) and 현재가N(2) > 분봉시가N(2))

if not (900 <= 시분초 <= 1518):
    매수 = False
elif not (1050 < 현재가 <= 47500):
    매수 = False
elif not (0.95 < 등락율 <= 27.0):
    매수 = False
elif 시가총액 < 1500:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 40 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.6):
        매수 = False
    elif not (체결강도 > 130):
        매수 = False
    elif not (전일비 > 29):
        매수 = False
    elif not (등락율각도(15) > 4):
        매수 = False
    elif not (MACD > 0):
        매수 = False

elif 시가총액 < 3000:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 42 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.5):
        매수 = False
    elif not (체결강도 > 126):
        매수 = False
    elif not (전일비 > 33):
        매수 = False
    elif not (등락율각도(15) > 5):
        매수 = False
    elif not (MACD > 0):
        매수 = False

else:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 44 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.42):
        매수 = False
    elif not (체결강도 > 122):
        매수 = False
    elif not (전일비 > 38):
        매수 = False
    elif not (등락율각도(15) > 6):
        매수 = False
    elif not (MACD > 0):
        매수 = False
```

### 매도 조건식
```python
# 매도 패턴
분봉실체_현재 = abs(현재가 - 분봉시가)
음봉 = (현재가 < 분봉시가)
강한음봉 = (음봉 and (분봉실체_현재 / (분봉고가 - 분봉저가 or 1) > 0.6))

# 긴위꼬리 (상승 후 반락)
긴위꼬리 = ((분봉고가 > 현재가) and ((분봉고가 - 현재가) > 분봉실체_현재 * 2))
유성패턴 = (긴위꼬리 and 음봉)

# 삼병 하락
삼병하락 = (현재가 < 분봉시가 and 현재가N(1) < 분봉시가N(1) and 현재가N(2) < 분봉시가N(2))

if 등락율 > 29.8:
    매도 = True
elif 수익률 >= 5.8:
    매도 = True
elif 수익률 <= -2.3:
    매도 = True
elif 최고수익률 - 수익률 >= 3.8:
    매도 = True
elif 보유시간 > 145:
    매도 = True
elif 유성패턴:
    if RSI > 69:
        매도 = True
elif 삼병하락 and 강한음봉:
    매도 = True
elif MACD < 0 and MACD_N(1) >= 0:
    매도 = True
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화
