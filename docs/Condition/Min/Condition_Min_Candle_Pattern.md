## 소개
**조건식 이름**: 분봉 패턴 인식 전략
**거래 시간**: 900 ~ 1518
**목표**: 분봉 캔들 패턴으로 반전 포착

## 조건식

### 매수 조건식
```python
# 패턴 정의
분봉실체 = abs(현재가 - 분봉시가)
분봉전체 = 분봉고가 - 분봉저가
실체비율 = 분봉실체 / (분봉전체 or 1)

# 양봉 확인
양봉 = (현재가 > 분봉시가)
강한양봉 = (양봉 and 실체비율 > 0.6)

# 해머형 (하락 후 반등)
하락추세 = (분봉고가N(1) > 분봉고가N(2) > 분봉고가N(3))
긴아래꼬리 = ((분봉저가 < 분봉시가) and ((분봉시가 - 분봉저가) > 분봉실체 * 2))
해머패턴 = (긴아래꼬리 and 양봉)

# 삼병형 (연속 상승)
삼병패턴 = (현재가 > 분봉시가 and 현재가N(1) > 분봉시가N(1) and 현재가N(2) > 분봉시가N(2))

if not (900 <= 시분초 <= 1518):
    매수 = False
elif not (1050 < 현재가 <= 47500):
    매수 = False
elif not (0.95 < 등락율 <= 27.0):
    매수 = False
elif 시가총액 < 1500:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 40 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.6):
        매수 = False
    elif not (체결강도 > 130):
        매수 = False
    elif not (전일비 > 29):
        매수 = False
    elif not (등락율각도(15) > 4):
        매수 = False
    elif not (MACD > 0):
        매수 = False

elif 시가총액 < 3000:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 42 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.5):
        매수 = False
    elif not (체결강도 > 126):
        매수 = False
    elif not (전일비 > 33):
        매수 = False
    elif not (등락율각도(15) > 5):
        매수 = False
    elif not (MACD > 0):
        매수 = False

else:
    if not (해머패턴 or (삼병패턴 and 강한양봉)):
        매수 = False
    elif not (RSI > 44 and RSI < 64):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(31) * 1.42):
        매수 = False
    elif not (체결강도 > 122):
        매수 = False
    elif not (전일비 > 38):
        매수 = False
    elif not (등락율각도(15) > 6):
        매수 = False
    elif not (MACD > 0):
        매수 = False
```

### 매도 조건식
```python
# 매도 패턴
분봉실체_현재 = abs(현재가 - 분봉시가)
음봉 = (현재가 < 분봉시가)
강한음봉 = (음봉 and (분봉실체_현재 / (분봉고가 - 분봉저가 or 1) > 0.6))

# 긴위꼬리 (상승 후 반락)
긴위꼬리 = ((분봉고가 > 현재가) and ((분봉고가 - 현재가) > 분봉실체_현재 * 2))
유성패턴 = (긴위꼬리 and 음봉)

# 삼병 하락
삼병하락 = (현재가 < 분봉시가 and 현재가N(1) < 분봉시가N(1) and 현재가N(2) < 분봉시가N(2))

if 등락율 > 29.8:
    매도 = True
elif 수익률 >= 5.8:
    매도 = True
elif 수익률 <= -2.3:
    매도 = True
elif 최고수익률 - 수익률 >= 3.8:
    매도 = True
elif 보유시간 > 145:
    매도 = True
elif 유성패턴:
    if RSI > 69:
        매도 = True
elif 삼병하락 and 강한음봉:
    매도 = True
elif MACD < 0 and MACD_N(1) >= 0:
    매도 = True
```
