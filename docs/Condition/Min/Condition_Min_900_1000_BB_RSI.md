## 소개

이 문서는 STOM 주식 자동거래 시스템에서 09:00:00 ~ 10:00:00 구간의 볼린저밴드와 RSI를 활용한 분봉 전략을 연구하기 위한 조건식과 최적화 범위를 제공합니다. 장 시작 1시간 동안 볼린저밴드 하단 터치 후 반등과 RSI 과매도 구간 탈출을 포착하는 역추세 전략입니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장 초반 볼린저밴드 + RSI 역추세 전략**을 정의한다.

- **대상 시간 구간**: 09:00:00 ~ 10:00:00 (장 시작 1시간)
- **대상 종목**: 시가총액 3,000억 미만, 가격대 1,000원~30,000원
- **전략 타입**: 볼린저밴드 하단 반등 + RSI 과매도 탈출 (Mean Reversion)
- **핵심 변수**: BBU, BBM, BBL, RSI, MACD, 이동평균(20/60), 체결강도, 분당거래대금
- **업데이트 이력**:
  - 초기 문서 작성

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.
- 분봉 데이터 특성상 시간 단위는 분(minute)이며, 틱 데이터보다 노이즈가 적고 추세 파악에 유리합니다.
- 최적화 조건식은 `self.vars[i]`로 치환하여 탐색 범위를 별도로 정의합니다.
- 최적화 범위 포맷: `self.vars[정수] = [[시작값, 종료값, 간격], 최적값]`
- 변수/함수 표기 및 사용 규칙은 Back_Testing_Guideline_Min.md를 따릅니다.

### 저장 이름 규칙

조건식 저장 및 관리 편의를 위해 다음 규칙을 사용합니다.

1. 기본 규칙:
   - `C`: Condition
   - `M`: Minute (분봉)
   - `900_1000`: 시간대 (09:00~10:00)
   - `BR`: Bollinger Bands + RSI (볼린저밴드 + RSI 전략)
   - `B`/`S`: 매수/매도 조건식
   - `BO`/`SO`: 매수/매도 최적화 조건식
   - `BOR`/`SOR`: 매수/매도 최적화 범위
   - `OR`: 매수/매도 통합 최적화 범위
   - `GAR`: GA 최적화 범위

2. 예시:
   - `C_M_900_1000_BR_B`, `C_M_900_1000_BR_S`
   - `C_M_900_1000_BR_BO`, `C_M_900_1000_BR_SO`
   - `C_M_900_1000_BR_BOR`, `C_M_900_1000_BR_SOR`, `C_M_900_1000_BR_OR`, `C_M_900_1000_BR_GAR`

# Condition - Min 900_1000 BB_RSI

## 전략 개요

장 시작 1시간(09:00~10:00) 동안 볼린저밴드 하단 터치 후 반등과 RSI 과매도 구간에서의 반전을 포착하는 분봉 기반 역추세 전략입니다.

주요 특징:
- **볼린저밴드 전략**: 하단 밴드 터치 후 복귀 신호 포착
- **RSI 반전**: 과매도(RSI < 30) 구간에서 반등 확인
- **MACD 보조**: MACD 골든크로스로 상승 모멘텀 확인
- **분봉 캔들 패턴**: 분봉고가/저가/시가 분석으로 반전 패턴 검증
- **거래량 필터**: 분당거래대금 및 체결강도로 신뢰도 향상

```python
#  공통 계산 지표
# ================================
전일종가              = 현재가 / (1 + (등락율 / 100))                                     # 단위: 원
시가등락율            = ((시가 - 전일종가) / 전일종가) * 100                               # 단위: 퍼센트
시가대비등락율        = ((현재가 - 시가) / 시가) * 100                                     # 단위: 퍼센트
분당순매수금액        = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000                  # 단위: 백만원
분봉몸통비율          = abs(현재가 - 분봉시가) / (분봉고가 - 분봉저가) if (분봉고가 - 분봉저가) != 0 else 0
분봉아래꼬리비율      = (min(현재가, 분봉시가) - 분봉저가) / (분봉고가 - 분봉저가) if (분봉고가 - 분봉저가) != 0 else 0

# 볼린저밴드 조건
BB하단터치여부        = (현재가N(1) <= BBL_N(1) and 현재가 > BBL)                          # 전봉에서 하단 터치, 현재 복귀
BB중간선상향여부      = (BBM > BBM_N(1))                                                   # 중간선 상승
BB밴드폭확대여부      = (BBU - BBL) > (BBU_N(1) - BBL_N(1))                                # 변동성 확대

# RSI 조건
RSI과매도탈출여부     = (RSI_N(1) < 30 and RSI > 35)                                      # RSI 과매도 탈출
RSI상승여부           = (RSI > RSI_N(1) and RSI_N(1) > RSI_N(2))                          # RSI 연속 상승

# MACD 조건
MACD골든크로스여부    = (MACD > MACDS and MACD_N(1) <= MACDS_N(1))                        # MACD 골든크로스
MACD히스토그램양전환  = (MACDH > 0 and MACDH_N(1) <= 0)                                    # 히스토그램 양전환

# 이동평균 조건
이평정배열여부        = (이동평균(5) > 이동평균(20) and 이동평균(20) > 이동평균(60))      # 이평선 정배열
가격이평상회여부      = (현재가 > 이동평균(5))                                             # 가격이 단기 이평 위


# ================================
#  매수 조건
# ================================
매수 = True

# 공통 필터 – 관심종목, 시간 범위, 가격/VI 안전장치
if not (관심종목 == 1):
    매수 = False
elif not (90000 <= 시분초 < 100000):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 10):                          # 최소 10분봉 데이터 확보
    매수 = False

# 등락율 필터
elif not (-5.0 < 등락율 <= 10.0):
    매수 = False
elif not (고저평균대비등락율 > -3.0):
    매수 = False

# 볼린저밴드 필터
elif not BB하단터치여부:
    매수 = False
elif not BB중간선상향여부:
    매수 = False

# RSI 필터
elif not RSI과매도탈출여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (-3.0 <= 시가등락율 < 8.0):
        매수 = False
    elif not (-2.0 <= 시가대비등락율 < 8.0):
        매수 = False
    elif not (-50 < 분당순매수금액 < 500):
        매수 = False
    elif not (분봉아래꼬리비율 >= 0.3):                # 아래꼬리가 30% 이상
        매수 = False
    elif not (현재가 > 분봉저가 + (분봉고가 - 분봉저가) * 0.4):
        매수 = False
    elif not (전일비 > 0 and 전일동시간비 > 0):
        매수 = False
    elif not (회전율 > 0.5):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (등락율각도(10) > -10 and 등락율각도(10) < 20):
        매수 = False
    elif not (당일거래대금각도(10) > -5):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(20) * 1.2):
        매수 = False
    elif not (분당매수수량 > 매도총잔량 * 0.20):
        매수 = False
    elif not (매도총잔량 < 매수총잔량 * 2.5):
        매수 = False
    elif not (체결강도 >= 70 and 체결강도 <= 300):
        매수 = False
    elif not (체결강도 > 체결강도평균(10)):
        매수 = False
    # MACD 추가 필터
    elif not MACD골든크로스여부:
        매수 = False
    # 스토캐스틱 필터 (옵션)
    elif 'STOCHSK' in globals():
        if not (STOCHSK > STOCHSD and STOCHSK_N(1) <= STOCHSD_N(1)):
            매수 = False
    # MFI 필터 (옵션)
    elif 'MFI' in globals():
        if not (20 <= MFI <= 50):
            매수 = False

else:
    if not (-3.0 <= 시가등락율 < 8.0):
        매수 = False
    elif not (-2.0 <= 시가대비등락율 < 8.0):
        매수 = False
    elif not (-50 < 분당순매수금액 < 500):
        매수 = False
    elif not (분봉아래꼬리비율 >= 0.3):
        매수 = False
    elif not (현재가 > 분봉저가 + (분봉고가 - 분봉저가) * 0.4):
        매수 = False
    elif not (전일비 > 0 and 전일동시간비 > 0):
        매수 = False
    elif not (회전율 > 0.5):
        매수 = False
    elif not (당일거래대금 > 50 * 100):
        매수 = False
    elif not (등락율각도(10) > -10 and 등락율각도(10) < 20):
        매수 = False
    elif not (당일거래대금각도(10) > -5):
        매수 = False
    elif not (분당거래대금 > 분당거래대금평균(20) * 1.2):
        매수 = False
    elif not (분당매수수량 > 매도총잔량 * 0.20):
        매수 = False
    elif not (매도총잔량 < 매수총잔량 * 2.5):
        매수 = False
    elif not (체결강도 >= 70 and 체결강도 <= 300):
        매수 = False
    elif not (체결강도 > 체결강도평균(10)):
        매수 = False
    elif not MACD골든크로스여부:
        매수 = False
    elif 'STOCHSK' in globals():
        if not (STOCHSK > STOCHSD and STOCHSK_N(1) <= STOCHSD_N(1)):
```

### 매도 조건식 - C_M_900_1000_BR_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율        = ((현재가 - 시가) / 시가) * 100
분봉등락율            = ((현재가 - 분봉시가) / 분봉시가) * 100 if 분봉시가 != 0 else 0
BB상단도달여부        = (현재가 >= BBU or 현재가N(1) >= BBU_N(1))                         # BB 상단 도달
RSI과매수여부         = (RSI >= 70)                                                        # RSI 과매수
MACD데드크로스여부    = (MACD < MACDS and MACD_N(1) >= MACDS_N(1))                        # MACD 데드크로스
이평역전여부          = (현재가 < 이동평균(5)) or (이동평균(5) < 이동평균(20))
급락신호              = (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.5 and 분봉등락율 < -1.0

매도 = False

# 상한가 부근 안전장치
if 등락율 > 29.5:
    매도 = True

# 목표 수익/손절
elif 수익률 >= 3.5:
    매도 = True
elif 수익률 <= -2.5:
    매도 = True

# 트레일링 스톱
elif 최고수익률 > 4.0 and 최고수익률 - 수익률 >= 2.0:
    매도 = True

# 시간별 보유 청산 (분봉 기준)
elif 보유시간 > 30 and 수익률 < 1.0:                   # 30분 이상 보유 시 1% 미만이면 청산
    매도 = True
elif 보유시간 > 60 and 수익률 < 2.0:                   # 60분 이상 보유 시 2% 미만이면 청산
    매도 = True

# 볼린저밴드 기반 청산
elif BB상단도달여부 and 수익률 > 1.5:
    매도 = True

# RSI 과매수 청산
elif RSI과매수여부 and 수익률 > 1.0:
    매도 = True

# MACD 데드크로스 청산
elif MACD데드크로스여부 and 수익률 > 0.5:
    매도 = True

# 위험 신호 기반 조기 청산
elif 급락신호:
    매도 = True
elif 이평역전여부 and 체결강도 < 체결강도평균(10) * 0.8:
    매도 = True

# 분봉 캔들 패턴 기반 청산
elif 분봉등락율 < -2.0 and 수익률 < 1.5:
    매도 = True

# 스토캐스틱 기반 청산 (옵션)
elif 'STOCHSK' in globals():
    if STOCHSK > 80 and 수익률 > 1.5:
        매도 = True
    elif STOCHSK < STOCHSD and STOCHSK_N(1) >= STOCHSD_N(1) and 수익률 > 0.5:
        매도 = True

# MFI 기반 청산 (옵션)
elif 'MFI' in globals():
    if MFI > 80 and 수익률 > 1.0:
        매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

## 최적화 조건식

### 매수 최적화 조건식 - C_M_900_1000_BR_BO

```python
# ================================
#  공통 계산 지표
# ================================
전일종가              = 현재가 / (1 + (등락율 / 100))
시가등락율            = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율        = ((현재가 - 시가) / 시가) * 100
분당순매수금액        = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
```

## 조건식 개선 방향 연구

1) **볼린저밴드 폭 분석**
   - BB 밴드폭이 확대될 때와 축소될 때를 구분
   - 변동성 확대 시 진입 신중, 축소 후 확대 시 적극 진입

2) **RSI Divergence 감지**
   - 가격은 하락하나 RSI는 상승하는 강세 다이버전스 감지
   - 다이버전스 발생 시 매수 신호 강화

3) **MACD 히스토그램 분석**
   - MACDH의 크기 증가/감소를 통한 모멘텀 세기 판단
   - 히스토그램 감소 시 조기 청산 신호

4) **분봉 캔들 패턴 고도화**
   - 망치형, 역망치형, 도지형 등 반전 캔들 패턴 자동 감지
   - 패턴 매칭 시 매수 신호 가중치 부여

5) **시간대별 세분화**
   - 09:00~09:20 / 09:20~09:40 / 09:40~10:00 세분화
   - 각 구간별 다른 RSI/볼린저밴드 임계값 적용

6) **Volume Profile 분석**
   - 누적분당거래대금 분석으로 지지/저항 구간 파악
   - 지지 구간 근처에서 매수 신호 강화

7) **리스크 관리 강화**
   - 동시 보유 종목 수 제한 (최대 3~5개)
   - 일일 최대 손실 한도 설정 (-5% 도달 시 매매 중단)
   - 연속 손실 횟수 제한 (3연속 손실 시 당일 매매 중단)

8) **최적화 프로세스**
   - 1단계: 넓은 범위 탐색 (간격 크게)
   - 2단계: 좁은 범위 정밀 탐색
   - 3단계: GA 최적화
   - 4단계: Walk-Forward 검증
   - 5단계: 실전 투입 전 Paper Trading
