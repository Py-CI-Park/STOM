# Condition - Min RSI Reversal (RSI 반전)

## 전략 개요

### 패턴 인식 로직

**RSI 반전 (RSI Reversal) 정의**:
- RSI가 과매도 구간(30 이하)에서 반등
- 이전봉: RSI <= 30
- 현재봉: RSI > 40
- 스토캐스틱 %K도 동반 반등 (20 이하 → 30 이상)

**반등 확인**:
- 현재가 > 현재가N(1) (가격 상승)
- 체결강도 >= 110 (매수세 우위)
- 분당매수수량 > 분당매도수량 * 1.2
- 분봉고가 > 분봉고가N(1) (고가 갱신)

**추가 필터**:
- 등락율 -5% ~ 15% 범위
- MFI (Money Flow Index) > 40 (자금 유입)
- 분당거래대금이 평균 이상

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- RSI 반등폭 최소 12 이상 (30 이하 → 42 이상)
- 체결강도 최소 120 이상
- 스토캐스틱 %K > 35

**대형주 (시가총액 >= 2500억)**:
- RSI 반등폭 최소 10 이상 (30 이하 → 40 이상)
- 체결강도 최소 110 이상
- 스토캐스틱 %K > 30

## 조건식

### 매수 조건식 - C_M_900_1518_RR_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
RSI반등여부 = (RSI_N(1) <= 30 and RSI > 40)
스토캐스틱반등여부 = (STOCHSK_N(1) <= 20 and STOCHSK > 30)
가격반등여부 = (현재가 > 현재가N(1) and 분봉고가 > 분봉고가N(1))

# ================================
#  개선: 변동성 지표 추가
# ================================
ATR값 = ATR(14)                                                      # 개선1: ATR 변동성 측정
변동성확대여부 = (ATR값 > ATR_N(14, 1) * 1.2)                        # ATR 20% 이상 증가
BB폭 = (BBU(20, 2) - BBL(20, 2)) / BBM(20, 2) * 100                  # 개선2: 볼린저밴드 폭 (%)
BB확대여부 = (BB폭 > BB폭_N(1, 20, 2) * 1.15)                        # BB 폭 15% 이상 확대


# ================================
#  매수 조건 (RSI 반전)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (-5.0 < 등락율 <= 15.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# RSI 및 스토캐스틱 반등 확인
elif not RSI반등여부:
    매수 = False
elif not 스토캐스틱반등여부:
    매수 = False

# 가격 반등 확인
elif not 가격반등여부:
    매수 = False
elif not (현재가 > 분봉시가):
    매수 = False

# 체결강도 및 거래량
elif not (체결강도 >= 110):
    매수 = False
elif not (분당매수수량 > 분당매도수량 * 1.2):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False
elif not (MFI > 40):
    매수 = False

# 변동성 확인 (개선: 변동성 확대 구간 포착)
elif not 변동성확대여부:                      # 개선1: ATR 증가 확인
    매수 = False
elif not BB확대여부:                          # 개선2: BB 폭 확대 확인
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
```
