## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

# Condition - Min Moving Average Golden Cross (이동평균선 골든크로스)

## 전략 개요

### 패턴 인식 로직

**이동평균선 골든크로스 (MA Golden Cross) 정의**:
- 단기 이동평균(20분)이 장기 이동평균(60분)을 상향 돌파
- 이전봉: 이동평균(20) <= 이동평균(60)
- 현재봉: 이동평균(20) > 이동평균(60)
- 두 이동평균선 모두 상승 추세

**추세 확인**:
- 이동평균(20) > 이동평균(20, 1) (단기 이평 상승)
- 이동평균(60) > 이동평균(60, 1) (장기 이평 상승)
- 현재가 > 이동평균(20) > 이동평균(60)
- 등락율각도(30) > 6도

**거래량 확인**:
- 분당거래대금이 평균(60) 이상
- 분당거래대금이 전봉 대비 증가
- 체결강도가 105 이상
- 전일비 및 전일동시간비가 양수

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- 등락율각도(30) 최소 8도 이상
- 체결강도 최소 115 이상
- 분당거래대금이 평균 대비 1.2배 이상

**대형주 (시가총액 >= 2500억)**:
- 등락율각도(30) 최소 6도 이상
- 체결강도 최소 105 이상
- 분당거래대금이 평균 대비 1.1배 이상

## 조건식

### 매수 조건식 - C_M_900_1518_MAG_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
골든크로스여부 = (이동평균(20) > 이동평균(60) and 이동평균(20, 1) <= 이동평균(60, 1)) if 데이터길이 > 60 else False
이평상승여부 = (이동평균(20) > 이동평균(20, 1) and 이동평균(60) > 이동평균(60, 1)) if 데이터길이 > 60 else False
가격위치정상여부 = (현재가 > 이동평균(20) and 이동평균(20) > 이동평균(60)) if 데이터길이 > 60 else False


# ================================
#  매수 조건 (이동평균선 골든크로스)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (0.8 < 등락율 <= 18.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# 골든크로스 확인
elif not 골든크로스여부:
    매수 = False
elif not 이평상승여부:
    매수 = False
elif not 가격위치정상여부:
    매수 = False

# 추세 확인
elif not (등락율각도(30) > 6):
    매수 = False
elif not (당일거래대금각도(30) > 5):
    매수 = False

# 거래량 확인
elif not (분당거래대금 > 분당거래대금평균(60)):
    매수 = False
elif not (분당거래대금 > 분당거래대금N(1)):
    매수 = False
elif not (체결강도 >= 105):
    매수 = False
elif not (전일비 > 0 and 전일동시간비 > 0):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (0.5 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (1.0 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (10 < 분당순매수금액 and 분당순매수금액 < 550):
            매수 = False
        elif not (전일비 > 28 and 전일동시간비 > 28):
            매수 = False
        elif not (회전율 > 0.65):
            매수 = False
```
