# 조건식 - MFI 자금 흐름 전략 (분봉)

- STOM 주식 자동거래에 사용하기 위한 분봉 데이터 조건식 문서
- [[Back_Testing_Guideline_Min]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **MFI 자금 흐름 전략**을 정의한다.

- **대상 시간 구간**: 09:00 ~ 15:18 (장 전체)
- **대상 종목**: 전 종목 대상
- **전략 타입**: MFI 과매도/과매수 기반 자금 흐름 분석 전략
- **핵심 변수**: MFI, OBV, RSI, 거래대금, 체결강도
- **업데이트 이력**:
  - 초기 문서 작성: MFI 자금 흐름 전략

## 소개

이 문서는 MFI(Money Flow Index) 지표를 활용하여 자금 유입/유출을 분석하는 분봉 기반 전략입니다.
MFI가 과매도 구간에서 상승 전환할 때 매수하고, 과매수 구간에서 매도합니다.

## 전략 개요

**핵심 아이디어**:
1. MFI가 과매도 구간(30 이하)에서 상승 시작
2. OBV(On Balance Volume) 증가 확인
3. 거래대금 증가와 체결강도 상승 동반
4. RSI와 MFI의 다이버전스 활용

## 조건식

### 매수 조건식

```python
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000

if not (93000 <= 시분초 < 143000):
    매수 = False
elif not (2000 < 현재가 <= 23000):
    매수 = False
elif not (0.5 < 등락율 <= 17.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif not (MFI_N(1) <= 30 and MFI > 30):
    매수 = False
elif not (MFI > MFI_N(1)):
    매수 = False
elif not (OBV > OBV_N(1)):
    매수 = False
elif not (OBV > OBV_N(5)):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(30) * 1.5):
    매수 = False
elif not (체결강도 > 120):
    매수 = False
elif not (RSI > 35 and RSI < 70):
    매수 = False
elif not (분당매수수량 > 분당매도수량):
    매수 = False
elif not (분당순매수금액 > 22):
    매수 = False
elif 시가총액 < 1500:
    if not (전일비 > 55):
        매수 = False
    elif not (회전율 > 2.3):
        매수 = False
    elif not (당일거래대금 > 320):
        매수 = False
elif 시가총액 < 3200:
    if not (전일비 > 42):
        매수 = False
    elif not (회전율 > 1.8):
        매수 = False
    elif not (당일거래대금 > 550):
        매수 = False
else:
    if not (전일비 > 32):
        매수 = False
    elif not (회전율 > 1.4):
        매수 = False
    elif not (당일거래대금 > 750):
        매수 = False
```

### 매도 조건식

```python
if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 3.3:
    매도 = True
elif 수익률 <= -2.1:
    매도 = True
elif 최고수익률 - 수익률 >= 1.9:
    매도 = True
elif MFI > 75 and 수익률 > 0.9:
    매도 = True
elif MFI_N(1) >= MFI and MFI_N(2) < MFI_N(1) and 수익률 > 0.5:
    매도 = True
elif OBV < OBV_N(1) and OBV_N(1) < OBV_N(2) and 수익률 > 1.0:
    매도 = True
elif RSI > 73 and 수익률 > 1.2:
    매도 = True
elif 보유시간 > 140:
    매도 = True
elif 시가총액 < 3200:
    if 체결강도 < 체결강도평균(30) - 22 and 수익률 > 0.4:
        매도 = True
    elif (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.62 and 수익률 > 0.2:
        매도 = True
    elif 분당거래대금 < 분당거래대금평균(30) * 0.6 and 수익률 > 0.8:
        매도 = True
else:
    if 체결강도 < 체결강도평균(30) - 16 and 수익률 > 0.4:
        매도 = True
    elif (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.52 and 수익률 > 0.2:
        매도 = True
    elif 분당거래대금 < 분당거래대금평균(30) * 0.7 and 수익률 > 0.8:
        매도 = True
```

## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화
