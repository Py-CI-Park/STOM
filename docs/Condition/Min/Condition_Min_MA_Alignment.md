## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **장중 전체 이동평균선 정배열 전략**을 정의한다.

- **대상 시간 구간**: 장중 전체 (시간 필터링 필요)
- **대상 종목**: 관심종목, 가격대 및 등락율 필터링 적용
- **전략 타입**: 이동평균선 정배열 + 추세 추종 (MA Alignment)
- **핵심 변수**: 이동평균(5/20/60), 분당거래대금, 체결강도, RSI
- **업데이트 이력**:
  - 초기 문서 작성

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 이동평균선 정배열 전략(Moving Average Alignment)을 위한 조건식과 최적화 범위를 제공합니다. 단기/중기/장기 이동평균선이 정배열을 이루며 상승하는 패턴을 포착하는 추세 추종 전략입니다.

## 가이드라인

- 모든 조건식은 Python 스타일의 의사코드로 작성되어 있으며, STOM 백테스터가 인식하는 변수/함수 규칙을 따릅니다.
- 최적화 조건식은 `self.vars[i]`로 치환하여 탐색 범위를 별도로 정의합니다.
- 최적화 범위 포맷: `self.vars[정수] = [[시작값, 종료값, 간격], 최적값]`
- 변수/함수 표기 및 사용 규칙은 Back_Testing_Guideline_Min.md를 따릅니다.

### 저장 이름 규칙

1. 기본 규칙:
   - `C`: Condition
   - `M`: Min (분봉)
   - `MAA`: Moving Average Alignment
   - `B`/`S`: 매수/매도 조건식
   - `BO`/`SO`: 매수/매도 최적화 조건식
   - `BOR`/`SOR`: 매수/매도 최적화 범위
   - `GAR`: GA 최적화 범위

# Condition - Min MA Alignment

## 전략 개요

이동평균선 정배열 전략은 명확한 상승 추세를 포착하는 전략입니다. 주요 특징:

- 이동평균선 5 > 20 > 60 정배열 확인
- 모든 이동평균선이 상승 중
- 현재가가 이동평균 5 위에 위치
- 분당 거래대금이 평균 이상

## 조건식

### 매수 조건식 - C_M_MAA_B

```python
# ================================
#  공통 계산 지표
# ================================
정배열여부            = (이동평균(5) > 이동평균(20) > 이동평균(60))
이평상승여부          = (이동평균(5) > 이동평균(5, 1) and 이동평균(20) > 이동평균(20, 1))
가격위치여부          = (현재가 > 이동평균(5))
거래량확인여부        = (분당거래대금 > 분당거래대금평균(20))

매수 = True

if not (관심종목 == 1):
    매수 = False
elif not (93000 <= 시분초 <= 150000):
    매수 = False
elif not (1000 < 현재가 <= 50000):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not 정배열여부:
    매수 = False
elif not 이평상승여부:
    매수 = False
elif not 가격위치여부:
    매수 = False
elif not 거래량확인여부:
    매수 = False

# 시가총액별 세부 조건
elif 시가총액 < 3000:
    if not (0.5 < 등락율 <= 15.0):
        매수 = False
    elif not (회전율 > 1.5):
        매수 = False
    elif not (체결강도 >= 110):
        매수 = False
    elif not (등락율각도(20) > 3):
        매수 = False
else:
    if not (0.3 < 등락율 <= 12.0):
        매수 = False
    elif not (회전율 > 1.0):
        매수 = False
    elif not (체결강도 >= 100):
        매수 = False
    elif not (등락율각도(20) > 2):
        매수 = False

if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_MAA_S

```python
# ================================
#  공통 계산 지표
# ================================
정배열이탈여부        = (이동평균(5) <= 이동평균(20))
이평하락여부          = (이동평균(5) < 이동평균(5, 1))
가격이탈여부          = (현재가 < 이동평균(20))

매도 = False

if 등락율 > 29.5:
    매도 = True
elif 수익률 >= 4.0 or 수익률 <= -2.5:
    매도 = True
elif 최고수익률 > 5.0 and 최고수익률 * 0.60 >= 수익률:
    매도 = True
elif 보유시간 > 180:  # 180분 (3시간)
    매도 = True
elif 정배열이탈여부:
    매도 = True
elif 이평하락여부 and 가격이탈여부:
    매도 = True
elif 등락율각도(10) < -5:
    매도 = True

if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```

가격위치여부          = (현재가 > 이동평균(5))

매수 = True

if not (관심종목 == 1):
    매수 = False
elif not (93000 <= 시분초 <= 150000):

## 조건식 개선 방향 연구

1) **골든크로스 확인**
   - 이동평균 5가 20을 상향 돌파한 시점 확인
   - 돌파 후 N분 이내 진입

2) **각도 측정**
   - 각 이동평균선의 상승 각도 계산
   - 일정 각도 이상일 때만 진입

3) **간격 확인**
   - 이동평균선 간 간격 측정
   - 너무 멀거나 가까우면 진입 금지

4) **추세 강도**
   - ADX 등 추세 강도 지표 추가
   - 강한 추세에서만 진입

5) **되돌림 매매**
   - 정배열 유지 중 이평 5 터치 시 재진입
   - 추세 내 조정 활용
