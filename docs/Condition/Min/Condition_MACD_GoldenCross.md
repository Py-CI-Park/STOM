## 소개

이 문서는 MACD(Moving Average Convergence Divergence) 지표의 골든크로스를 활용한 추세 추종 전략입니다.
MACD 선이 시그널 선을 상향 돌파하고, 이동평균선이 정배열되며, 볼린저밴드 중심선 이상에 위치할 때 매수합니다.
MACD 데드크로스 발생, 볼린저밴드 중심선 이탈, 또는 수익률 목표 달성 시 매도하는 중장기 추세 추종 전략입니다.

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **MACD Golden Cross (MACD 골든크로스 추세 추종 전략)**을 정의한다.

- **대상 시간 구간**: 09:30:00 ~ 14:30:00
- **대상 종목**: 시가총액 1,000억~3,000억 중형주
- **전략 타입**: 분봉 기반 MACD 골든크로스 추세 추종 전략
- **핵심 변수**: MACD, MACD Signal, 이동평균선, 볼린저밴드
- **업데이트 이력**:
  - 초기 문서 작성: MACD 골든크로스 추세 추종 전략

   - `S`: 매도 조건식을 의미합니다.

## 전략 개요

### 핵심 아이디어

1. **MACD 골든크로스**: MACD 선이 시그널 선을 상향 돌파하는 시점 포착
2. **이동평균 정배열**: 단기(20분) > 중기(60분) > 장기(120분) 이동평균 확인
3. **볼린저밴드 위치**: 현재가가 볼린저밴드 중심선(BBM) 이상 위치
4. **거래량 확인**: 평균 이상의 거래량으로 추세 신뢰도 검증
5. **추세 지속**: MACD 히스토그램이 양수이며 증가하는 추세

### 매수 전략

- **시간대**: 09:30 ~ 14:30 (장초반 급등락 제외, 장마감 직전 제외)
- **가격대**: 2,000원 ~ 100,000원
- **등락율**: -5% ~ 25% (급락/상한가 제외)
- **MACD 조건**: MACD > MACDS and MACD_N(1) <= MACDS_N(1) (골든크로스)
- **이동평균 정배열**: 이동평균(20) > 이동평균(60) > 이동평균(120)
- **볼린저밴드**: 현재가 >= BBM (중심선 이상)
- **RSI**: 40 < RSI < 70 (과매도/과매수 구간 제외)
- **거래량**: 분당거래대금 > 분당거래대금평균(60) * 1.2

### 매도 전략

- **목표 수익률**: 5% 익절
- **손절 수익률**: -3% 손절
- **MACD 데드크로스**: MACD < MACDS and MACD_N(1) >= MACDS_N(1)
- **볼린저밴드 이탈**: 현재가 < BBM (중심선 하향 돌파)
- **이동평균 이탈**: 현재가 < 이동평균(20) (단기 이평선 하향 돌파)
- **고점 대비 하락**: 최고수익률 - 수익률 >= 3%
- **보유시간**: 최대 240분 (4시간) 경과 시 강제 청산

## 조건식

### 매수 조건식 - C_M_MACD_GC_B

```python
# MACD 골든크로스 추세 추종 전략 - 매수 조건식
# 작성일: 2025-01-17
# 전략: MACD 골든크로스 발생 시 이동평균 정배열 확인 후 매수

# 분당순매수금액 계산
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
고가대비하락율 = ((고가 - 현재가) / 고가) * 100

# 공통 선결 조건
if not (시분초 >= 93000 and 시분초 < 143000):                               # 09:30~14:30 장중만 매수
    매수 = False
elif not (2000 < 현재가 <= 100000):                                          # 현재가 2,000원~100,000원 범위
    매수 = False
elif not (-5.0 <= 등락율 <= 25.0):                                           # 등락율 -5%~25% 범위
    매수 = False
elif not (현재가 < VI아래5호가):                                              # VI 발동 방지
    매수 = False
elif 라운드피겨위5호가이내:                                                    # 라운드피겨 근처 회피
    매수 = False

# MACD 골든크로스 조건
elif not (MACD > MACDS):                                                     # 현재 MACD가 시그널선 위
    매수 = False
elif not (MACD_N(1) <= MACDS_N(1)):                                          # 이전 분봉에서 MACD가 시그널선 아래 (골든크로스)
    매수 = False
elif not (MACD > 0):                                                         # MACD가 0선 위
    매수 = False
elif not (MACDH > 0):                                                        # MACD 히스토그램 양수
    매수 = False
elif not (MACDH > MACDH_N(1)):                                               # MACD 히스토그램 증가 추세
    매수 = False

# 이동평균선 정배열 조건
elif not (이동평균(20) > 이동평균(60)):                                       # 단기 이평 > 중기 이평
    매수 = False
elif not (이동평균(60) > 이동평균(120)):                                      # 중기 이평 > 장기 이평
    매수 = False
elif not (현재가 >= 이동평균(20)):                                            # 현재가가 단기 이평선 위
    매수 = False
elif not (이동평균(20) > 이동평균(20, 1)):                                    # 단기 이평선 상승 추세
    매수 = False

# 볼린저밴드 조건
elif not (현재가 >= BBM):                                                    # 현재가가 볼린저밴드 중심선 이상
    매수 = False
elif not (현재가 < BBU):                                                     # 현재가가 볼린저밴드 상단 미만 (과열 방지)
    매수 = False
elif not (BBM > BBM_N(1)):                                                   # 볼린저밴드 중심선 상승 추세
    매수 = False

# RSI 조건
elif not (RSI > 40):                                                         # RSI 40 이상 (과매도 구간 제외)
    매수 = False
elif not (RSI < 70):                                                         # RSI 70 미만 (과매수 구간 제외)
    매수 = False
elif not (RSI > RSI_N(1)):                                                   # RSI 상승 추세
    매수 = False

# 거래량 조건
elif not (분당거래대금 > 분당거래대금평균(60) * 1.2):                          # 분당거래대금이 60분 평균의 120% 이상
    매수 = False
elif not (분당거래대금 > 100):                                                # 분당거래대금 1억 이상
    매수 = False
elif not (분당매수수량 > 분당매도수량):                                       # 매수 수량이 매도 수량보다 많음
    매수 = False
elif not (당일거래대금 >= 1000 * 100):                                        # 당일거래대금 1,000억 이상
    매수 = False

# 체결강도 조건
elif not (체결강도 >= 105):                                                   # 체결강도 105 이상
    매수 = False
elif not (체결강도평균(30) > 100):                                            # 30분봉 평균 체결강도 100 이상
    매수 = False

# 등락율 각도 조건
elif not (등락율각도(30) > 0):                                                # 등락율 각도 양수 (상승 추세)
    매수 = False
elif not (등락율각도(30) < 45):                                               # 등락율 각도 45도 미만 (급등 제외)
    매수 = False

# 시가총액 조건
elif not (시가총액 >= 2000):                                                  # 시가총액 2,000억 이상
    매수 = False
elif not (시가총액 <= 100000):                                                # 시가총액 10조 이하
    매수 = False

# 분봉 캔들 조건
elif not (현재가 > 분봉시가):                                                 # 양봉
    매수 = False
elif not (분봉고가 > 분봉고가N(1)):                                            # 고가 갱신
    매수 = False
elif not ((현재가 - 분봉저가) > (분봉고가 - 현재가) * 2):                      # 아래 꼬리가 위 꼬리의 2배 이하 (강한 양봉)
    매수 = False

# 호가 잔량 조건
elif not (매수총잔량 > 매도총잔량 * 0.9):                                     # 매수 잔량이 매도 잔량의 90% 이상
    매수 = False

# 전일비 조건
elif not (전일비 > 80):                                                       # 전일비 80% 이상 (거래 활발)
    매수 = False
elif not (전일동시간비 > 100):                                                # 전일 동시간 대비 거래량 증가
    매수 = False

# 최종 매수 실행
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_MACD_GC_S

```python
# MACD 골든크로스 추세 추종 전략 - 매도 조건식
# 작성일: 2025-01-17
# 전략: MACD 데드크로스, 볼린저밴드 이탈, 목표 수익률 달성 시 매도

# VI 상한가 근접 시 즉시 매도
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성 시 매도
elif 수익률 >= 5.0:
    매도 = True

# 손절 조건
elif 수익률 <= -3.0:
    매도 = True

# MACD 데드크로스 시 매도
elif MACD < MACDS and MACD_N(1) >= MACDS_N(1):                               # MACD 데드크로스
    매도 = True

# MACD 히스토그램 음수 전환 시 매도
elif MACD < 0:                                                               # MACD가 0선 하향 돌파
    매도 = True

# 볼린저밴드 중심선 이탈 시 매도
elif 현재가 < BBM and 현재가N(1) >= BBM_N(1):                                 # 볼린저밴드 중심선 하향 돌파
    매도 = True

# 이동평균선 이탈 시 매도
elif 현재가 < 이동평균(20) and 현재가N(1) >= 이동평균(20, 1):                  # 단기 이평선 하향 돌파
    매도 = True

# RSI 과매수 구간 진입 시 매도
elif RSI >= 75:                                                              # RSI 과매수 (75 이상)
    매도 = True

# 고점 대비 하락 시 매도
elif 최고수익률 >= 3.0 and 최고수익률 - 수익률 >= 3.0:                        # 고점 대비 3% 하락
    매도 = True

# 보유시간 초과 시 매도
elif 보유시간 > 240:                                                         # 4시간(240분) 초과 보유 시 매도
    매도 = True

# 시가총액별 차별화된 매도 조건
elif 시가총액 < 5000:
    # 시가총액 5,000억 미만 종목 (변동성 큰 종목)
    if 수익률 > 0 and 체결강도 < 95:                                          # 수익 구간에서 체결강도 약화
        매도 = True
    elif 분당매도수량 > 분당매수수량 * 1.8:                                    # 매도 수량이 매수 수량의 1.8배 초과
        매도 = True
    elif 현재가 < 분봉저가N(1):                                                # 이전 분봉 저가 하향 돌파
        매도 = True
    elif 등락율각도(30) < -15:                                                # 등락율 각도 급락
        매도 = True

else:
```
