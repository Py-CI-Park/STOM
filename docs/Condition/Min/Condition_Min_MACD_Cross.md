# 조건식 (Condition)

- STOM 주식 자동거래에 사용하기 위한 조건식 문서
- [[Back_Testing_Guideline_Min]] 을(를) 기반으로 작성
- [[Condition_Document_Template_Guideline]] 을(를) 바탕으로 템플릿 구조를 적용한 문서

## 개요

본 문서는 STOM 주식 자동거래 시스템에서 **MACD 크로스 전략**을 정의한다.

- **대상 시간 구간**: 09:00 ~ 15:18 (장 전체)
- **대상 종목**: 전 종목 대상
- **전략 타입**: MACD 크로스 기반 추세 전환 전략
- **핵심 변수**: MACD, MACDS, MACDH
- **업데이트 이력**:
  - 초기 문서 작성: MACD 크로스 전략

## 소개

이 문서는 STOM 주식 자동거래 시스템에서 사용되는 **MACD 크로스 패턴** 조건식을 설명합니다.

**전략의 핵심 개념**:
- **시간대**: 09:00:00 ~ 15:18:00 (장중 전체)
- **패턴**: MACD 골든크로스 발생 시 매수
- **원리**: MACD 선이 시그널 선을 상향 돌파할 때 상승 추세 시작
- **특징**:
  - MACD > MACDS이고 이전봉에서는 MACD <= MACDS (골든크로스)
  - MACD 히스토그램이 양수로 전환
  - RSI가 50 이상 (상승 모멘텀 확인)
  - 현재가가 볼린저밴드 중심선 이상

## 가이드라인

- 모든 조건식은 Python으로 작성되어 있으며, 주식 거래의 자동화를 목표로 합니다.
- 각 조건식은 특정 조건을 만족할 때 매수 또는 매도를 실행합니다.

### 저장 이름 규칙

1. **기본 규칙**:
   - `C`: Condition을 의미합니다.
   - `M`: Min (분봉)을 의미합니다.
   - `MC`: MACD Cross (MACD 크로스)를 의미합니다.
   - `B`: 매수 조건식을 의미합니다.
   - `S`: 매도 조건식을 의미합니다.

2. **예시**:
   - `C_M_900_1518_MC_B`: 900-1518 MACD 크로스 패턴의 매수 조건식
   - `C_M_900_1518_MC_S`: 900-1518 MACD 크로스 패턴의 매도 조건식

# Condition - Min MACD Cross (MACD 크로스)

## 전략 개요

### 패턴 인식 로직

**MACD 크로스 (MACD Cross) 정의**:
- MACD 선이 시그널 선을 상향 돌파 (골든크로스)
- 이전봉: MACD <= MACDS
- 현재봉: MACD > MACDS
- MACD 히스토그램(MACDH)이 양수

**추세 확인**:
- RSI > 50 (상승 모멘텀)
- 현재가 >= BBM (볼린저밴드 중심선 이상)
- 이동평균(20) > 이동평균(60) (단기 > 중기)
- 등락율각도(30) > 5도

**거래량 확인**:
- 분당거래대금이 평균 이상
- 체결강도가 105 이상
- 분당매수수량 > 분당매도수량

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- RSI 최소 55 이상
- 체결강도 최소 115 이상
- 등락율각도(30) 최소 7도 이상

**대형주 (시가총액 >= 2500억)**:
- RSI 최소 50 이상
- 체결강도 최소 105 이상
- 등락율각도(30) 최소 5도 이상

## 조건식

### 매수 조건식 - C_M_900_1518_MC_B

```python
# ================================
#  공통 계산 지표
# ================================
분당순매수금액 = (분당매수수량 - 분당매도수량) * 현재가 / 1_000_000
전일종가 = 현재가 / (1 + (등락율 / 100))
시가등락율 = ((시가 - 전일종가) / 전일종가) * 100
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
MACD골든크로스여부 = (MACD > MACDS and MACD_N(1) <= MACDS_N(1))
MACD히스토그램양수여부 = (MACDH > 0)
상승모멘텀여부 = (RSI > 50 and 현재가 >= BBM)


# ================================
#  매수 조건 (MACD 크로스)
# ================================
매수 = True


# 공통 필터
if not (시분초 >= 90000 and 시분초 <= 151800):
    매수 = False
elif not (1000 < 현재가 <= 30000):
    매수 = False
elif not (0.5 < 등락율 <= 18.0):
    매수 = False
elif not (현재가 < VI아래5호가):
    매수 = False
elif 라운드피겨위5호가이내:
    매수 = False
elif not (데이터길이 >= 60):
    매수 = False

# MACD 골든크로스 확인
elif not MACD골든크로스여부:
    매수 = False
elif not MACD히스토그램양수여부:
    매수 = False

# 추세 및 모멘텀 확인
elif not 상승모멘텀여부:
    매수 = False
elif not (이동평균(20) > 이동평균(60)) if 데이터길이 > 60 else True:
    매수 = False
elif not (등락율각도(30) > 5):
    매수 = False

# 체결강도 및 거래량
elif not (체결강도 >= 105):
    매수 = False
elif not (분당매수수량 > 분당매도수량):
    매수 = False
elif not (분당거래대금 > 분당거래대금평균(30)):
    매수 = False

# 시가총액별 조건
else:
    if 시가총액 < 2500:
        if not (0.3 <= 시가등락율 and 시가등락율 < 12.0):
            매수 = False
        elif not (0.8 <= 시가대비등락율 and 시가대비등락율 < 14.0):
            매수 = False
        elif not (8 < 분당순매수금액 and 분당순매수금액 < 500):
            매수 = False
        elif not (전일비 > 25 and 전일동시간비 > 25):
            매수 = False
        elif not (회전율 > 0.6):
            매수 = False
        elif not (당일거래대금 > 90 * 100):
            매수 = False
        elif not (RSI > 55):
            매수 = False
        elif not (체결강도 >= 115):
            매수 = False
        elif not (등락율각도(30) > 7):
            매수 = False
        elif not (MACD > 0):
            매수 = False
    else:
        if not (0.5 <= 시가등락율 and 시가등락율 < 14.0):
            매수 = False
        elif not (1.0 <= 시가대비등락율 and 시가대비등락율 < 16.0):
            매수 = False
        elif not (15 < 분당순매수금액 and 분당순매수금액 < 700):
            매수 = False
        elif not (전일비 > 20 and 전일동시간비 > 20):
            매수 = False
        elif not (회전율 > 0.4):
            매수 = False
        elif not (당일거래대금 > 160 * 100):
            매수 = False
        elif not (RSI > 50):
            매수 = False
        elif not (체결강도 >= 105):
            매수 = False
        elif not (등락율각도(30) > 5):
            매수 = False


# ================================
#  매수 호출
# ================================
if 매수:
    self.Buy(종목코드, 종목명, 매수수량, 현재가, 매도호가1, 매수호가1, 데이터길이)
```

### 매도 조건식 - C_M_900_1518_MC_S

```python
# ================================
#  공통 계산 지표
# ================================
시가대비등락율 = ((현재가 - 시가) / 시가) * 100
MACD데드크로스여부 = (MACD < MACDS and MACD_N(1) >= MACDS_N(1))
MACD히스토그램음수여부 = (MACDH < 0)


# ================================
#  매도 조건 (MACD 크로스)
# ================================
매도 = False


# 상한가 근접
if 등락율 > 29.5:
    매도 = True

# 목표 수익률 달성
elif 수익률 >= 4.5:
    매도 = True

# 손절 조건
elif 수익률 <= -2.2:
    매도 = True

# 최고수익률 대비 하락
elif 최고수익률 > 6.0 and 최고수익률 * 0.72 >= 수익률:
    매도 = True

# MACD 데드크로스
elif MACD데드크로스여부 and 수익률 > 0.5:
    매도 = True

# MACD 히스토그램 음수 전환
elif MACD히스토그램음수여부 and 최고수익률 > 2.5:
    매도 = True

# RSI 과매수
elif RSI > 72 and 수익률 > 1.0:
    매도 = True

# 볼린저밴드 중심선 하향 이탈
elif 현재가 < BBM and 현재가N(1) >= BBM_N(1) and 수익률 > 0.3:
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < 2500:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.58:
        매도 = True
    elif 수익률 < -1.3 and 체결강도 < 98:
        매도 = True
    elif 최고수익률 > 3.5 and MACD < 0:
        매도 = True
else:
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * 0.52:
        매도 = True
    elif 수익률 < -1.2 and 체결강도 < 95:
        매도 = True
    elif 최고수익률 > 3.0 and MACD < 0:
        매도 = True

# 보유시간 초과
if 보유시간 > 160:
    매도 = True


# ================================
#  매도 호출
# ================================
if 매도:
    self.Sell(종목코드, 종목명, 매도수량, 현재가, 매도호가1, 매수호가1, 강제청산)
```
