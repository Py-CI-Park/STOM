## 조건식 개선 방향 연구

### 1. 추가 지표 활용 연구

**우선순위: 높음**
- **TA-Lib 지표 조합 확대**
  - ADX (추세 강도) + DMI (방향성) 조합
  - Ichimoku Cloud (일목균형표) 활용
  - 구현 난이도: 중

- **변동성 지표 추가**
  - ATR (Average True Range) 기반 손절폭 설정
  - Bollinger Band Width로 변동성 확대 구간 포착
  - 구현 난이도: 낮음

### 2. 시간대별 전략 세분화 연구

**우선순위: 높음**
- **장 시간대별 분봉 전략 차등화**
  - 09:00-10:00: 5분봉 전략
  - 10:00-14:00: 10분봉 전략
  - 14:00-15:00: 3분봉 전략
  - 구현 난이도: 중

### 3. 매도 조건 고도화 연구

**우선순위: 높음**
- **TA-Lib 매도 신호 강화**
  - MACD 데드크로스 + RSI 과매수 조합
  - Stochastic %K/%D 하향 교차 시 매도
  - 구현 난이도: 낮음

- **이동평균선 이탈 기반 매도**
  - 5MA 하향 돌파 시 경고
  - 20MA 하향 돌파 시 청산
  - 구현 난이도: 낮음

### 4. 리스크 관리 강화 연구

**우선순위: 높음**
- **다중 시간프레임 분석**
  - 1분봉 + 5분봉 + 10분봉 동시 확인
  - 상위 시간프레임 추세와 부합 시만 진입
  - 구현 난이도: 중

- **거래량 프로파일 분석**
  - 가격대별 거래량 분포 확인
  - 고거래량 구간 지지/저항 활용
  - 구현 난이도: 높음

### 5. 최적화 전략 개선 연구

**우선순위: 중간**
- **TA-Lib 파라미터 최적화**
  - RSI 기간: 14 → 10~20 범위 탐색
  - MACD (12,26,9) → (8,17,9) 등 변형 테스트
  - 구현 난이도: 중

- **기계학습 기반 지표 가중치 최적화**
  - 각 TA-Lib 지표의 예측력 평가
  - 가중치 동적 조정
  - 구현 난이도: 높음

### 구현 우선순위

1. **1단계 (즉시 구현)**:
   - 변동성 지표 추가 (ATR, BB Width)
   - TA-Lib 매도 신호 강화
   - 이동평균선 이탈 기반 매도

2. **2단계 (1개월 내)**:
   - TA-Lib 지표 조합 확대 (ADX, DMI, Ichimoku)
   - 시간대별 분봉 전략 차등화
   - 다중 시간프레임 분석

3. **3단계 (3개월 내)**:
   - TA-Lib 파라미터 최적화
   - 거래량 프로파일 분석
   - 기계학습 기반 최적화

# Condition - Min MACD Cross (MACD 크로스)

## 전략 개요

### 패턴 인식 로직

**MACD 크로스 (MACD Cross) 정의**:
- MACD 선이 시그널 선을 상향 돌파 (골든크로스)
- 이전봉: MACD <= MACDS
- 현재봉: MACD > MACDS
- MACD 히스토그램(MACDH)이 양수

**추세 확인**:
- RSI > 50 (상승 모멘텀)
- 현재가 >= BBM (볼린저밴드 중심선 이상)
- 이동평균(20) > 이동평균(60) (단기 > 중기)
- 등락율각도(30) > 5도

**거래량 확인**:
- 분당거래대금이 평균 이상
- 체결강도가 105 이상
- 분당매수수량 > 분당매도수량

### 시장 상황별 전략

**소형주 (시가총액 < 2500억)**:
- RSI 최소 55 이상
- 체결강도 최소 115 이상
- 등락율각도(30) 최소 7도 이상

**대형주 (시가총액 >= 2500억)**:
- RSI 최소 50 이상
- 체결강도 최소 105 이상
- 등락율각도(30) 최소 5도 이상

매도 = False

    매도 = True

elif MACDH < 0 and 최고수익률 > self.vars[5]:                                   # MACDH 음수 시 최고수익률 기준 최적화
    매도 = True

    매도 = True

elif 현재가 < BBM and 현재가N(1) >= BBM_N(1) and 수익률 > self.vars[8]:         # BBM 이탈 시 수익률 최적화
    매도 = True

elif 보유시간 > self.vars[9]:                                                   # 최대 보유시간 최적화
    매도 = True

# 시가총액별 매도 조건
elif 시가총액 < self.vars[10]:                                                  # 시가총액 기준 최적화
    if -1 < 수익률 <= 10 and (분당매도수량 - 분당매수수량) >= 매수총잔량 * self.vars[11]:  # 매도세 비율 최적화 - 소형
        매도 = True
