## Prompt
- 종목코드로 테이블 이름을 가지며, 위의 형태의 데이터 베이스로 index 에 yyyymmddhhmmss 의 형태로 시간 정보를 가지고 있는 주식 틱, 분 데이터 베이스가 있어.
여기서 
@Back_Testing_Guideline_Tick.md 파일과, @Back_Testing_Guideline_Min.md 파일의 조건식 설정에 따라, condition 폴더의 condition*.md 파일들 내부의 조건식의 형태로 조건식을 만들어서 백테스팅을 진행하고 있어. 하지만 여기서 하나씩 조건식 수정하면서 테스트 하면서 조건식을 찾기 너무 어려움이 있어.
그래서 위의 데이터베이스를 깊이 이해하고, 데이터 베이스를 이용해서 @Back_Testing_Guideline_Tick.md 파일과, @Back_Testing_Guideline_Min.md 파일의 조건식 설정 하여, 그 설정에 맞게 작성된 조건식의 변수들의 변화에 따른 수익률의 변화를 머신러닝 딥러닝으로 분석하고 싶은 생각이 있어.
하지만 정확하게 어떻게 해야하는지 아이디어가 떠오르지 않고, 데이터 분석가, 머신러닝, 딥러닝 엔지니어가 아니라서 효율적인 아이디어를 못찾고 있어.
그래서 너는 최고의 데이터 분석가, 머신러닝 전문가, 세계 최고의 퀀트 분석가로서 어떻게 하면 내가 가지고 있는 좋은 그래픽 카드 파워로 더 파워풀하게 조건식 연구에 머신러닝 딥러닝을 연구할 수 있을 찌 Idea 폴더에 아이디어 연구에서 마크다운 문서로 작성해줘.
