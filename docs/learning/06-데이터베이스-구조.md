## 데이터베이스 목록

### 위치

`_database/` 폴더

### DB 파일

- `setting.db`: 설정 및 계정 정보
- `stock_tick.db`: 주식 틱 데이터
- `stock_min.db`: 주식 분봉 데이터
- `coin_tick.db`: 코인 틱 데이터
- `coin_min.db`: 코인 분봉 데이터
- `tradelist.db`: 거래 내역
- `strategy.db`: 전략 코드
- `backtest.db`: 백테스트 결과

---

## 설정 DB

**파일:** `_database/setting.db`

### 주요 테이블

#### main
- 증권사, 거래소 설정
- 리시버 설정

#### stock
- 주식 거래 설정
- 투자금, 최대종목수

#### sacc
- 주식 계정 정보 (암호화)

#### stockbuyorder
- 주식 매수 주문 설정

#### stocksellorder
- 주식 매도 주문 설정

---

## 시장 데이터 DB

### stock_tick.db

#### moneytop 테이블
- 거래대금 순위

#### [종목코드] 테이블
- 체결시간
- 현재가, 시가, 고가, 저가
- 거래량, 거래대금
- 체결강도
- 호가 데이터

---

## 거래 내역 DB

**파일:** `_database/tradelist.db`

### 테이블

- `s_chegeollist`: 주식 체결
- `s_jangolist`: 주식 잔고
- `s_tradelist`: 주식 거래 내역
- `c_chegeollist`: 코인 체결
- `c_jangolist`: 코인 잔고
- `c_tradelist`: 코인 거래 내역
