## 백테스트 개요

### 백테스트 엔진 구조

```
[Total Process] - 관리
    ↓
[BackEngine 0-4] - 5개 병렬 실행
    ↓
[결과 집계] - 수익률, MDD, 승률 등
```

### 주요 파일

- `backtester/backtest.py`: 총괄 관리
- `backtester/backengine_kiwoom_tick.py`: 주식 틱 백테스트
- `backtester/backengine_kiwoom_min.py`: 주식 분봉 백테스트
- `backtester/back_static.py`: 성과 계산 함수

---

## 백테스트 실행

### GUI에서 실행

1. `백테스트` 탭
2. 전략 선택
3. 기간 설정 (예: 최근 3개월)
4. `백테스트 시작` 버튼

### 코드로 실행

```python
from backtester.backengine_kiwoom_tick import BackEngineKiwoomTick

# 백테스트 실행
engine = BackEngineKiwoomTick(vars)
result = engine.run()

print('총 수익률:', result['총수익률'])
print('승률:', result['승률'])
print('MDD:', result['MDD'])
```

---

## 성과 지표

- **총 수익률**: (최종 자산 - 초기 자산) / 초기 자산
- **승률**: 익절 건수 / 전체 거래 건수
- **평균 수익률**: 수익률 합계 / 거래 건수
- **MDD**: 최대 낙폭률
- **Sharpe Ratio**: (수익률 - 무위험 수익률) / 변동성
