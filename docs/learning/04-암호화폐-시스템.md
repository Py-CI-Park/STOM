## 업비트 시스템

### 구조

```
[Upbit REST API + WebSocket]
    ↓
[Upbit Receiver] - 실시간 데이터
    ↓
[Upbit Strategy] - 전략 연산
    ↓
[Upbit Trader] - 주문 실행
```

### 주요 파일

- `coin/upbit_websocket.py`: WebSocket 관리
- `coin/upbit_receiver_tick.py`: 데이터 수신
- `coin/upbit_strategy_tick.py`: 전략 연산
- `coin/upbit_trader.py`: 주문 실행

### 코드 예시

```python
# Receiver
class UpbitReceiverTick:
    def __init__(self):
        self.ws = WebSocketReceiver()
        
    def run(self):
        # WebSocket 구독
        self.ws.subscribe(['KRW-BTC', 'KRW-ETH'])
        
        while True:
            data = self.ws.recv()
            self.process(data)
```
