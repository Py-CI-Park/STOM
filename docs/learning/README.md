## 학습 개요

### STOM이란?

STOM은 한국 주식시장(키움증권)과 암호화폐시장(업비트, 바이낸스)을 위한 **전문가급 고빈도 자동매매 시스템**입니다.

**핵심 특징:**
- ⚡ **고성능 멀티프로세스 아키텍처**: 실시간 틱 데이터 처리
- 📊 **포괄적인 백테스팅**: Optuna 기반 전략 최적화
- 🎨 **PyQt5 GUI**: 직관적인 사용자 인터페이스
- 🔒 **안전성**: 리스크 관리, 암호화, 블랙리스트 등
- 🔧 **확장성**: 커스텀 전략 개발 및 추가 용이

### 학습 목표

이 학습 가이드를 완료하면 다음을 할 수 있습니다:

1. ✅ STOM의 전체 아키텍처와 동작 원리 이해
2. ✅ 주식 및 암호화폐 자동매매 시스템 운영
3. ✅ 커스텀 매매 전략 개발 및 백테스팅
4. ✅ 전략 최적화 및 성능 튜닝
5. ✅ 시스템 커스터마이징 및 기능 확장
6. ✅ 문제 해결 및 디버깅
