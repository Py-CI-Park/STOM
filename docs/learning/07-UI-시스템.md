## UI 구조

### 주요 컴포넌트

- **MainWindow**: 메인 윈도우 (`ui/ui_mainwindow.py`)
- **Writer**: UI 업데이트 스레드
- **UpdateTablewidget**: 테이블 업데이트
- **DrawChart**: 백테스트 차트
- **DrawRealChart**: 실시간 차트

---

## 메인 윈도우

**파일:** `ui/ui_mainwindow.py`

### 구조

```python
class MainWindow(QMainWindow):
    def __init__(self):
        # Queue 생성
        self.windowQ = Queue()
        # ...
        
        # 탭 생성
        self.create_tabs()
        
        # Writer 스레드
        self.writer = Writer(self.windowQ)
        self.writer.start()
```

### 탭 구성

- 로그 탭: 시스템 로그
- 주식 매매 탭: 주식 거래
- 코인 매매 탭: 코인 거래
- 백테스트 탭: 백테스트 실행
- 설정 탭: 시스템 설정

## 차트 시스템

### 실시간 차트

**파일:** `ui/ui_draw_realchart.py`

```python
class DrawRealChart:
    def __init__(self):
        self.plot = pg.PlotWidget()
        
    def update(self, data):
        self.plot.plot(data['price'])
```
