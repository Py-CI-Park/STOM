# 주식 틱데이터 데이터베이스 구조

이 문서는 STOM 주식 틱, 분 데이터를 SQLite3 형태로 수집했을 때 생성되는 데이터베이스의 구조를 설명합니다. 각 열은 주식 데이터의 다양한 측면을 나타내며, 데이터 타입은 `INTEGER` 또는 `REAL`입니다.

|---------|-------------|------|
| index | INTEGER | 데이터의 인덱스 번호 |
| 현재가 | REAL | 현재 주식 가격 |
| 시가 | REAL | 당일 시작 가격 |
| 고가 | REAL | 당일 최고 가격 |
